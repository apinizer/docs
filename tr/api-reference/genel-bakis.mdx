## Genel Bakış

Apinizer Management API, platformun tüm özelliklerine programatik erişim sağlar. RESTful API standartlarına uygun olarak tasarlanmıştır.

## API Endpoint'leri

### Base URL

API endpoint'leri şu base URL üzerinden erişilebilir:

```
https://{apinizer-manager}/api/v1
```

### Ana Endpoint Kategorileri

* **API Proxy Management**: API Proxy oluşturma, güncelleme, silme
* **Policy Management**: Policy yönetimi
* **User Management**: Kullanıcı yönetimi
* **Environment Management**: Ortam yönetimi
* **Analytics**: Analitik verilerine erişim

## Authentication

API'ye erişim için authentication gereklidir:

* **OAuth2**: OAuth2 token ile authentication
* **API Key**: API Key ile authentication
* **Bearer Token**: Bearer token ile authentication

## Request/Response Formatları

### Request Format

```http
POST /api/v1/api-proxies HTTP/1.1
Content-Type: application/json
Authorization: Bearer {token}

{
  "name": "example-api",
  "description": "Example API Proxy"
}
```

### Response Format

```json
{
  "id": "api-proxy-id",
  "name": "example-api",
  "description": "Example API Proxy",
  "status": "active"
}
```

## API Dokümantasyonu

Detaylı API dokümantasyonu için:

* **API Endpoint Listesi**: Tüm endpoint'lerin listesi
