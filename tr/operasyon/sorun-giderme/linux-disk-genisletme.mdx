## Diskleri Kontrol Etme

Sistemdeki diskleri listelemek için:

```bash
lsblk
```

Görselde görüldüğü üzere, eklenen disk **/dev/sdb** olarak görünmektedir. Örnek komutlar da **/dev/sdb** üzerinden ilerleyecektir.

## Yeni Disk Üzerinde Bölüm Oluşturma

```bash
sudo fdisk /dev/sdb
```

`fdisk` arayüzünde aşağıdaki adımları takip edin:

<Steps>
  <Step title="Yeni Bölüm Ekleyin">
    <code>n</code> tuşuna basın ve yeni bir bölüm oluşturun.
  </Step>
  <Step title="Tipi Seçin">
    <code>p</code> tuşuna basarak birincil bölüm oluşturun.
  </Step>
  <Step title="Bölüm Numarasını Girin">
    <code>1</code> yazarak bölüm numarasını belirtin.
  </Step>
  <Step title="Varsayılan Ayarları Kullanın">
    Tüm varsayılan ayarları kabul etmek için <code>Enter</code> tuşuna basın (iki kez).
  </Step>
  <Step title="Yaz ve Çık">
    Değişiklikleri kaydetmek için <code>w</code> tuşuna basın.
  </Step>
</Steps>

## Fiziksel Volume Oluşturma

```bash
sudo pvcreate /dev/sdb1
```

## Yeni Diski Var Olan Volume Group'a Dahil Etme

Sisteminizde bulunan volume group'a `/dev/sdb1` diskini eklemek için aşağıdaki komutu kullanabilirsiniz:

```bash
sudo vgextend <VG-NAME> /dev/sdb1
```

## Logical Volume'u Genişletme

**ubuntu-lv** isimli logical volume'u, ubuntu-vg isimli volume group içinde 100% boş alanla genişletmek için aşağıdaki komutu çalıştırabilirsiniz:

```bash
sudo lvextend -l +100%FREE /dev/ubuntu-vg/ubuntu-lv
```

## Dosya Sistemini Genişletme

```bash
sudo resize2fs /dev/ubuntu-vg/ubuntu-lv
```
