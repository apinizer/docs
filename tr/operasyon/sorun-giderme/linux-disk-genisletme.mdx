# Linux Üzerinde Disk Genişletme

![](/apinizer-dokumantasyonu/_/05BD9F48017A5D1E2319A7812864C3E5/1699865923290/images/common/note-macro-icon.svg)

Çok Önemli

**Bu işlem riskli bir işlemdir ve sunucunun yedeği yoksa gerçekleştirilmesi önerilmez.**

Linux sistemine yeni bir disk eklendikten sonra bu diskin **LVM**(Logical Volume Manager) kullanılarak nasıl yapılandırılacağı adım adım anlatılmaktadır.

## Diskleri kontrol etme:
[code] 
    lsblk
[/code]

CODE

![](/apinizer-dokumantasyonu/files/114262017/114262110/1/1728570727720/image-2024-10-10_17-32-6.png)

Görselde görüldüğü üzere, eklenen disk **/dev/sdb** olarak görünmektedir. Örnek komutlar da **/dev/sdb** üzerinden ilerleyecektir.

## Yeni Disk Üzerinde Bölüm Oluşturma
[code] 
    sudo fdisk /dev/sdb
[/code]

CODE

`fdisk` arayüzünde aşağıdaki adımları takip edin:

  1. **Yeni Bölüm Ekleyin** : `n` tuşuna basın ve yeni bir bölüm oluşturun.
  2. **Tipi Seçin** : `p` tuşuna basarak birincil bölüm oluşturun.
  3. **Bölüm Numarasını Girin** : `1` yazarak bölüm numarasını belirtin.
  4. **Varsayılan Ayarları Kullanın** : Tüm varsayılan ayarları kabul etmek için `Enter` tuşuna basın.
  5. **Varsayılan Ayarları Kullanın** : Tüm varsayılan ayarları kabul etmek için `Enter` tuşuna basın.
  6. **Yaz ve Çık** : Değişiklikleri kaydetmek için `w` tuşuna basın.

## Fiziksel Volume Oluşturma
[code] 
    sudo pvcreate /dev/sdb1
[/code]

CODE

## Volume Groupları Listeleme
[code] 
    vgdisplay
[/code]

CODE

![](/apinizer-dokumantasyonu/files/114262017/114262111/1/1728570805287/image-2024-10-10_17-33-24.png)

## Yeni Diski Var Olan Volume Group’a Dahil Etme

Sisteminizde bulunan volume group'a `/dev/sdb1` diskini eklemek için aşağıdaki komutu kullanabilirsiniz.
[code] 
    sudo vgextend &#60;VG-NAME&#62; /dev/sdb1
[/code]

CODE

## Logic Volumeleri Listeleme
[code] 
    lvdisplay
[/code]

CODE

![](/apinizer-dokumantasyonu/files/114262017/114262115/1/1728571511574/image-2024-10-10_17-45-10.png)

## Logic Volume’u Genişletme

**Ubuntu-lv** isimli logic volume'u, ubuntu-vg isimli volume group içinde 100% boş alanla genişletmek için aşağıdaki komutu çalıştırabilirsiniz:
[code] 
    sudo lvextend -l +100%FREE /dev/ubuntu-vg/ubuntu-lv
[/code]

CODE

## Dosya sistemini genişletin
[code] 
    sudo resize2fs /dev/ubuntu-vg/ubuntu-lv
[/code]

CODE
