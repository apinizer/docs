# API Developer Portal'ı SSL ile Başlatma

Bu sayfa, API Developer Portal'ın SSL/TLS sertifikası ile başlatılması işlemini açıklamaktadır.

## Genel Bakış

API Developer Portal'ı SSL/TLS sertifikaları ile güvenli şekilde başlatabilirsiniz.

## Yapılandırma

### 1. Sertifika Hazırlığı

Sertifikaları hazırlayın:

- SSL/TLS sertifikası oluşturun veya temin edin
- Private key'i hazırlayın
- Certificate chain'i hazırlayın

### 2. Kubernetes Secret Oluşturma

Sertifikaları Kubernetes secret olarak oluşturun:

```bash
kubectl create secret tls portal-tls \
  --cert=portal.crt \
  --key=portal.key \
  -n apinizer
```

### 3. Portal Deployment Yapılandırması

Portal deployment yapılandırmasında SSL ayarlarını yapın:

```yaml
env:
  - name: SSL_ENABLED
    value: "true"
  - name: SSL_KEY_STORE
    value: "/etc/ssl/certs/portal.jks"
  - name: SSL_KEY_STORE_PASSWORD
    valueFrom:
      secretKeyRef:
        name: portal-tls
        key: password
  - name: SERVER_PORT
    value: "8443"
```

### 4. Ingress Yapılandırması

Ingress yapılandırmasında TLS ayarlarını yapın:

```yaml
apiVersion: networking.k8s.io/v1
kind: Ingress
metadata:
  name: portal-ingress
spec:
  tls:
  - hosts:
    - portal.example.com
    secretName: portal-tls
  rules:
  - host: portal.example.com
    http:
      paths:
      - path: /
        pathType: Prefix
        backend:
          service:
            name: portal-service
            port:
              number: 8443
```
