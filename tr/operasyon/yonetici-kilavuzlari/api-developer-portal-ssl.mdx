## Genel Bakış

API Developer Portal'ı SSL/TLS sertifikaları ile güvenli şekilde başlatabilirsiniz.

## Yapılandırma

### Sertifika Hazırlığı

SSL/TLS sertifikası yapılandırması için gerekli bileşenler:

- SSL/TLS sertifikası oluşturma veya temin etme
- Private key hazırlama
- Certificate chain hazırlama

### Kubernetes Secret Oluşturma

Sertifikaları Kubernetes secret olarak oluşturun:

```bash
kubectl create secret tls portal-tls \
  --cert=portal.crt \
  --key=portal.key \
  -n apinizer
```

### Portal Deployment Yapılandırması

Portal deployment yapılandırmasında SSL ayarlarını yapılandırın:

```yaml
env:
  - name: SSL_ENABLED
    value: "true"
  - name: SSL_KEY_STORE
    value: "/etc/ssl/certs/portal.jks"
  - name: SSL_KEY_STORE_PASSWORD
    valueFrom:
      secretKeyRef:
        name: portal-tls
        key: password
  - name: SERVER_PORT
    value: "8443"
```

### Ingress Yapılandırması

Ingress yapılandırmasında TLS ayarlarını yapılandırın:

```yaml
apiVersion: networking.k8s.io/v1
kind: Ingress
metadata:
  name: portal-ingress
spec:
  tls:
  - hosts:
    - portal.example.com
    secretName: portal-tls
  rules:
  - host: portal.example.com
    http:
      paths:
      - path: /
        pathType: Prefix
        backend:
          service:
            name: portal-service
            port:
              number: 8443
```
