# Apinizer Cache için Grafana Üzerinde Örnek Dashboardlar

<Info>
  Grafana Kurulumu ve Veri Kaynağı Olarak Prometheus Entegrasyonu için [ilgili dokümantasyona göz atabilirsiniz](/tr/bakim-operasyon/yonetici-kilavuzlari/grafana-prometheus-data-source.mdx).

## Panel 1: Saniye Başına Önbellek Aramaları

<Card title="PromQL Sorgusu" icon="code">
  ```promql
  topk(5, sum by (cache) (rate(cache_gets_total{cache=~".*"}[2h])))
  ```
</Card>

**Görselleştirme:** Gauge
