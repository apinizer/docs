# Sunucu IP Değişikliğinde Yapılacaklar

Bu sayfa, Apinizer sunucu IP adresi değişikliğinde yapılması gereken işlemleri açıklamaktadır.

## Genel Bakış

Sunucu IP adresi değiştiğinde Apinizer yapılandırmalarında güncellemeler yapılması gerekmektedir.

## Yapılması Gerekenler

### 1. Environment Yapılandırması

Environment yapılandırmasını güncelleyin:

- Gateway adreslerini güncelleyin
- Manager adreslerini güncelleyin
- Cache adreslerini güncelleyin

### 2. DNS Kayıtları

DNS kayıtlarını güncelleyin:

- A kayıtlarını güncelleyin
- CNAME kayıtlarını güncelleyin
- DNS cache'i temizleyin

### 3. Load Balancer Yapılandırması

Load balancer yapılandırmasını güncelleyin:

- Backend server IP'lerini güncelleyin
- Health check endpoint'lerini kontrol edin

### 4. Firewall Kuralları

Firewall kurallarını güncelleyin:

- Yeni IP adreslerine izin verin
- Eski IP adreslerini kaldırın

### 5. Certificate Kontrolü

Certificate'leri kontrol edin:

- Certificate'lerde IP adresi varsa güncelleyin
- SAN (Subject Alternative Name) kayıtlarını kontrol edin
