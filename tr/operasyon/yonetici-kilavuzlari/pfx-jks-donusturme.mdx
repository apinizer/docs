# PFX Dosyasının JKS Dosyasına Çevrilmesi

Bu sayfa, PFX formatındaki sertifikanın JKS formatına dönüştürülmesi işlemini açıklamaktadır.

## Genel Bakış

PFX (PKCS#12) formatındaki sertifikaları JKS (Java KeyStore) formatına dönüştürebilirsiniz.

## Dönüşüm İşlemi

### 1. keytool Kullanarak Dönüşüm

keytool kullanarak PFX'ten JKS'e dönüştürün:

```bash
keytool -importkeystore \
  -srckeystore certificate.pfx \
  -srcstoretype PKCS12 \
  -srcstorepass <pfx-password> \
  -destkeystore keystore.jks \
  -deststoretype JKS \
  -deststorepass <jks-password>
```

### 2. openssl ve keytool Kullanarak Dönüşüm

openssl ve keytool kombinasyonu:

```bash
# PFX'ten private key ve certificate çıkarma
openssl pkcs12 -in certificate.pfx -nocerts -out private.key -nodes
openssl pkcs12 -in certificate.pfx -clcerts -nokeys -out certificate.crt

# JKS oluşturma
keytool -import -alias apinizer -file certificate.crt -keystore keystore.jks
keytool -importkey -alias apinizer -keypass <key-password> -keystore keystore.jks
```

### 3. Sertifika Bilgilerini Kontrol Etme

JKS içeriğini kontrol edin:

```bash
keytool -list -v -keystore keystore.jks
```
