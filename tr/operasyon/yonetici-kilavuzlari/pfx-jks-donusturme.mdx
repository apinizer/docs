---
title: "PFX Dosyasının JKS Dosyasına Çevrilmesi"
description: "PFX (PKCS#12) formatındaki sertifikaları JKS (Java KeyStore) formatına dönüştürür ve Apinizer modüllerinde kullanıma hazır hale getirir."
---

## Dönüşüm Yöntemleri

<Tabs>
    <Tab title="keytool ile Dönüşüm">
        `keytool` komutunu kullanarak doğrudan PFX'ten JKS'e dönüştürebilirsiniz:

        ```bash
        keytool -importkeystore \
          -srckeystore certificate.pfx \
          -srcstoretype PKCS12 \
          -srcstorepass <pfx-password> \
          -destkeystore keystore.jks \
          -deststoretype JKS \
          -deststorepass <jks-password>
        ```

        <Note>
            `<pfx-password>` yerine PFX dosyanızın şifresini, `<jks-password>` yerine ise oluşturulacak JKS dosyası için belirleyeceğiniz şifreyi yazın.
        </Note>
    </Tab>

    <Tab title="openssl ve keytool ile Dönüşüm">
        `openssl` ve `keytool` kombinasyonu ile iki aşamalı dönüşüm yapabilirsiniz:

        ```bash
        # PFX'ten private key ve certificate çıkarma
        openssl pkcs12 -in certificate.pfx -nocerts -out private.key -nodes
        openssl pkcs12 -in certificate.pfx -clcerts -nokeys -out certificate.crt

        # JKS oluşturma
        keytool -import -alias apinizer -file certificate.crt -keystore keystore.jks
        keytool -importkey -alias apinizer -keypass <key-password> -keystore keystore.jks
        ```

        <Warning>
            Bu yöntemde private key'i ayrı bir dosyaya çıkardığınız için güvenlik önlemlerini almayı unutmayın.
        </Warning>
    </Tab>
</Tabs>

## Sertifika Bilgilerini Kontrol Etme

```bash
keytool -list -v -keystore keystore.jks
```

<Check>
    Komut başarıyla çalıştığında, JKS içindeki sertifika bilgileri, alias adları ve geçerlilik tarihleri görüntülenir.
</Check>
