# Sıklıkla Sorulan Sorular

API Developer Portal kullanıcılarının sıkça yönelttiği soruların ve yanıtlarının yönetilmesini sağlar. Yönetici kullanıcılar, hem İngilizce hem Türkçe dillerinde soru-cevap içerikleri tanımlayarak portalın kullanıcı deneyimini zenginleştirir. Bu sayede, API tüketicileri portal üzerinden sıkça merak edilen bilgilere hızlıca erişebilir.

Sık Sorulan Sorular listesi, kullanıcıların API Portal'ı, API kullanım yöntemlerini veya genel kavramları daha iyi anlamalarına yardımcı olacak biçimde yapılandırılır.

## Sık Sorulan Soruların Kullanımı

Tanımlanan sorular, API Developer Portal üzerinde "Frequently Asked Questions" sayfasında görüntülenir. Kullanıcılar bu alanda sık sorulan soruları kategoriye göre filtreleyebilir, detaylı yanıtları inceleyebilir.

Bu yapı sayesinde:

- API tüketicileri sık karşılaşılan konularda hızlı çözüm bulur,
- Destek yükü azalır,
- Portalın kullanım kolaylığı ve rehberlik gücü artar.

**Sık Sorulan Sorular liste ekran detaylarını içeren görsele aşağıda yer verilmiştir:**

![Sık Sorulan Sorular Liste Ekranı](/images/faq-list.png)

### Liste Ekranı Alanları

| Alan | Açıklama |
|------|----------|
| **Soru (Question)** | İngilizce olarak sık sorulan sorunun metnidir. API Portal'da kullanıcı dil tercihi İngilizce olduğunda bu alan görüntülenir. |
| **Cevap (Answer)** | Soruya ait yanıt metnidir. Açıklayıcı ve bilgilendirici olmalıdır. |
| **Categori (Category)** | Soru hangi konuya ait ise burada kategori seçilir. |
| **Soru(TR) (Question(TR))** | Çoklu dil desteği aktif olduğunda görüntülenir. Sorunun türkçe versiyonu bu alana girilir. |
| **Cevap(TR) (Answer(TR))** | Çoklu dil desteği aktif olduğunda görüntülenir. Sorunun türkçe yanıtı bu alana girilir. |
| **Categori(TR) (Category(TR))** | Türkçe kategori adıdır. Portal türkçe dilinde görüntülendiğinde bu değer gösterilir. |

**Not:** Portal ayarlarında hem Türkçe Dil Seçeneği hem de İngilizce Dil Seçeneği etkinleştirildiğinde, sistem otomatik olarak Çoklu Dil (Multilanguage) moduna geçer. Eğer Çoklu Dil özelliği [Portal Ayarlarından](/tr/api-portal/api-portal) aktif hale getirilmişse, form alanları hem İngilizce hem Türkçe olarak görüntülenir, her dil içinde ayrı veri girişi yapılabilir.

## Yeni Soru Oluşturma

Yeni bir sık sorulan soru oluşturmak için ekranın sağ üst kısmındaki Create (+) butonuna tıklanır. Açılan pencerede (modal) aşağıdaki alanlar doldurularak kayıt işlemi tamamlanır. Zorunlu alanlar (* ile işaretli) doldurulduktan sonra Save butonuna basılarak kayıt sisteme eklenir.

### Soru Oluşturma Adımları

1. **Liste ekranını açın** - Sol menüden Portal Management başlığını genişletin. FAQ seçeneğine tıklayın.
2. **Yeni soru formunu açın** - Sağ üstteki Create (+) butonuna tıklayın. FAQ formu modal olarak açılır.
3. **Soru bilgilerini girin** - Question alanına soruyu yazın. Answer alanına yanıtı yazın. Category alanından uygun kategoriyi seçin. Çoklu dil desteği aktifse Türkçe alanları da doldurun.
4. **Kaydetme** - Zorunlu alanların dolu olduğunu kontrol edin. Save butonuna tıklayın.
