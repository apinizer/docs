Tek ekranda hangi organizasyonun hangi hesaplar üzerinden hangi API ürünlerine ulaştığını izleyebilir. Erişim durumlarını (aktif/pasif), yöneticilik rollerini ve kategori bilgilerini raporlayabilir."
---

**Ne İşe Yarar?**

* Organizasyon bazında hangi API ürünlerinin görüntülendiğini ve abonelik durumunu inceleme
* Hesapların aktif/pasif durumlarını ve organizasyon yöneticisi yetkilerini kontrol etme
* API ürünlerinin yayın durumlarını doğrulama ve kategori bazlı analiz yapma

**Kimler Kullanır?**

* **Portal Yöneticileri:** Organizasyonlar arası API erişim dağılımını görmek için
* **Destek Ekipleri:** Hesap veya API bazlı sorunları teşhis etmek için
* **Ürün Sahipleri:** API ürünlerinin hedef organizasyonlara erişip erişmediğini izlemek için

## Ekrana Erişim ve Yetkiler

* Portal sol menüsünde **Portal Management → API Usage by Organization** yolunu izleyin.
* Sadece yetkili roller (Portal Manager, Admin, Portal Business) sayfayı görüntüleyebilir.

| Alan | Açıklama |
|------|----------|
| **Organization** | Organizasyon adı. |
| **Account Full Name / Email** | Portal hesabının adı ve e-postası. |
| **Account Status** | Aktif/ Pasif Durumu |
| **Organization Manager** | Hesabın organizasyon yöneticisi olup olmadığı bilgisidir. |
| **API Product Name/ Title** | API ürünü tanımlayıcı bilgileri |
| **API Product Status** | Ürünün yayın durumu |
| **Categories** | Eklenmiş kategorileri temsil eder. |

**Kullanım Senaryoları:**

* Belirli bir organizasyonun hangi API ürünlerine eriştiğini öğrenir.
* Şikâyet gelen bir hesabın aktif olup olmadığını, yetkilerini ve eriştiği API ürünlerini görür.
* Yeni yayınlanan bir API ürününün hedef organizasyonlarda görünür olup olmadığını kontrol eder.
* Hangi kategorilerin hangi organizasyonlarda yoğun kullanıldığını fark eder.
