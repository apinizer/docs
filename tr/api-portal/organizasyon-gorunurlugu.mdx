# Organizasyon Görünürlüğü

Organizasyonel birimler ve ekipler arasında API görünürlüğünü kontrol edin. Belirli hizmetleri kimlerin keşfedip kullanabileceğini tanımlayan ayrıntılı erişim izinlerini yönetin.

Bu özellik, API'lerinizin kurum içinde veya dışında hangi organizasyonlar tarafından görüntülenebileceğini ve kullanılabileceğini kontrol etmenizi sağlar. Bu sayede, hassas API'lerin yalnızca yetkili organizasyonlar tarafından erişilebilir olmasını garanti edebilirsiniz.

## Genel Bakış

Organizasyon görünürlüğü ayarları, API Ürünleri oluşturulurken veya düzenlenirken **Görünürlük (Visibility)** sekmesi üzerinden yapılandırılır. Bu ayarlar sayesinde:

- API'lerinizin hangi organizasyonlar tarafından görüntülenebileceğini belirleyebilirsiniz
- Hassas API'leri belirli organizasyonlarla sınırlandırabilirsiniz
- Kurum içi ve dışı erişimleri ayrı ayrı yönetebilirsiniz

Detaylı bilgi için [API Ürünü](/tr/api-portal/api-urunu) sayfasındaki Görünürlük (Visibility) bölümünü inceleyebilirsiniz.
