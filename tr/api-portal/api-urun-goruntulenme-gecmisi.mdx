**API Product View History** raporu, Apinizer API Portal üzerinde yayınlanan API Product'ların kullanıcılar tarafından ne sıklıkla görüntülendiğini sunar. Bu ekran, API dokümantasyonlarının kullanım yoğunluğunu ve kullanıcı ilgisini ölçmek için temel analiz aracıdır.

## Genel Bakış

Bu rapor, geliştirici portalını ziyaret eden kullanıcıların hangi API ürünlerine ilgi gösterdiğini anlamanızı sağlar. Özellikle yeni yayınlanan bir API'nin farkındalığını ölçmek veya mevcut dokümantasyonun etkinliğini değerlendirmek için kullanılır.

### Zaman Bazlı Filtreleme
Rapor verilerini analiz etmek istediğiniz periyodu sol üst köşedeki butonlar aracılığıyla değiştirebilirsiniz:

- **Daily:** Günlük görüntülenme istatistiklerini listeler.
- **Weekly:** Haftalık (son 7 gün) verileri baz alır.
- **Monthly:** Aylık görüntülenme trendlerini gösterir.

## Rapor Alanları

Tabloda sunulan verilerin detayları aşağıdadır:

| Alan Adı | Açıklama |
| :--- | :--- |
| **API Product Name** | Portalda yayınlanan API ürününün adıdır. |
| **Account Name** | Sayfayı görüntüleyen kullanıcının kimliğidir. <br/> - **Public:** Giriş yapmamış (anonim) ziyaretçileri temsil eder.<br/> - **E-posta:** Sisteme giriş yapmış kayıtlı kullanıcıları temsil eder. |
| **View Count** | İlgili zaman diliminde, o ürünün detay sayfasının kaç kez açıldığını gösterir. |

## Kullanım Senaryoları

Bu raporu aşağıdaki analizleri gerçekleştirmek için kullanabilirsiniz:

1.  **Popülerlik Ölçümü:** Hangi API ürünlerinin geliştiriciler tarafından daha çok merak edildiğini tespit etmek.
2.  **Kullanıcı Segmentasyonu:** Dokümantasyonun daha çok anonim ziyaretçiler tarafından mı (Public), yoksa kayıtlı partnerler/geliştiriciler tarafından mı (Authenticated) incelendiğini görmek.
3.  **İçerik Stratejisi:** Düşük görüntülenme alan API'ların görünürlüğünü artırmak veya dokümantasyonunu iyileştirmek için karar almak.

<Warning>
  **Önemli Ayrım: Görüntülenme vs. Çalıştırma**

  Bu rapordaki veriler **API çağrı trafiğini (runtime usage)** yansıtmaz. Buradaki sayılar, sadece API Portal üzerindeki dokümantasyon sayfasının **görüntülenme** sayısıdır.

  Gerçek API kullanım istatistikleri, response süreleri ve performans metrikleri için lütfen **Analytics** ve **Traffic Logs** ekranlarını kullanınız.
</Warning>
