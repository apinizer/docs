## Genel Bakış

FTP Connector sayesinde:

* FTP/SFTP sunucularına bağlanabilirsiniz
* Dosyaları okuyabilirsiniz
* Dosyaları yazabilirsiniz
* Dosya listelerini alabilirsiniz
* Dosyaları silebilirsiniz

## FTP Bağlantı Yapılandırması

FTP Connector kullanmak için önce bir FTP Connection oluşturmanız gerekir.

### Connection Oluşturma

1. **Connection Management** menüsüne gidin
2. **FTP/SFTP Configuration** sekmesine tıklayın
3. **New Connection** butonuna tıklayın
4. Bağlantı bilgilerini girin:
   * **Name**: Bağlantı adı
   * **Protocol**: Protokol (FTP/SFTP)
   * **Host**: FTP sunucu adresi
   * **Port**: Port numarası
   * **Username**: Kullanıcı adı
   * **Password**: Şifre
   * **Passive Mode**: Pasif mod (FTP için)
   * **SSL/TLS**: SSL/TLS kullanımı
5. **Test Connection** butonuna tıklayın
6. **Save** butonuna tıklayın

## FTP Connector İşlemleri

### Dosya Yükleme (Upload)

FTP/SFTP sunucusuna dosya yüklemek için:

**Yapılandırma:**

* **Local File Path**: Yerel dosya yolu
* **Remote File Path**: Sunucudaki hedef yol
* **Overwrite**: Mevcut dosyaların üzerine yazma
* **Mode**: Transfer modu (ASCII veya Binary)

**Örnek:**

```json
{
  "operation": "upload",
  "localFilePath": "${localFilePath}",
  "remoteFilePath": "/uploads/${fileName}",
  "overwrite": true,
  "mode": "binary"
}
```

### Dosya İndirme (Download)

FTP/SFTP sunucusundan dosya indirmek için:

**Yapılandırma:**

* **Remote File Path**: Sunucudaki dosya yolu
* **Local File Path**: Yerel sistemdeki hedef yol
* **Overwrite**: Mevcut dosyaların üzerine yazma
* **Mode**: Transfer modu (ASCII veya Binary)

**Örnek:**

```json
{
  "operation": "download",
  "remoteFilePath": "/data/${fileName}",
  "localFilePath": "${downloadPath}/${fileName}",
  "overwrite": true,
  "mode": "binary"
}
```

### Dosya Okuma (Read)

Dosya okumak için:

1. Task Flow'a **FTP Connector** ekleyin
2. **Operation Type** olarak **Read** seçin
3. Yapılandırma:
   * **Remote Path**: Uzak dosya yolu
   * **Local Path**: Yerel dosya yolu (opsiyonel)
   * **Encoding**: Dosya kodlaması
   * **Delete After Read**: Okuduktan sonra sil (opsiyonel)

**Örnek:**

```json
{
  "operation": "read",
  "remotePath": "/incoming/data.csv",
  "encoding": "UTF-8",
  "deleteAfterRead": false
}
```

### Dosya Yazma (Write)

Dosya yazmak için:

1. Task Flow'a **FTP Connector** ekleyin
2. **Operation Type** olarak **Write** seçin
3. Yapılandırma:
   * **Remote Path**: Uzak dosya yolu
   * **Content**: Dosya içeriği
   * **Encoding**: Dosya kodlaması
   * **Mode**: Yazma modu (overwrite/append)

**Örnek:**

```json
{
  "operation": "write",
  "remotePath": "/outgoing/report.txt",
  "content": "${variables.reportContent}",
  "encoding": "UTF-8",
  "mode": "overwrite"
}
```

### Dosya Listeleme (List)

Dosya listesi almak için:

**Yapılandırma:**

* **Remote Directory**: Sunucudaki dizin yolu
* **Pattern**: Dosya pattern filtresi (opsiyonel)
* **Recursive**: Özyinelemeli listeleme

**Örnek:**

```json
{
  "operation": "list",
  "remoteDirectory": "/data",
  "pattern": "*.csv",
  "recursive": false
}
```

**Sonuç Formatı:**

```json
{
  "files": [
    {
      "name": "file1.csv",
      "size": 1024,
      "modified": "2024-01-15T10:30:00Z",
      "type": "file"
    },
    {
      "name": "file2.csv",
      "size": 2048,
      "modified": "2024-01-15T11:00:00Z",
      "type": "file"
    }
  ]
}
```

### Dosya Silme (Delete)

FTP/SFTP sunucusundan dosya silmek için:

**Yapılandırma:**

* **Remote File Path**: Silinecek dosya yolu

**Örnek:**

```json
{
  "operation": "delete",
  "remoteFilePath": "/data/${fileName}"
}
```

### Dizin Oluşturma (Create Directory)

FTP/SFTP sunucusunda dizin oluşturmak için:

**Yapılandırma:**

* **Remote Directory Path**: Oluşturulacak dizin yolu

**Örnek:**

```json
{
  "operation": "createDirectory",
  "remoteDirectoryPath": "/data/${newDirectory}"
}
```

### Dizin Silme (Delete Directory)

FTP/SFTP sunucusundan dizin silmek için:

**Yapılandırma:**

* **Remote Directory Path**: Silinecek dizin yolu
* **Recursive**: Özyinelemeli silme

**Örnek:**

```json
{
  "operation": "deleteDirectory",
  "remoteDirectoryPath": "/data/${directory}",
  "recursive": true
}
```

## Dosya Pattern'leri

### Pattern Eşleştirme

Dosyaları filtrelemek için pattern kullanın:

**Pattern Örnekleri:**

* `*.csv` - Tüm CSV dosyaları
* `data_*.txt` - "data_" ile başlayan dosyalar
* `*.log` - Tüm log dosyaları
* `2024-*.csv` - "2024-" ile başlayan CSV dosyaları

**Kullanım:**

```json
{
  "operation": "list",
  "remoteDirectory": "/data",
  "pattern": "*.csv"
}
```

## Transfer Modları

### Binary Mod

İkili dosyalar için binary mod kullanın:

**Kullanım Senaryoları:**

* Görüntüler, PDF'ler, çalıştırılabilir dosyalar
* Sıkıştırılmış dosyalar
* Veritabanı dosyaları

**Yapılandırma:**

```json
{
  "mode": "binary"
}
```

### ASCII Mod

Metin dosyaları için ASCII mod kullanın:

**Kullanım Senaryoları:**

* Metin dosyaları
* CSV dosyaları
* Yapılandırma dosyaları

**Yapılandırma:**

```json
{
  "mode": "ascii"
}
```

## Kullanım Senaryoları

### Senaryo 1: Dosya İşleme

```
FTP Connector (Dosya listesi)
├─→ Loop (Her dosya için)
│   ├─→ FTP Connector (Dosya indirme)
│   ├─→ Script Task (Dosya işleme)
│   ├─→ Database Connector (Veri kaydetme)
│   └─→ FTP Connector (Dosya silme veya taşıma)
```

### Senaryo 2: Rapor Gönderme

```
Database Connector (Veri çekme)
├─→ Script Task (Rapor oluşturma)
└─→ FTP Connector (Rapor yükleme)
```

### Senaryo 3: Dosya Senkronizasyonu

```
FTP Connector (Kaynak: Dosya listesi)
├─→ FTP Connector (Hedef: Dosya listesi)
├─→ Script Task (Karşılaştırma)
└─→ FTP Connector (Eksik dosyaları kopyalama)
```

## Yaygın Desenler

### Dosya İşleme Deseni

Dosya indirme, işleme ve yükleme:

```
Task Flow:
  ├─ Dosya İndirme (FTP Connector)
  ├─ Dosya İşleme (Script Connector)
  └─ İşlenmiş Dosya Yükleme (FTP Connector)
```

### Dosya İzleme Deseni

Yeni dosyalar için dizin izleme:

```
Task Flow:
  ├─ Dosya Listeleme (FTP Connector)
  ├─ Yeni Dosyaları Filtreleme
  └─ Her Dosyayı İşleme
```

### Dosya Arşivleme Deseni

Dosyaları arşiv dizinine taşıma:

```
Task Flow:
  ├─ Dosya Listeleme (FTP Connector)
  ├─ Dosyaları İşleme
  └─ Arşive Taşıma (FTP Connector)
```

## Hata Yönetimi

### Dosya İşlemi Hataları

Dosya işlemi hatalarını yönetme:

**Hata Türleri:**

* **Connection Error**: FTP sunucusuna bağlanılamadı
* **Authentication Error**: Geçersiz kimlik bilgileri
* **File Not Found**: Dosya bulunamadı
* **Permission Error**: Yetersiz yetkiler
* **Timeout Error**: İşlem zaman aşımı

**Hata Yönetimi:**

* **Retry Logic**: Geçici hatalarda yeniden deneme
* **Error Logging**: Dosya işlemi hatalarını loglama
* **Fallback**: Alternatif dosya kaynağı kullanma

### Retry Mekanizması

Dosya transferi hatalarında:

* **Retry Configuration**: Retry yapılandırması
* **Connection Retry**: Bağlantı retry
* **Transfer Retry**: Transfer retry

### Hata Bildirimleri

Dosya transferi başarısız olduğunda:

* **Error Logging**: Hata loglama
* **Notification**: Bildirim gönderme
* **Fallback**: Alternatif yöntemler

## Performans Optimizasyonu

### Batch Processing
