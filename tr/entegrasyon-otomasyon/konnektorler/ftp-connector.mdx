FTP Connector, Task Flow'larda FTP ve SFTP sunucularıyla dosya transferi yapmanızı sağlar. Dosya okuma, yazma, listeleme ve silme işlemlerini gerçekleştirebilirsiniz.

## Genel Bakış

FTP Connector sayesinde:

* FTP/SFTP sunucularına bağlanabilirsiniz
* Dosyaları okuyabilirsiniz
* Dosyaları yazabilirsiniz
* Dosya listelerini alabilirsiniz
* Dosyaları silebilirsiniz

## FTP Bağlantı Yapılandırması

FTP Connector kullanmak için önce bir FTP Connection oluşturmanız gerekir.

### Connection Oluşturma

1. **Connection Management** menüsüne gidin
2. **FTP/SFTP Configuration** sekmesine tıklayın
3. **New Connection** butonuna tıklayın
4. Bağlantı bilgilerini girin:
   * **Name**: Bağlantı adı
   * **Protocol**: Protokol (FTP/SFTP)
   * **Host**: FTP sunucu adresi
   * **Port**: Port numarası
   * **Username**: Kullanıcı adı
   * **Password**: Şifre
   * **Passive Mode**: Pasif mod (FTP için)
   * **SSL/TLS**: SSL/TLS kullanımı
5. **Test Connection** butonuna tıklayın
6. **Save** butonuna tıklayın

## FTP Connector İşlemleri

### Dosya Okuma (Read)

Dosya okumak için:

1. Task Flow'a **FTP Connector** ekleyin
2. **Operation Type** olarak **Read** seçin
3. Yapılandırma:
   * **Remote Path**: Uzak dosya yolu
   * **Local Path**: Yerel dosya yolu (opsiyonel)
   * **Encoding**: Dosya kodlaması
   * **Delete After Read**: Okuduktan sonra sil (opsiyonel)

**Örnek:**

```javascript
{
  "remotePath": "/incoming/data.csv",
  "encoding": "UTF-8",
  "deleteAfterRead": false
}
```

### Dosya Yazma (Write)

Dosya yazmak için:

1. Task Flow'a **FTP Connector** ekleyin
2. **Operation Type** olarak **Write** seçin
3. Yapılandırma:
   * **Remote Path**: Uzak dosya yolu
   * **Content**: Dosya içeriği
   * **Encoding**: Dosya kodlaması
   * **Mode**: Yazma modu (overwrite/append)

**Örnek:**

```javascript
{
  "remotePath": "/outgoing/report.txt",
  "content": "${variables.reportContent}",
  "encoding": "UTF-8",
  "mode": "overwrite"
}
```

### Dosya Listeleme (List)

Dosya listesi almak için:

1. Task Flow'a **FTP Connector** ekleyin
2. **Operation Type** olarak **List** seçin
3. Yapılandırma:
   * **Remote Path**: Uzak dizin yolu
   * **Pattern**: Dosya pattern'i (opsiyonel, örn: *.csv)
   * **Recursive**: Alt dizinleri de listele (true/false)

**Örnek:**

```javascript
{
  "remotePath": "/incoming",
  "pattern": "*.csv",
  "recursive": false
}
```

### Dosya Silme (Delete)

Dosya silmek için:

1. Task Flow'a **FTP Connector** ekleyin
2. **Operation Type** olarak **Delete** seçin
3. Yapılandırma:
   * **Remote Path**: Silinecek dosya yolu

**Örnek:**

```javascript
{
  "remotePath": "/incoming/processed/data.csv"
}
```

## Kullanım Senaryoları

### Senaryo 1: Dosya İşleme

```
FTP Connector (Dosya listesi)
├─→ Loop (Her dosya için)
│   ├─→ FTP Connector (Dosya okuma)
│   ├─→ Script Task (Dosya işleme)
│   ├─→ Database Connector (Veri kaydetme)
│   └─→ FTP Connector (Dosya silme veya taşıma)
```

### Senaryo 2: Rapor Gönderme

```
Database Connector (Veri çekme)
├─→ Script Task (Rapor oluşturma)
└─→ FTP Connector (Rapor yazma)
```

### Senaryo 3: Dosya Senkronizasyonu

```
FTP Connector (Kaynak: Dosya listesi)
├─→ FTP Connector (Hedef: Dosya listesi)
├─→ Script Task (Karşılaştırma)
└─→ FTP Connector (Eksik dosyaları kopyalama)
```
