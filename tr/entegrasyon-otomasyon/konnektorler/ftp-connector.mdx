FTP Connector, Task Flow'larda FTP ve SFTP protokolleri ile dosya işlemleri yapmanızı sağlayan konnektördür. Bu connector sayesinde dosyaları okuyabilir, yazabilir, listeleyebilir ve yönetebilirsiniz.

## Genel Bakış

FTP Connector şu özellikleri sunar:

* FTP ve SFTP protokolleri
* Dosya okuma ve yazma
* Dosya listeleme
* Dosya yükleme ve indirme
* Dizin işlemleri
* Güvenli bağlantılar (SFTP)

## FTP Connector Kullanımı

FTP Connector'ı Task Flow'da kullanmak için:

1. Task Flow tasarım ekranında **FTP Task** ekleyin
2. FTP connection'ı seçin
3. İşlem tipini belirleyin (Read, Write, List)
4. Dosya yolunu belirleyin

## Connection Yapılandırması

FTP connection yapılandırmak için:

1. **Connection Management** → **FTP/SFTP Configuration** bölümüne gidin
2. Yeni connection oluşturun

**FTP Ayarları:**
* **Protocol**: FTP veya SFTP
* **Host**: Sunucu adresi
* **Port**: Port (21 for FTP, 22 for SFTP)
* **Username**: Kullanıcı adı
* **Password**: Şifre
* **Passive Mode**: Pasif mod
* **SSL/TLS**: SSL/TLS ayarları (SFTP için)

## Dosya Okuma (FTP Read Connector)

Dosya okumak için:

### Ayarlar

* **File Path**: Dosya yolu
* **Encoding**: Karakter kodlaması (UTF-8, ISO-8859-1)
* **File Type**: Dosya tipi (Text, Binary)

### Örnek

```
File Path: /data/users.csv
Encoding: UTF-8
File Type: Text
Output: ${fileContent}
```

## Dosya Listeleme (FTP List Connector)

Dosyaları listelemek için:

### Ayarlar

* **Directory Path**: Dizin yolu
* **Pattern**: Dosya deseni (wildcard)
* **Recursive**: Alt dizinleri dahil et

### Örnek

```
Directory Path: /data
Pattern: *.csv
Recursive: No
Output: ${fileList}
```

## Dosya Yazma

Dosya yazmak için:

### Ayarlar

* **File Path**: Dosya yolu
* **Content**: İçerik
* **Encoding**: Karakter kodlaması
* **Mode**: Yazma modu (Overwrite, Append)

### Örnek

```
File Path: /data/output.txt
Content: ${data}
Encoding: UTF-8
Mode: Overwrite
```

## Dosya Yükleme ve İndirme

### Dosya Yükleme

```
Local Path: /local/file.txt
Remote Path: /remote/file.txt
```

### Dosya İndirme

```
Remote Path: /remote/file.txt
Local Path: /local/file.txt
```

## Variable Kullanımı

Variable'ları dosya yollarında kullanın:

```
File Path: /data/${fileName}
Directory Path: /data/${date}/files
```

## Best Practices

### Güvenlik

* **SFTP**: Mümkünse SFTP kullanın
* **Credentials**: Kimlik bilgilerini güvenli saklayın
* **Permissions**: Dosya izinlerini kontrol edin

### Performance

* **Connection Pooling**: Bağlantı havuzlama kullanın
* **Batch Operations**: Toplu işlemler yapın
* **Compression**: Büyük dosyalar için sıkıştırma kullanın

### Reliability

* **Error Handling**: Hata yönetimi yapın
* **Retry Logic**: Yeniden deneme mantığı ekleyin
* **File Locking**: Dosya kilitleme kullanın

## Örnek Senaryolar

### Senaryo 1: Dosya Okuma

```
Task: Read CSV File
Connector: FTP Read
File Path: /data/users.csv
Output: ${csvData}
```

### Senaryo 2: Dosya Listeleme

```
Task: List CSV Files
Connector: FTP List
Directory Path: /data
Pattern: *.csv
Output: ${fileList}
```

### Senaryo 3: Dosya Yükleme

```
Task: Upload Report
Connector: FTP Write
File Path: /reports/${reportName}.pdf
Content: ${reportContent}
```
