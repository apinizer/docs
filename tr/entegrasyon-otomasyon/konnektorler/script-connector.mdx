Script Connector, Task Flow'larda JavaScript veya Groovy script'leri çalıştırmanızı sağlayan konnektördür. Bu connector sayesinde veri dönüşümleri yapabilir, özel iş mantığı ekleyebilir ve karmaşık hesaplamalar gerçekleştirebilirsiniz.

## Genel Bakış

Script Connector şu özellikleri sunar:

* JavaScript ve Groovy desteği
* Veri dönüşümleri
* Özel iş mantığı
* Hesaplamalar
* String işlemleri
* Date/time işlemleri

## Script Connector Kullanımı

Script Connector'ı Task Flow'da kullanmak için:

1. Task Flow tasarım ekranında **Script Task** ekleyin
2. Script dilini seçin (JavaScript veya Groovy)
3. Script kodunu yazın
4. Input ve output variable'larını yapılandırın

## JavaScript Script

JavaScript script örneği:

```javascript
// Input: userData
var fullName = userData.firstName + " " + userData.lastName;
var email = userData.email.toLowerCase();
var timestamp = new Date().toISOString();

// Output
return {
    fullName: fullName,
    email: email,
    timestamp: timestamp
};
```

## Groovy Script

Groovy script örneği:

```groovy
// Input: userData
def fullName = "${userData.firstName} ${userData.lastName}"
def email = userData.email.toLowerCase()
def timestamp = new Date().format("yyyy-MM-dd HH:mm:ss")

// Output
[
    fullName: fullName,
    email: email,
    timestamp: timestamp
]
```

## Veri Dönüşümleri

### JSON Dönüşümü

```javascript
// Input: oldFormat
var newFormat = {
    id: oldFormat.user_id,
    name: oldFormat.full_name,
    email: oldFormat.email_address
};
return newFormat;
```

### Array İşlemleri

```javascript
// Input: items
var filtered = items.filter(item => item.status === "active");
var sorted = filtered.sort((a, b) => a.name.localeCompare(b.name));
return sorted;
```

### String İşlemleri

```javascript
// Input: text
var cleaned = text.trim().toLowerCase();
var words = cleaned.split(" ");
var capitalized = words.map(word => 
    word.charAt(0).toUpperCase() + word.slice(1)
).join(" ");
return capitalized;
```

## Date/Time İşlemleri

```javascript
// Current date/time
var now = new Date();
var formatted = now.toISOString();

// Date calculation
var tomorrow = new Date();
tomorrow.setDate(tomorrow.getDate() + 1);

// Date formatting
var formatted = now.toLocaleDateString("tr-TR");
```

## Variable Erişimi

Task Flow variable'larına erişim:

```javascript
// Input variables
var userId = input.userId;
var userName = input.userName;

// Global variables (if accessible)
var environment = global.environment;

// Output
return {
    processedUserId: userId,
    processedUserName: userName
};
```

## Error Handling

Script içinde hata yönetimi:

```javascript
try {
    // Process data
    var result = processData(input);
    return result;
} catch (error) {
    // Log error
    console.error("Error:", error.message);
    // Return error or throw
    throw new Error("Processing failed: " + error.message);
}
```

## Best Practices

### Performance

* **Efficient Code**: Verimli kod yazın
* **Avoid Loops**: Gereksiz döngülerden kaçının
* **Cache Results**: Sonuçları cache'leyin

### Security

* **Input Validation**: Girdi doğrulaması yapın
* **Sanitization**: Veri temizleme yapın
* **No Eval**: eval() kullanmayın

### Maintainability

* **Clear Code**: Açık ve anlaşılır kod yazın
* **Comments**: Yorumlar ekleyin
* **Error Messages**: Anlamlı hata mesajları kullanın

## Örnek Senaryolar

### Senaryo 1: Veri Dönüşümü

```javascript
// Input: oldData
var newData = {
    id: oldData.user_id,
    fullName: `${oldData.first_name} ${oldData.last_name}`,
    email: oldData.email_address.toLowerCase()
};
return newData;
```

### Senaryo 2: Hesaplama

```javascript
// Input: order
var total = order.items.reduce((sum, item) => 
    sum + (item.price * item.quantity), 0
);
var tax = total * 0.18;
var grandTotal = total + tax;

return {
    subtotal: total,
    tax: tax,
    grandTotal: grandTotal
};
```

### Senaryo 3: Koşullu İşlem

```javascript
// Input: user
if (user.status === "active" && user.verified) {
    return {
        canLogin: true,
        message: "Kullanıcı giriş yapabilir"
    };
} else {
    return {
        canLogin: false,
        message: "Kullanıcı giriş yapamaz"
    };
}
```
