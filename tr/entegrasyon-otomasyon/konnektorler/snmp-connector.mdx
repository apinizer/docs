## SNMP Connection

### Connection Configuration

SNMP Connector kullanmak için önce bir SNMP Connection oluşturmanız gerekir:

1. **Connection Management** sayfasına gidin
2. **SNMP Connections** sekmesine tıklayın
3. **New Connection** butonuna tıklayın
4. Bağlantı bilgilerini girin:
   * **Name**: Bağlantı adı
   * **SNMP Host**: SNMP sunucu adresi
   * **SNMP Port**: Port numarası (varsayılan: 162)
   * **SNMP Version**: SNMP versiyonu (v1, v2c, v3)
   * **Community**: Community string (v1/v2c için)
   * **Username**: Kullanıcı adı (v3 için)
   * **Authentication**: Kimlik doğrulama ayarları (v3 için)

Daha fazla bilgi için [SNMP Connections](/tr/entegrasyon-otomasyon/baglanti-yonetimi/snmp-connections) sayfasına bakın.

## SNMP Versions

### SNMP v1

SNMP v1 desteği:

* **Community-based**: Community tabanlı kimlik doğrulama
* **Basic Trap**: Temel trap gönderme
* **Limited Security**: Sınırlı güvenlik

### SNMP v2c

SNMP v2c desteği:

* **Community-based**: Community tabanlı kimlik doğrulama
* **Enhanced Trap**: Gelişmiş trap gönderme
* **Better Performance**: Daha iyi performans

### SNMP v3

SNMP v3 desteği:

* **User-based**: Kullanıcı tabanlı kimlik doğrulama
* **Encryption**: Şifreleme desteği
* **Enhanced Security**: Gelişmiş güvenlik

## SNMP Trap Configuration

### Basic Trap

Temel trap gönderme:

```json
{
  "operation": "trap",
  "host": "${snmpHost}",
  "oid": "1.3.6.1.4.1.12345.1.1",
  "value": "${trapValue}",
  "community": "${community}"
}
```

### Detailed Trap Configuration

Detaylı trap yapılandırması:

```json
{
  "operation": "sendTrap",
  "host": "${snmpHost}",
  "port": 162,
  "version": "v2c",
  "community": "${community}",
  "trapOID": "1.3.6.1.4.1.12345.1.0.1",
  "variables": [
    {
      "oid": "1.3.6.1.4.1.12345.1.1.1",
      "type": "OctetString",
      "value": "${eventMessage}"
    }
  ]
}
```

## SNMP Operations

### SNMP GET Operation

SNMP GET işlemi:

```json
{
  "operation": "get",
  "host": "${snmpHost}",
  "oid": "1.3.6.1.2.1.1.1.0",
  "community": "${community}",
  "version": "v2c"
}
```

### SNMP SET Operation

SNMP SET işlemi:

```json
{
  "operation": "set",
  "host": "${snmpHost}",
  "oid": "1.3.6.1.2.1.1.5.0",
  "value": "${newValue}",
  "type": "OctetString",
  "community": "${community}",
  "version": "v2c"
}
```

### Trap with Multiple Variables

Çoklu değişkenli trap:

```json
{
  "operation": "sendTrap",
  "host": "${snmpHost}",
  "port": 162,
  "version": "v2c",
  "community": "${community}",
  "trapOID": "1.3.6.1.4.1.12345.1.0.2",
  "variables": [
    {
      "oid": "1.3.6.1.4.1.12345.1.1.1",
      "type": "OctetString",
      "value": "${eventType}"
    },
    {
      "oid": "1.3.6.1.4.1.12345.1.1.2",
      "type": "Integer",
      "value": "${severity}"
    },
    {
      "oid": "1.3.6.1.4.1.12345.1.1.3",
      "type": "OctetString",
      "value": "${eventMessage}"
    }
  ]
}
```

### SNMP v3 Trap

SNMP v3 trap:

```json
{
  "operation": "sendTrap",
  "host": "${snmpHost}",
  "port": 162,
  "version": "v3",
  "username": "${snmpUsername}",
  "authProtocol": "SHA",
  "authPassword": "${authPassword}",
  "privProtocol": "AES",
  "privPassword": "${privPassword}",
  "trapOID": "1.3.6.1.4.1.12345.1.0.1",
  "variables": [
    {
      "oid": "1.3.6.1.4.1.12345.1.1.1",
      "type": "OctetString",
      "value": "${eventMessage}"
    }
  ]
}
```

## Variable Types

### Supported Types

Desteklenen değişken tipleri:

* **OctetString**: Metin değerleri
* **Integer**: Tam sayı değerleri
* **Counter32**: 32-bit sayaç
* **Counter64**: 64-bit sayaç
* **Gauge32**: 32-bit gauge
* **TimeTicks**: Zaman damgası
* **IpAddress**: IP adresi
* **ObjectIdentifier**: OID

## OID Management

### Standard OIDs

Standart OID'ler:

* **System OIDs**: Sistem OID'leri
* **Interface OIDs**: Arayüz OID'leri
* **Custom OIDs**: Özel OID'ler

### OID Examples

OID örnekleri:

```
1.3.6.1.2.1.1.1.0 - sysDescr
1.3.6.1.2.1.1.3.0 - sysUpTime
1.3.6.1.2.1.1.5.0 - sysName
```

## Usage Examples

### Example 1: Error Notification

```
Error Handler
└─→ SNMP Connector (Hata trap'i)
    - Trap OID: 1.3.6.1.4.1.12345.1.0.1
    - Variables:
        - Error Type: ${errorType}
        - Error Message: ${errorMessage}
        - Severity: ${severity}
```

### Example 2: System Alert

```
System Monitor
└─→ SNMP Connector (Sistem uyarısı)
    - Trap OID: 1.3.6.1.4.1.12345.1.0.2
    - Variables:
        - Alert Type: ${alertType}
        - Resource: ${resourceName}
        - Status: ${status}
```

### Example 3: Performance Metric

```
Performance Monitor
└─→ SNMP Connector (Performans metrikleri)
    - Trap OID: 1.3.6.1.4.1.12345.1.0.3
    - Variables:
        - Metric Name: ${metricName}
        - Value: ${metricValue}
        - Threshold: ${threshold}
```
