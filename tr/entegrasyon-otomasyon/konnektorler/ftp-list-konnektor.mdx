---

FTP dosya listeleme connector ile FTP bağlantınızdaki tanımlı dizin içerisindeki dosyaları listeyelebilir. Bu listeleme işleminin sonucunu dönen API Proxy'ler oluşturabilirsiniz.

Oluşturduğunuz FTP dosya listeleme connector'üne HTTP GET isteği atarak ilgili FTP bağlantı tanımındaki path ve bu path altında bulunan dosya ve dizinleri listeyebilirsiniz.

## Örnek İstek

```bash
curl --location --request GET "https://<APINIZER_ACCESS_URL>/<RELATIVE_PATH>"
```
