<Info>
  Oluşturduğunuz FTP dosya listeleme connector'üne HTTP GET isteği atarak ilgili FTP bağlantı tanımındaki path ve bu path altında bulunan dosya ve dizinleri listeleyebilirsiniz.
</Info>

## Örnek İstek

```bash
curl --location --request GET "https://<APINIZER_ACCESS_URL>/<RELATIVE_PATH>"
```
