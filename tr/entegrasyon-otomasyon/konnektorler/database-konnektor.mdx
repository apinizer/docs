Bir eylem gerçekleştiğinde sonucun seçili veritabanı bağlantı tanımında işlem görmesi gerekebilir.

## Konfigürasyon

Veritabanı Connector konfigürasyonu için kullanılan alanlar aşağıdaki tabloda görülmektedir.

| Alan | Açıklama |
|------|----------|
| **Tip** (Type) | Veritabanı tipi seçilir. |
| **Bağlantı Havuzu Tanımı** (Connection Pool Definition) | Seçilen veritabanı türüne göre veritabanı bağlantısına ait tanım havuzdan seçilir. Bağlantı oluşturmak hakkında detaylı bilgi için [tıklayınız](/tr/entegrasyon-otomasyon/baglanti-yonetimi/database-connections). |
| **Operasyon** (Operation) | Operasyon tipi seçilir. |
| **Sorgu Zaman Aşımı (sn)** (Query Timeout (sec)) | Sorgunun çalışma süresine ait aşım süresi bilgisi girilir. |
| **SQL** | Eylem içinde çalıştırılacak sorgu yazılır. SQL alanında değişkenler kullanılabilir. |

<Info>
SQL alanında değişkenler kullanılabilir.
</Info>
