## Konfigürasyon

<img
    src="/images/yonetici/konnektor/database.png"
    alt="Veritabanı Connector Konfigürasyonu"
    width="600"
    className="mr-auto"
/>

Konfigürasyon alanları:

<CardGroup cols={2}>
  <Card title="Tip" icon="database">
    **Alan:** Tip (Type)
    
    Veritabanı tipi seçilir.
  </Card>
  
  <Card title="Bağlantı Havuzu Tanımı" icon="plug">
    **Alan:** Bağlantı Havuzu Tanımı (Connection Pool Definition)
    
    Seçilen veritabanı türüne göre veritabanı bağlantısına ait tanım havuzdan seçilir. Bağlantı oluşturmak hakkında detaylı bilgi için [tıklayınız](/tr/entegrasyon-otomasyon/baglanti-yonetimi/database).
  </Card>
  
  <Card title="Operasyon" icon="gear">
    **Alan:** Operasyon (Operation)
    
    Operasyon tipi seçilir.
  </Card>
  
  <Card title="Sorgu Zaman Aşımı" icon="clock">
    **Alan:** Sorgu Zaman Aşımı (sn) (Query Timeout (sec))
    
    Sorgunun çalışma süresine ait aşım süresi bilgisi girilir.
  </Card>
  
  <Card title="SQL" icon="code">
    **Alan:** SQL
    
    Eylem içinde çalıştırılacak sorgu yazılır.
    
    <Info>
      SQL alanında değişkenler kullanılabilir.
    </Info>
  </Card>
</CardGroup>
