## Linux Connection

### Connection Configuration

Linux Script Connector kullanmak için önce bir Linux Connection oluşturmanız gerekir:

1. **Connection Management** sayfasına gidin
2. **Linux Connections** sekmesine tıklayın
3. **New Connection** butonuna tıklayın
4. Bağlantı bilgilerini girin:
   * **Name**: Bağlantı adı
   * **SSH Host**: SSH sunucu adresi
   * **SSH Port**: Port numarası (varsayılan: 22)
   * **Username**: Kullanıcı adı
   * **Authentication**: Kimlik doğrulama yöntemi
     * **Password**: Şifre (Secret Manager'da saklanır)
     * **SSH Key**: SSH anahtarı
   * **Timeout**: Bağlantı zaman aşımı

Daha fazla bilgi için [Linux Connections](/tr/entegrasyon-otomasyon/baglanti-yonetimi/linux-connections) sayfasına bakın.

## Script Execution

### Basic Script

Temel script çalıştırma:

```json
{
  "operation": "executeScript",
  "connection": "${linuxConnection}",
  "script": "#!/bin/bash\necho 'Hello World'\ndate",
  "workingDirectory": "/tmp"
}
```

### Script with Parameters

Parametreli script:

```json
{
  "operation": "executeScript",
  "connection": "${linuxConnection}",
  "script": "#!/bin/bash\necho $1\necho $2",
  "parameters": ["${param1}", "${param2}"],
  "workingDirectory": "/tmp"
}
```

### Script File Execution

Dosyadan script çalıştırma:

```json
{
  "operation": "executeScriptFile",
  "connection": "${linuxConnection}",
  "scriptPath": "/opt/scripts/backup.sh",
  "parameters": ["${backupPath}", "${retentionDays}"],
  "workingDirectory": "/opt/scripts"
}
```

## Command Execution

### Single Command

Tek komut çalıştırma:

```json
{
  "command": "ls -la /var/log",
  "workingDirectory": "/home/user",
  "timeout": 30
}
```

### Detailed Command Configuration

Detaylı komut yapılandırması:

```json
{
  "operation": "executeCommand",
  "connection": "${linuxConnection}",
  "command": "ls -la /tmp",
  "workingDirectory": "/tmp"
}
```

### Multiple Commands

Çoklu komut çalıştırma:

```json
{
  "operation": "executeCommands",
  "connection": "${linuxConnection}",
  "commands": [
    "cd /opt/app",
    "tar -czf backup.tar.gz logs/",
    "mv backup.tar.gz /backup/"
  ],
  "workingDirectory": "/opt/app"
}
```

### Command with Output

Çıktı ile komut:

```json
{
  "operation": "executeCommand",
  "connection": "${linuxConnection}",
  "command": "df -h",
  "outputVariable": "diskUsage"
}
```

## File Operations

### File Upload

Dosya yükleme:

```json
{
  "operation": "uploadFile",
  "connection": "${linuxConnection}",
  "localPath": "/local/path/file.txt",
  "remotePath": "/remote/path/file.txt",
  "permissions": "644"
}
```

### File Download

Dosya indirme:

```json
{
  "operation": "downloadFile",
  "connection": "${linuxConnection}",
  "remotePath": "/remote/path/file.txt",
  "localPath": "/local/path/file.txt"
}
```

### File Content

Dosya içeriği okuma:

```json
{
  "operation": "readFile",
  "connection": "${linuxConnection}",
  "remotePath": "/var/log/app.log",
  "outputVariable": "fileContent"
}
```

## Usage Examples

### Example 1: Backup Script

```
Scheduled Job
└─→ Linux Script Connector (Backup script)
    - Script: backup.sh
    - Parameters: [backupPath, retentionDays]
    - Output: backupStatus
```

### Example 2: Log Analysis

```
Log Analysis Task
├─→ Linux Script Connector (Log dosyasını oku)
│   - Command: tail -n 100 /var/log/app.log
│   - Output: logContent
└─→ Script Connector (Log analizi)
```

### Example 3: File Transfer

```
File Processing Task
├─→ Linux Script Connector (Dosya indir)
│   - Operation: downloadFile
│   - Remote Path: /remote/data.csv
│   - Local Path: /local/data.csv
└─→ Script Connector (Dosya işleme)
```
