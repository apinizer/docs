Groovy veya Javascript kullanarak herhangi programatik bir işlem sürece dahil edilebilir.

## Konfigürasyon

Script ayarlarını içeren görsele aşağıda yer verilmiştir:

<img
    src="/images/yonetici/konnektor/script.png"
    alt="Script Connector Konfigürasyonu"
    width="600"
    className="mr-auto"
/>

Script konfigürasyonu için kullanılan alanlar aşağıdaki tabloda görülmektedir.

| Alan | Açıklama |
|------|----------|
| **Script Tipi** (Script Type) | Eylemde çalıştırılacak script dili seçilir. Bunlar; JavaScript, Groovy. |
| **Script** | resultMap değişkeni ile eylemin çıktısı üretilir. |

<Info>
Script alanında değişkenler kullanılabilir.
</Info>
