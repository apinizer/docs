## SMTP Configuration

### Email Connection

Email Connector kullanmak için önce bir Email SMTP Connection oluşturmanız gerekir:

1. **Connection Management** sayfasına gidin
2. **Email SMTP Configuration** sekmesine tıklayın
3. **New Connection** butonuna tıklayın
4. Bağlantı bilgilerini girin:
   * **Name**: Bağlantı adı
   * **SMTP Host**: SMTP sunucu adresi
   * **SMTP Port**: SMTP port numarası (genellikle 25, 465 veya 587)
   * **Username**: SMTP kullanıcı adı
   * **Password**: SMTP şifresi (Secret Manager'da saklanır)
   * **Use TLS/SSL**: TLS/SSL kullanımı
   * **From Address**: Varsayılan gönderen adresi

Daha fazla bilgi için [Email SMTP Configuration](/tr/entegrasyon-otomasyon/baglanti-yonetimi/email-smtp-configuration) sayfasına bakın.

## Email Configuration

### Basic Email

Temel e-posta gönderme:

* **To**: Alıcı e-posta adresi(leri)
* **CC**: Karbon kopya alıcıları
* **BCC**: Gizli karbon kopya alıcıları
* **Subject**: E-posta konusu
* **Body**: E-posta içeriği

### Email Format

E-posta formatı:

* **Plain Text**: Düz metin e-posta
* **HTML**: HTML formatında e-posta
* **Multipart**: Hem HTML hem metin içeren e-posta

### HTML Email Example

```html
<html>
<body>
  <h1>Merhaba ${customerName},</h1>
  <p>Siparişiniz başarıyla oluşturuldu.</p>
  <p><strong>Sipariş No:</strong> ${orderId}</p>
  <p><strong>Tutar:</strong> ${totalAmount} TL</p>
  <p>Teşekkürler!</p>
</body>
</html>
```

### Plain Text Email Example

```
Merhaba ${customerName},

Siparişiniz başarıyla oluşturuldu.

Sipariş No: ${orderId}
Tutar: ${totalAmount} TL

Teşekkürler!
```

## Email Templates

### Template Variables

Şablon değişkenleri:

* **Variable Substitution**: Değişken yerine koyma
* **Dynamic Content**: Dinamik içerik
* **Conditional Content**: Koşullu içerik

### Template Example

```html
<html>
<body>
  <h1>${greeting}</h1>
  <p>${message}</p>
  {{#if hasItems}}
  <ul>
    {{#each items}}
    <li>${this.name}: ${this.value}</li>
    {{/each}}
  </ul>
  {{/if}}
</body>
</html>
```

## Dynamic Email Content

### Variable Substitution

Değişken yerine koyma:

**Subject:**
```
Sipariş Onayı - ${orderId}
```

**Body:**
```
Sayın ${customerName},

Siparişiniz (${orderId}) ${orderDate} tarihinde oluşturuldu.
Toplam tutar: ${totalAmount} TL

Sipariş Detayları:
${orderItems}
```

### Conditional Content

Koşullu içerik:

```html
{{#if orderStatus == 'completed'}}
<p>Siparişiniz tamamlandı!</p>
{{else if orderStatus == 'pending'}}
<p>Siparişiniz bekliyor.</p>
{{else}}
<p>Siparişiniz işleniyor.</p>
{{/if}}
```

## Error Handling

### Email Errors

E-posta hataları:

* **SMTP Connection Errors**: SMTP bağlantı hataları
* **Authentication Errors**: Kimlik doğrulama hataları
* **Invalid Email Addresses**: Geçersiz e-posta adresleri
* **Attachment Errors**: Ek hataları

### Error Handling Strategies

Hata yönetimi stratejileri:

* **Retry on Failure**: Başarısızlıkta yeniden deneme
* **Error Logging**: Hata loglama
* **Fallback Notification**: Yedek bildirim
* **Error Notification**: Hata bildirimi

## Usage Examples

### Example 1: Simple Email

```
Email Connector:
- Connection: SMTP Connection
- To: ${customerEmail}
- Subject: Sipariş Onayı
- Body: Siparişiniz başarıyla oluşturuldu. Sipariş No: ${orderId}
- Format: Plain Text
```

### Example 2: HTML Email with Template

```
Email Connector:
- Connection: SMTP Connection
- To: ${customerEmail}
- CC: ${managerEmail}
- Subject: Sipariş Detayları - ${orderId}
- Body: ${emailTemplate}
- Format: HTML
- Template Variables:
    customerName: ${customerName}
    orderId: ${orderId}
    orderItems: ${orderItems}
    totalAmount: ${totalAmount}
```

### Example 3: Email with Attachment

```
Email Connector:
- Connection: SMTP Connection
- To: ${recipientEmail}
- Subject: Rapor - ${reportDate}
- Body: Ekte raporu bulabilirsiniz.
- Format: HTML
- Attachments:
    - File Name: report.pdf
      File Content: ${reportContent}
      Content Type: application/pdf
```

### Example 4: Error Notification Email

```
Email Connector:
- Connection: SMTP Connection
- To: ${adminEmail}
- Subject: Hata Bildirimi - ${errorType}
- Body: 
  <html>
  <body>
    <h2>Hata Oluştu</h2>
    <p><strong>Hata Tipi:</strong> ${errorType}</p>
    <p><strong>Hata Mesajı:</strong> ${errorMessage}</p>
    <p><strong>Zaman:</strong> ${errorTime}</p>
    <p><strong>Detaylar:</strong></p>
    <pre>${errorDetails}</pre>
  </body>
  </html>
- Format: HTML
```

## Best Practices

### Email Design

* **Clear Subject Lines**: Net konu satırları
* **Professional Formatting**: Profesyonel formatlama
* **Mobile Friendly**: Mobil uyumlu tasarım
* **Accessibility**: Erişilebilirlik

### Security

* **Use TLS/SSL**: TLS/SSL kullanın
* **Secure Credentials**: Kimlik bilgilerini Secret Manager'da saklayın
* **Input Validation**: Girdi doğrulama
* **SPF/DKIM**: SPF/DKIM kayıtları

### Performance

* **Async Sending**: Asenkron gönderme
* **Batch Sending**: Toplu gönderme
* **Rate Limiting**: Hız sınırlama
* **Connection Pooling**: Bağlantı havuzlama

### Error Handling

* **Retry Transient Errors**: Geçici hatalarda yeniden deneme
* **Log Email Failures**: E-posta başarısızlıklarını loglayın
* **Monitor Email Delivery**: E-posta teslimatını izleyin
* **Handle Bounces**: Geri dönen e-postaları yönetin
