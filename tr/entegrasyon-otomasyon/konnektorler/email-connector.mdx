Email Connector, Task Flow'larda e-posta gönderme ve okuma işlemlerini gerçekleştirmenizi sağlayan konnektördür. Bu connector sayesinde bildirim e-postaları gönderebilir, e-posta şablonları kullanabilir ve e-posta okuma işlemleri yapabilirsiniz.

## Genel Bakış

Email Connector şu özellikleri sunar:

* E-posta gönderme (SMTP)
* E-posta okuma (IMAP, POP3)
* HTML ve metin formatları
* E-posta şablonları
* Ek dosyalar
* Çoklu alıcılar

## Email Connector Kullanımı

Email Connector'ı Task Flow'da kullanmak için:

1. Task Flow tasarım ekranında **Email Task** ekleyin
2. Email connection'ı seçin
3. E-posta ayarlarını yapılandırın
4. Alıcıları ve içeriği belirleyin

## Connection Yapılandırması

Email connection yapılandırmak için:

1. **Connection Management** → **Email SMTP Configuration** bölümüne gidin
2. Yeni connection oluşturun

**SMTP Ayarları:**
* **Host**: SMTP sunucu adresi
* **Port**: SMTP port (25, 465, 587)
* **Username**: Kullanıcı adı
* **Password**: Şifre
* **SSL/TLS**: SSL/TLS ayarları
* **Authentication**: Kimlik doğrulama

## E-posta Gönderme

E-posta göndermek için:

### Temel Ayarlar

* **To**: Alıcı adresi(leri)
* **Cc**: Karbon kopya
* **Bcc**: Gizli karbon kopya
* **Subject**: Konu
* **Body**: İçerik

### İçerik Formatları

**Plain Text:**
```
Body Type: Plain Text
Content: Merhaba ${userName}, hoş geldiniz!
```

**HTML:**
```
Body Type: HTML
Content: <h1>Merhaba ${userName}</h1><p>Hoş geldiniz!</p>
```

### E-posta Şablonları

Şablon kullanımı:

```
Template: welcome-email
Variables:
  userName = ${userName}
  userEmail = ${userEmail}
```

### Ek Dosyalar

Ek dosya ekleme:

```
Attachments:
  - path: /path/to/file.pdf
    name: document.pdf
  - path: ${filePath}
    name: ${fileName}
```

## E-posta Okuma

E-posta okumak için:

### IMAP/POP3 Ayarları

* **Protocol**: IMAP veya POP3
* **Host**: Sunucu adresi
* **Port**: Port numarası
* **Username**: Kullanıcı adı
* **Password**: Şifre

### E-posta Filtreleme

```
Filter:
  From: sender@example.com
  Subject: contains "Important"
  Date: last 7 days
```

### E-posta İşleme

```
Read Emails → Process → Mark as Read → Archive
```

## Variable Kullanımı

Variable'ları e-posta içeriğinde kullanın:

```
To: ${userEmail}
Subject: Hoş geldiniz ${userName}!
Body: Merhaba ${userName}, hesabınız ${accountType} olarak oluşturuldu.
```

## Best Practices

### Güvenlik

* **Authentication**: Güvenli kimlik doğrulama kullanın
* **SSL/TLS**: SSL/TLS şifreleme kullanın
* **Credentials**: Kimlik bilgilerini güvenli saklayın

### Performance

* **Async Sending**: Asenkron gönderim kullanın
* **Batch Sending**: Toplu gönderim yapın
* **Connection Pooling**: Bağlantı havuzlama kullanın

### Reliability

* **Error Handling**: Hata yönetimi yapın
* **Retry Logic**: Yeniden deneme mantığı ekleyin
* **Monitoring**: İzleme yapın

## Örnek Senaryolar

### Senaryo 1: Hoş Geldin E-postası

```
Task: Send Welcome Email
To: ${userEmail}
Subject: Hoş geldiniz!
Body: Merhaba ${userName}, hesabınız başarıyla oluşturuldu.
```

### Senaryo 2: Bildirim E-postası

```
Task: Send Notification Email
To: ${adminEmail}
Subject: Yeni kullanıcı kaydı
Body: ${userName} adlı kullanıcı kayıt oldu.
```

### Senaryo 3: Toplu E-posta

```
Task: Send Bulk Email
To: ${emailList}
Subject: ${emailSubject}
Body: ${emailBody}
Template: newsletter-template
```
