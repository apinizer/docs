Email Connector, Task Flow'larda e-posta göndermenizi sağlar. SMTP sunucularına bağlanarak HTML veya metin tabanlı e-postalar gönderebilir, ekler ekleyebilir ve şablonlar kullanabilirsiniz.

## Genel Bakış

Email Connector sayesinde:

* SMTP sunucularına bağlanabilirsiniz
* HTML veya metin e-postaları gönderebilirsiniz
* E-posta ekleri ekleyebilirsiniz
* E-posta şablonları kullanabilirsiniz
* Toplu e-posta gönderebilirsiniz

## SMTP Bağlantı Yapılandırması

Email Connector kullanmak için önce bir SMTP Connection oluşturmanız gerekir.

### Connection Oluşturma

1. **Connection Management** menüsüne gidin
2. **Email SMTP Configuration** sekmesine tıklayın
3. **New Connection** butonuna tıklayın
4. Bağlantı bilgilerini girin:
   * **Name**: Bağlantı adı
   * **Host**: SMTP sunucu adresi
   * **Port**: Port numarası (varsayılan: 587 veya 465)
   * **Username**: Kullanıcı adı
   * **Password**: Şifre
   * **SSL/TLS**: SSL/TLS kullanımı
   * **Authentication**: Kimlik doğrulama yöntemi
5. **Test Connection** butonuna tıklayın
6. **Save** butonuna tıklayın

## Email Connector Yapılandırması

### Connector Ekleme

1. Task Flow tasarımında **Email Connector**'ı ekleyin
2. Connector özelliklerinde **Connection** seçin
3. E-posta bilgilerini girin:
   * **From**: Gönderen e-posta adresi
   * **To**: Alıcı e-posta adresi(leri)
   * **Cc**: Karbon kopya (opsiyonel)
   * **Bcc**: Gizli karbon kopya (opsiyonel)
   * **Subject**: E-posta konusu
   * **Body**: E-posta içeriği
   * **Content Type**: İçerik türü (text/plain, text/html)

### Basit E-posta Gönderme

```javascript
{
  "from": "noreply@example.com",
  "to": ["user@example.com"],
  "subject": "Hoş Geldiniz",
  "body": "Hoş geldiniz! Hesabınız başarıyla oluşturuldu.",
  "contentType": "text/plain"
}
```

### HTML E-posta Gönderme

```javascript
{
  "from": "noreply@example.com",
  "to": ["user@example.com"],
  "subject": "Hoş Geldiniz",
  "body": "<html><body><h1>Hoş Geldiniz!</h1><p>Hesabınız başarıyla oluşturuldu.</p></body></html>",
  "contentType": "text/html"
}
```

### Dinamik İçerik

Değişkenler kullanarak dinamik içerik:

```javascript
{
  "from": "noreply@example.com",
  "to": ["${input.email}"],
  "subject": "Hoş Geldiniz ${input.name}",
  "body": `
    <html>
      <body>
        <h1>Merhaba ${input.name}!</h1>
        <p>Hesabınız başarıyla oluşturuldu.</p>
        <p>Kullanıcı adınız: ${input.username}</p>
      </body>
    </html>
  `,
  "contentType": "text/html"
}
```

### E-posta Ekleri

Dosya ekleri eklemek için:

```javascript
{
  "from": "noreply@example.com",
  "to": ["user@example.com"],
  "subject": "Rapor",
  "body": "Rapor ektedir.",
  "attachments": [
    {
      "filename": "report.pdf",
      "content": "${variables.reportContent}",
      "contentType": "application/pdf"
    },
    {
      "filename": "data.csv",
      "content": "${variables.csvData}",
      "contentType": "text/csv"
    }
  ]
}
```

### Toplu E-posta Gönderme

Birden fazla alıcıya e-posta göndermek için:

```javascript
{
  "from": "noreply@example.com",
  "to": ["${input.recipients}"], // Array
  "subject": "Toplu Bildirim",
  "body": "Bu bir toplu bildirimdir."
}
```

## E-posta Şablonları

Şablon kullanarak e-posta göndermek için:

### Şablon Tanımlama

1. **Email Templates** menüsüne gidin
2. **New Template** butonuna tıklayın
3. Şablon bilgilerini girin:
   * **Name**: Şablon adı
   * **Subject**: Konu şablonu
   * **Body**: İçerik şablonu
   * **Variables**: Kullanılacak değişkenler

### Şablon Kullanımı

```javascript
{
  "template": "welcome-email",
  "variables": {
    "name": "${input.name}",
    "username": "${input.username}",
    "activationLink": "${variables.activationLink}"
  }
}
```

## Hata Yönetimi

### Retry Mekanizması

E-posta gönderme hatalarında:

* **Retry Configuration**: Retry yapılandırması
* **Max Retries**: Maksimum deneme sayısı
* **Retry Interval**: Denemeler arası bekleme süresi

### Hata Bildirimleri

E-posta gönderme başarısız olduğunda:

* **Error Logging**: Hata loglama
* **Notification**: Bildirim gönderme
* **Fallback**: Alternatif yöntemler

## Kullanım Senaryoları

### Senaryo 1: Kullanıcı Kayıt Bildirimi

```
Database Connector (Kullanıcı kaydı)
├─→ Email Connector (Hoş geldiniz e-postası)
└─→ Logback Connector (Log kaydı)
```

### Senaryo 2: Hata Bildirimi

```
Catch Block (Hata yakalama)
├─→ Email Connector (Hata bildirimi)
└─→ Logback Connector (Hata logu)
```

### Senaryo 3: Rapor Gönderme

```
Database Connector (Veri çekme)
├─→ Script Task (Rapor oluşturma)
└─→ Email Connector (Rapor gönderme)
```
