HTTP/REST Connector, Task Flow'larda HTTP/REST API'lerine çağrı yapmanızı sağlar. GET, POST, PUT, DELETE gibi HTTP metodlarını kullanarak dış API'lerle entegrasyon yapabilirsiniz.

## Genel Bakış

HTTP/REST Connector sayesinde:

* REST API'lerine çağrı yapabilirsiniz
* HTTP metodlarını kullanabilirsiniz (GET, POST, PUT, DELETE, PATCH)
* Header'lar ve authentication yapılandırabilirsiniz
* Request/Response işleme yapabilirsiniz
* Retry ve error handling yapabilirsiniz

## HTTP/REST Connector Yapılandırması

### Connector Ekleme

1. Task Flow tasarımında **HTTP/REST Connector**'ı ekleyin
2. Connector özelliklerinde yapılandırma yapın:
   * **URL**: API endpoint URL'i
   * **Method**: HTTP metodu (GET, POST, PUT, DELETE, PATCH)
   * **Headers**: HTTP header'ları
   * **Body**: Request body (POST, PUT için)
   * **Authentication**: Kimlik doğrulama

### GET İsteği

GET isteği yapmak için:

```javascript
{
  "url": "https://api.example.com/users/${input.userId}",
  "method": "GET",
  "headers": {
    "Accept": "application/json",
    "Authorization": "Bearer ${variables.token}"
  }
}
```

### POST İsteği

POST isteği yapmak için:

```javascript
{
  "url": "https://api.example.com/users",
  "method": "POST",
  "headers": {
    "Content-Type": "application/json",
    "Authorization": "Bearer ${variables.token}"
  },
  "body": JSON.stringify({
    name: "${input.name}",
    email: "${input.email}",
    status: "active"
  })
}
```

### PUT İsteği

PUT isteği yapmak için:

```javascript
{
  "url": "https://api.example.com/users/${input.userId}",
  "method": "PUT",
  "headers": {
    "Content-Type": "application/json",
    "Authorization": "Bearer ${variables.token}"
  },
  "body": JSON.stringify({
    name: "${input.name}",
    email: "${input.email}"
  })
}
```

### DELETE İsteği

DELETE isteği yapmak için:

```javascript
{
  "url": "https://api.example.com/users/${input.userId}",
  "method": "DELETE",
  "headers": {
    "Authorization": "Bearer ${variables.token}"
  }
}
```

## Authentication

### Basic Authentication

```javascript
{
  "authentication": {
    "type": "Basic",
    "username": "${variables.apiUsername}",
    "password": "${variables.apiPassword}"
  }
}
```

### Bearer Token

```javascript
{
  "authentication": {
    "type": "Bearer",
    "token": "${variables.accessToken}"
  }
}
```

### OAuth 2.0

```javascript
{
  "authentication": {
    "type": "OAuth2",
    "tokenUrl": "https://auth.example.com/oauth/token",
    "clientId": "${variables.clientId}",
    "clientSecret": "${variables.clientSecret}",
    "grantType": "client_credentials"
  }
}
```

## Header Yönetimi

### Custom Headers

```javascript
{
  "headers": {
    "X-API-Key": "${variables.apiKey}",
    "X-Request-ID": "${variables.requestId}",
    "X-Correlation-ID": "${variables.correlationId}",
    "Content-Type": "application/json",
    "Accept": "application/json"
  }
}
```

## Request/Response İşleme

### Request Body Mapping

```javascript
{
  "body": {
    "user": {
      "name": "${input.name}",
      "email": "${input.email}"
    },
    "metadata": {
      "source": "taskflow",
      "timestamp": new Date().toISOString()
    }
  }
}
```

### Response Handling

Response'u işlemek için:

```javascript
// Response'u değişkene kaydet
variables.set("apiResponse", response.body);

// Response'dan veri çıkar
var userId = response.body.id;
var status = response.body.status;

// Status code kontrolü
if (response.statusCode === 200) {
    // Başarılı
} else {
    // Hata
}
```

## Hata Yönetimi

### Retry Mekanizması

HTTP hatalarında retry:

```yaml
Retry Configuration:
  Enabled: true
  Max Retries: 3
  Retry On:
    - 5xx Server Errors
    - Connection Errors
    - Timeout Errors
  Strategy: Exponential Backoff
```

### Error Handling

```javascript
try {
    var response = httpConnector.execute();
    if (response.statusCode >= 400) {
        throw new Error(`HTTP Error: ${response.statusCode}`);
    }
} catch (error) {
    logger.error("HTTP request failed", error);
    // Fallback logic
}
```

## Timeout Yapılandırması

```javascript
{
  "timeout": {
    "connectionTimeout": 5000,  // 5 saniye
    "readTimeout": 30000        // 30 saniye
  }
}
```

## Kullanım Senaryoları

### Senaryo 1: API Entegrasyonu

```
HTTP Connector (External API çağrısı)
├─→ Script Task (Response işleme)
└─→ Database Connector (Veri kaydetme)
```

### Senaryo 2: Webhook Gönderme

```
Task Completion
├─→ HTTP Connector (Webhook POST)
└─→ Logback Connector (Log kaydı)
```

### Senaryo 3: API Chain

```
HTTP Connector (API 1)
├─→ Script Task (Veri işleme)
└─→ HTTP Connector (API 2)
    └─→ Database Connector (Sonuç kaydetme)
```
