Webhook Connector, dış sistemlere webhook göndermenizi sağlar. Webhook'lar, dış sistemleri event'ler hakkında bilgilendiren HTTP callback'leridir.

## Genel Bakış

Webhook Connector, dış sistemlere webhook bildirimleri gönderme yetenekleri sağlar ve event-driven entegrasyon desenlerini etkinleştirir.

## Yapılandırma

### Webhook URL

Webhook endpoint'ini yapılandırın:

**Yapılandırma:**

* **URL**: Webhook endpoint URL'i
* **Method**: HTTP metodu (POST, PUT, vb.)
* **Headers**: Özel başlıklar
* **Payload**: Webhook payload'ı

**Örnek:**

```json
{
  "url": "https://api.example.com/webhooks/order-created",
  "method": "POST",
  "headers": {
    "Content-Type": "application/json",
    "X-Webhook-Signature": "${signature}"
  }
}
```

### Payload Yapılandırması

Webhook payload'ını yapılandırın:

**JSON Payload:**

```json
{
  "payload": {
    "event": "order.created",
    "timestamp": "${timestamp}",
    "data": {
      "orderId": "${orderId}",
      "customerId": "${customerId}",
      "total": "${orderTotal}"
    }
  }
}
```

### Kimlik Doğrulama

Webhook kimlik doğrulamasını yapılandırın:

**İmza:**

```javascript
const payload = JSON.stringify(webhookPayload);
const signature = generateSignature(payload, secret);
```

**Bearer Token:**

```json
{
  "headers": {
    "Authorization": "Bearer ${webhookToken}"
  }
}
```

## Hata Yönetimi

### Retry Mantığı

Başarısızlık durumunda retry yapılandırın:

**Retry Yapılandırması:**

* **Max Retries**: Maksimum retry denemeleri
* **Retry Interval**: Retry'lar arasındaki süre
* **Retry Strategy**: Retry stratejisi
