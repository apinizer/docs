FTP List Connector, Task Flow'larda FTP ve SFTP protokolleri ile dosya listeleme işlemleri yapmanızı sağlayan konnektördür.

## Genel Bakış

FTP List Connector şu özellikleri sunar:

* FTP ve SFTP protokolleri
* Dosya listeleme
* Wildcard pattern desteği
* Recursive dizin tarama
* Güvenli bağlantılar (SFTP)

## FTP List Connector Kullanımı

FTP List Connector'ı Task Flow'da kullanmak için:

1. Task Flow tasarım ekranında **FTP Task** ekleyin
2. FTP connection'ı seçin
3. İşlem tipini **List** olarak belirleyin
4. Dizin yolunu ve pattern'i belirleyin

## Connection Yapılandırması

FTP connection yapılandırmak için:

1. **Connection Management** → **FTP/SFTP Configuration** bölümüne gidin
2. Yeni connection oluşturun

**FTP Ayarları:**
* **Protocol**: FTP veya SFTP
* **Host**: Sunucu adresi
* **Port**: Port (21 for FTP, 22 for SFTP)
* **Username**: Kullanıcı adı
* **Password**: Şifre
* **Passive Mode**: Pasif mod
* **SSL/TLS**: SSL/TLS ayarları (SFTP için)

## Dosya Listeleme Ayarları

### Ayarlar

* **Directory Path**: Dizin yolu
* **Pattern**: Dosya deseni (wildcard) - örn: *.csv, data_*.txt
* **Recursive**: Alt dizinleri dahil et (Yes/No)

### Örnek

```
Directory Path: /data
Pattern: *.csv
Recursive: No
Output: ${fileList}
```

### Pattern Örnekleri

* `*.csv` - Tüm CSV dosyaları
* `data_*.txt` - data_ ile başlayan TXT dosyaları
* `*.log` - Tüm LOG dosyaları
* `report_2024*.pdf` - 2024 içeren PDF dosyaları

## Variable Kullanımı

Variable'ları dizin yollarında ve pattern'lerde kullanın:

```
Directory Path: /data/${date}/files
Pattern: ${filePrefix}_*.csv
```

## Best Practices

### Güvenlik

* **SFTP**: Mümkünse SFTP kullanın
* **Credentials**: Kimlik bilgilerini güvenli saklayın
* **Permissions**: Dizin izinlerini kontrol edin

### Performance

* **Connection Pooling**: Bağlantı havuzlama kullanın
* **Pattern Optimization**: Mümkün olduğunca spesifik pattern kullanın
* **Recursive**: Gereksiz recursive taramadan kaçının

### Reliability

* **Error Handling**: Hata yönetimi yapın
* **Retry Logic**: Yeniden deneme mantığı ekleyin
* **Timeout**: Uzun süren işlemler için timeout ayarlayın

## Örnek Senaryolar

### Senaryo 1: CSV Dosyalarını Listeleme

```
Task: List CSV Files
Connector: FTP List
Directory Path: /data
Pattern: *.csv
Recursive: No
Output: ${fileList}
```

### Senaryo 2: Recursive Dosya Arama

```
Task: Find All Log Files
Connector: FTP List
Directory Path: /logs
Pattern: *.log
Recursive: Yes
Output: ${logFiles}
```

### Senaryo 3: Tarih Bazlı Dosya Arama

```
Task: List Today's Reports
Connector: FTP List
Directory Path: /reports/${today}
Pattern: report_*.pdf
Recursive: No
Output: ${todayReports}
```
