## Konfigürasyon

Linux Script ayarlarını içeren görsele aşağıda yer verilmiştir:

<img
    src="/images/yonetici/konnektor/linux-script.png"
    alt="Linux Script Connector Konfigürasyonu"
    width="600"
    className="mr-auto"
/>

Linux Script konfigürasyonu için kullanılan alanlar aşağıdaki tabloda görülmektedir.

| Alan | Açıklama |
|------|----------|
| **Linux Script Bağlantı Yapılandırılması** (Linux Script Connection Configuration) | Linux Script bağlantısına ait tanım, havuzdan seçilir ya da eklenebilir. Bağlantı oluşturmak hakkında detaylı bilgi için [tıklayınız](/tr/entegrasyon-otomasyon/baglanti-yonetimi/linux-connections). |
| **Script** | Eylemde çalıştırılacak Linux bash komutları yazılır. |

<Info>
Script alanında değişkenler kullanılabilir.
</Info>
