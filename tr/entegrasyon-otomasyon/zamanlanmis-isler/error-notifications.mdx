Zamanlanmış işlerde hata bildirimleri, job başarısız olduğunda veya belirli koşullar sağlandığında bildirim göndermenizi sağlar. Bu sayfa, hata bildirimi yapılandırmasını detaylı olarak açıklar.

## Genel Bakış

Error notifications sayesinde:

* Job başarısız olduğunda bildirim alabilirsiniz
* Belirli koşullarda uyarı alabilirsiniz
* Çoklu bildirim kanalları kullanabilirsiniz
* Bildirim şablonları kullanabilirsiniz

## Notification Configuration

### Basic Configuration

Temel yapılandırma:

1. **Job Definition** sayfasında ilgili job'ı seçin
2. **Error Notifications** sekmesine gidin
3. **Enable Notifications** seçeneğini aktif edin
4. Bildirim kanallarını yapılandırın

### Email Notification

E-posta bildirimi:

```yaml
Error Notifications:
  Enabled: true
  Channels:
    - Type: Email
      Recipients:
        - admin@example.com
        - team@example.com
      Subject: Job Failed: ${job.name}
      Template: job-error-template
      Include:
        - Error Message
        - Stack Trace
        - Execution Details
```

### Webhook Notification

Webhook bildirimi:

```yaml
Error Notifications:
  Enabled: true
  Channels:
    - Type: Webhook
      URL: https://api.example.com/webhooks/job-error
      Method: POST
      Headers:
        Authorization: Bearer ${token}
        Content-Type: application/json
      Body:
        jobName: ${job.name}
        errorMessage: ${error.message}
        executionTime: ${execution.time}
```

### Notification Connector

Notification Connector kullanımı:

```yaml
Error Notifications:
  Enabled: true
  Channels:
    - Type: Notification Connector
      Connector: notification-connector-1
      Channel: email
      Recipient: ${adminEmail}
      Priority: high
```

### SNMP Notification

SNMP bildirimi:

```yaml
Error Notifications:
  Enabled: true
  Channels:
    - Type: SNMP
      Host: ${snmpHost}
      Port: 162
      TrapOID: 1.3.6.1.4.1.12345.1.0.1
      Variables:
        - oid: 1.3.6.1.4.1.12345.1.1.1
          value: ${job.name}
        - oid: 1.3.6.1.4.1.12345.1.1.2
          value: ${error.message}
```

## Notification Conditions

### On Failure

Başarısızlık durumunda:

* **Any Failure**: Herhangi bir başarısızlık
* **Specific Error**: Belirli hata türü
* **Error Pattern**: Hata deseni
* **Error Count**: Hata sayısı

### On Success

Başarı durumunda:

* **Always**: Her zaman
* **After Failure**: Başarısızlıktan sonra
* **Custom Condition**: Özel koşul

### On Timeout

Zaman aşımı durumunda:

* **Job Timeout**: Job zaman aşımı
* **Task Timeout**: Task zaman aşımı
* **Connection Timeout**: Bağlantı zaman aşımı

### Custom Conditions

Özel koşullar:

* **Duration Threshold**: Süre eşiği
* **Error Rate Threshold**: Hata oranı eşiği
* **Custom Expression**: Özel ifade

## Notification Templates

### Template Definition

Şablon tanımlama:

```yaml
Template: job-error-template
Subject: Job Failed: ${job.name}
Body: |
  Job Execution Failed
  
  Job Name: ${job.name}
  Execution Time: ${execution.time}
  Duration: ${execution.duration}ms
  Error Message: ${error.message}
  Stack Trace:
  ${error.stackTrace}
  
  Execution Details:
  ${execution.details}
```

### Template Variables

Şablon değişkenleri:

* **${job.name}**: Job adı
* **${job.type}**: Job tipi
* **${execution.time}**: Çalıştırma zamanı
* **${execution.duration}**: Çalıştırma süresi
* **${error.message}**: Hata mesajı
* **${error.stackTrace}**: Stack trace
* **${execution.details}**: Çalıştırma detayları

## Multiple Channels

### Channel Priority

Kanal önceliği:

* **Primary Channel**: Birincil kanal
* **Fallback Channel**: Yedek kanal
* **Parallel Channels**: Paralel kanallar

### Channel Selection

Kanal seçimi:

* **Condition-based**: Koşul tabanlı
* **Error-type-based**: Hata tipi tabanlı
* **Severity-based**: Önem derecesi tabanlı

## Notification Frequency

### Rate Limiting

Hız sınırlama:

* **Max Notifications per Hour**: Saatte maksimum bildirim sayısı
* **Notification Cooldown**: Bildirim bekleme süresi
* **Deduplication**: Tekrarları önleme

### Notification Aggregation

Bildirim toplama:

* **Batch Notifications**: Toplu bildirimler
* **Summary Notifications**: Özet bildirimler
* **Digest Notifications**: Özet bildirimler
