Zamanlanmış işlerde hata bildirimleri, job başarısız olduğunda veya belirli koşullar sağlandığında bildirim göndermenizi sağlar.

## Genel Bakış

Error notifications sayesinde:

* Job başarısız olduğunda bildirim alabilirsiniz
* Belirli koşullarda uyarı alabilirsiniz
* Çoklu bildirim kanalları kullanabilirsiniz
* Bildirim şablonları kullanabilirsiniz

## Bildirim Yapılandırması

### Email Bildirimi

```yaml
Error Notifications:
  Enabled: true
  Channels:
    - Type: Email
      Recipients:
        - admin@example.com
        - team@example.com
      Subject: Job Failed: ${job.name}
      Template: job-error-template
```

### Webhook Bildirimi

```yaml
Error Notifications:
  Enabled: true
  Channels:
    - Type: Webhook
      URL: https://api.example.com/webhooks/job-error
      Method: POST
      Headers:
        Authorization: Bearer ${token}
```

## Bildirim Koşulları

* **On Failure**: Başarısızlık durumunda
* **On Success**: Başarı durumunda
* **On Timeout**: Zaman aşımı durumunda
* **Custom Conditions**: Özel koşullar

## Bildirim Şablonları

Bildirim şablonları kullanarak özelleştirilmiş bildirimler gönderebilirsiniz.
