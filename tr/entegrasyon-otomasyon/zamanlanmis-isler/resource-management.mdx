Zamanlanmış işlerde kaynak yönetimi, job'ların sistem kaynaklarını verimli kullanmasını sağlar.

## Genel Bakış

Resource management sayesinde:

* CPU ve memory kullanımını kontrol edebilirsiniz
* Eşzamanlılık (concurrency) ayarlayabilirsiniz
* Timeout ayarları yapabilirsiniz
* Kaynak limitleri belirleyebilirsiniz

## CPU Management

CPU kullanımı kontrolü:

* **CPU Limit**: CPU limiti
* **CPU Priority**: CPU önceliği
* **CPU Affinity**: CPU affinity

## Memory Management

Bellek yönetimi:

* **Memory Limit**: Bellek limiti
* **Memory Allocation**: Bellek tahsisi
* **Garbage Collection**: Çöp toplama

## Concurrency

Eşzamanlılık kontrolü:

* **Max Concurrent Jobs**: Maksimum eşzamanlı job sayısı
* **Job Queue**: Job kuyruğu
* **Priority Queue**: Öncelik kuyruğu

## Timeout

Zaman aşımı ayarları:

* **Job Timeout**: Job zaman aşımı
* **Task Timeout**: Task zaman aşımı
* **Connection Timeout**: Bağlantı zaman aşımı
