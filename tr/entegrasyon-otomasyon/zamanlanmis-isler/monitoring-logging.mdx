Zamanlanmış işlerin izlenmesi ve loglanması, job'ların sağlığını ve performansını anlamak için kritik öneme sahiptir. Bu sayfa, job izleme ve loglama özelliklerini detaylı olarak açıklar.

## Genel Bakış

Monitoring ve logging sayesinde:

* Job çalıştırma geçmişini görüntüleyebilirsiniz
* Performans metriklerini izleyebilirsiniz
* Hata durumlarını analiz edebilirsiniz
* Log kayıtlarını inceleyebilirsiniz

## Execution History

### Viewing Execution History

Job çalıştırma geçmişini görüntüleme:

1. **Scheduled Jobs** sayfasına gidin
2. İlgili job'ı seçin
3. **Execution History** sekmesine tıklayın
4. Çalıştırma geçmişi listelenir

### Execution Details

Çalıştırma detayları:

* **Execution Time**: Çalıştırma zamanı
* **Duration**: Süre (milisaniye)
* **Status**: Durum (Success, Failed, Running, Cancelled)
* **Output**: Çıktı (varsa)
* **Error**: Hata mesajı (varsa)
* **Logs**: Log kayıtları

### Execution Filtering

Çalıştırma filtreleme:

* **Date Range**: Tarih aralığı
* **Status Filter**: Durum filtresi
* **Search**: Arama

### Execution Export

Çalıştırma export:

* **CSV Export**: CSV formatında export
* **JSON Export**: JSON formatında export
* **Excel Export**: Excel formatında export

## Performance Metrics

### Key Metrics

Anahtar metrikler:

* **Average Duration**: Ortalama süre
* **Success Rate**: Başarı oranı (%)
* **Failure Rate**: Başarısızlık oranı (%)
* **Throughput**: İşlem hızı (job/saat)
* **Peak Duration**: En uzun süre
* **Min Duration**: En kısa süre

### Metric Visualization

Metrik görselleştirme:

* **Line Charts**: Çizgi grafikleri
* **Bar Charts**: Çubuk grafikleri
* **Pie Charts**: Pasta grafikleri
* **Trend Analysis**: Trend analizi

### Performance Trends

Performans trendleri:

* **Duration Trends**: Süre trendleri
* **Success Rate Trends**: Başarı oranı trendleri
* **Failure Rate Trends**: Başarısızlık oranı trendleri
* **Throughput Trends**: İşlem hızı trendleri

## Log Management

### Log Levels

Log seviyeleri:

* **TRACE**: En detaylı log seviyesi
* **DEBUG**: Debug bilgileri
* **INFO**: Bilgilendirme mesajları
* **WARN**: Uyarı mesajları
* **ERROR**: Hata mesajları

### Log Viewing

Log görüntüleme:

* **Real-time Logs**: Gerçek zamanlı loglar
* **Historical Logs**: Geçmiş loglar
* **Filtered Logs**: Filtrelenmiş loglar
* **Search Logs**: Log arama

### Log Filtering

Log filtreleme:

* **By Level**: Log seviyesine göre
* **By Date**: Tarihe göre
* **By Keyword**: Anahtar kelimeye göre
* **By Execution**: Çalıştırmaya göre

### Log Export

Log export:

* **CSV Export**: CSV formatında export
* **JSON Export**: JSON formatında export
* **Text Export**: Metin formatında export

### Log Retention

Log saklama:

* **Retention Period**: Saklama süresi
* **Log Rotation**: Log rotasyonu
* **Log Archival**: Log arşivleme
* **Log Compression**: Log sıkıştırma

## Error Analysis

### Error Types

Hata türleri:

* **Connection Errors**: Bağlantı hataları
* **Timeout Errors**: Zaman aşımı hataları
* **Validation Errors**: Doğrulama hataları
* **Execution Errors**: Çalıştırma hataları
* **System Errors**: Sistem hataları

### Error Frequency

Hata sıklığı:

* **Error Count**: Hata sayısı
* **Error Rate**: Hata oranı
* **Error Distribution**: Hata dağılımı
* **Error Trends**: Hata trendleri

### Error Details

Hata detayları:

* **Error Message**: Hata mesajı
* **Error Stack**: Hata stack trace
* **Error Context**: Hata bağlamı
* **Error Time**: Hata zamanı

### Root Cause Analysis

Kök neden analizi:

* **Error Patterns**: Hata desenleri
* **Error Correlation**: Hata korelasyonu
* **Error Impact**: Hata etkisi
* **Error Resolution**: Hata çözümü
