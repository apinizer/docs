Zamanlanmış işler farklı türlerde olabilir. Bu sayfada API Call, Script ve Task Flow job türlerini öğreneceksiniz.

## Genel Bakış

Apinizer'da üç temel job türü desteklenir:

* **API Call Job**: HTTP/REST API çağrıları
* **Script Job**: JavaScript/Groovy script'leri
* **Task Flow Job**: Task Flow çalıştırma

## API Call Job

API endpoint'ine çağrı yapan job türüdür.

### Yapılandırma

```yaml
Job Type: API Call
Configuration:
  URL: https://api.example.com/sync
  Method: POST
  Headers:
    Content-Type: application/json
    Authorization: Bearer ${token}
  Body:
    action: sync
    timestamp: ${currentTimestamp}
  Timeout: 30000
  Retry:
    Enabled: true
    Max Retries: 3
```

### Kullanım Senaryoları

* **Data Synchronization**: Veri senkronizasyonu
* **Health Checks**: Sağlık kontrolü
* **Webhook Triggers**: Webhook tetikleme
* **API Monitoring**: API izleme

## Script Job

JavaScript veya Groovy script çalıştıran job türüdür.

### Yapılandırma

```yaml
Job Type: Script
Configuration:
  Language: JavaScript
  Script: |
    // Veri çekme
    var data = fetchData();
    
    // Veri işleme
    var processed = processData(data);
    
    // Sonuç kaydetme
    saveResults(processed);
  Input Variables:
    source: database
    batchSize: 100
```

### Kullanım Senaryoları

* **Data Processing**: Veri işleme
* **Calculations**: Hesaplamalar
* **Validations**: Doğrulamalar
* **Custom Logic**: Özel mantık

## Task Flow Job

Task Flow çalıştıran job türüdür.

### Yapılandırma

```yaml
Job Type: Task Flow
Configuration:
  Task Flow: data-sync-taskflow
  Input Parameters:
    source: database
    target: api
    batchSize: 100
  Timeout: 600000
```

### Kullanım Senaryoları

* **Complex Workflows**: Karmaşık iş akışları
* **Multi-Step Processes**: Çok adımlı işlemler
* **Integration Scenarios**: Entegrasyon senaryoları
* **Data Pipelines**: Veri pipeline'ları

## Job Türü Seçimi

### Ne Zaman API Call Job?

* Basit API çağrıları için
* Dış sistem entegrasyonları için
* Webhook tetikleme için

### Ne Zaman Script Job?

* Veri işleme için
* Hesaplamalar için
* Özel mantık için

### Ne Zaman Task Flow Job?

* Karmaşık iş akışları için
* Çok adımlı işlemler için
* Entegrasyon senaryoları için
