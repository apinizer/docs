Cron Expressions, zamanlanmış işlerin ne zaman çalışacağını belirlemek için kullanılan ifadelerdir. Standart cron syntax'ını kullanarak karmaşık zamanlama desenleri tanımlayabilirsiniz.

## Genel Bakış

Cron Expression formatı:

```
┌───────────── saniye (0-59)
│ ┌─────────── dakika (0-59)
│ │ ┌───────── saat (0-23)
│ │ │ ┌─────── gün (1-31)
│ │ │ │ ┌───── ay (1-12 veya JAN-DEC)
│ │ │ │ │ ┌─── haftanın günü (0-7 veya SUN-SAT)
│ │ │ │ │ │
* * * * * * *
```

## Cron Expression Format

### Standard Format

Standart cron formatı (6 veya 7 alan):

**6 Alan Formatı (Saniye dahil):**
```
saniye dakika saat gün ay haftanın-günü
```

**7 Alan Formatı (Yıl dahil):**
```
saniye dakika saat gün ay haftanın-günü yıl
```

### Field Values

Alan değerleri:

* **Saniye (Seconds)**: 0-59
* **Dakika (Minutes)**: 0-59
* **Saat (Hours)**: 0-23
* **Gün (Day of Month)**: 1-31
* **Ay (Month)**: 1-12 veya JAN-DEC
* **Haftanın Günü (Day of Week)**: 0-7 veya SUN-SAT (0 ve 7 = Pazar)
* **Yıl (Year)**: 1970-2099 (opsiyonel)

## Special Characters

### Asterisk (*)

Tüm değerler:

```
* * * * * *  → Her saniye
0 * * * * *  → Her dakikanın başında
0 0 * * * *  → Her saatin başında
```

### Comma (,)

Değer listesi:

```
0 0 9,12,15 * * *  → Saat 09:00, 12:00 ve 15:00'de
0 0 * * * 1,3,5    → Pazartesi, Çarşamba ve Cuma günleri
```

### Hyphen (-)

Değer aralığı:

```
0 0 9-17 * * *  → Saat 09:00 ile 17:00 arası her saatte
0 0 * * 1-5 *   → Pazartesi-Cuma arası her gün
```

### Slash (/)

Adım değeri:

```
0 */15 * * * *  → Her 15 dakikada bir
0 0 */2 * * *   → Her 2 saatte bir
0 0 0 */7 * *   → Her 7 günde bir
```

### Question Mark (?)

Belirsiz değer (sadece gün ve haftanın günü için):

```
0 0 0 ? * *     → Her gün saat 00:00'da
0 0 0 * * ?     → Her gün saat 00:00'da
```

### Hash (#)

Haftanın günü için n. gün:

```
0 0 0 ? * MON#2  → Her ayın ikinci Pazartesi günü
0 0 0 ? * FRI#1  → Her ayın ilk Cuma günü
```

### L

Son değer:

```
0 0 0 L * *     → Ayın son günü
0 0 0 ? * L     → Haftanın son günü (Cumartesi)
0 0 0 ? * 5L    → Ayın son Cuma günü
```

### W

En yakın iş günü:

```
0 0 0 15W * *   → Ayın 15'i en yakın iş gününe
```

## Common Cron Examples

### Every Minute

```
0 * * * * *     → Her dakikanın başında
```

### Every Hour

```
0 0 * * * *     → Her saatin başında
```

### Daily

```
0 0 0 * * *     → Her gün saat 00:00'da
0 0 9 * * *     → Her gün saat 09:00'da
```

### Weekly

```
0 0 0 * * MON   → Her Pazartesi saat 00:00'da
0 0 9 * * FRI   → Her Cuma saat 09:00'da
```

### Monthly

```
0 0 0 1 * *     → Her ayın 1'inde saat 00:00'da
0 0 0 15 * *    → Her ayın 15'inde saat 00:00'da
```

### Yearly

```
0 0 0 1 1 *     → Her yılın 1 Ocak'ında saat 00:00'da
```

### Business Days

```
0 0 9 * * MON-FRI  → Hafta içi her gün saat 09:00'da
```

### Every N Minutes

```
0 */5 * * * *   → Her 5 dakikada bir
0 */15 * * * *  → Her 15 dakikada bir
0 */30 * * * *  → Her 30 dakikada bir
```

### Every N Hours

```
0 0 */2 * * *   → Her 2 saatte bir
0 0 */6 * * *   → Her 6 saatte bir
0 0 */12 * * *  → Her 12 saatte bir
```

### Specific Times

```
0 0 9 * * *     → Her gün saat 09:00'da
0 30 14 * * *   → Her gün saat 14:30'da
0 0 9,12,15 * * *  → Her gün saat 09:00, 12:00 ve 15:00'de
```

## Advanced Examples

### Every 15 Minutes During Business Hours

```
0 */15 9-17 * * MON-FRI  → Hafta içi 09:00-17:00 arası her 15 dakikada bir
```

### First Monday of Every Month

```
0 0 0 ? * MON#1  → Her ayın ilk Pazartesi günü saat 00:00'da
```

### Last Day of Every Month

```
0 0 0 L * *      → Her ayın son günü saat 00:00'da
```

### Every 30 Seconds

```
*/30 * * * * *   → Her 30 saniyede bir
```

### Complex Schedule

```
0 0 9-17 * * MON-FRI  → Hafta içi 09:00-17:00 arası her saatte
```

## Time Zone Support

### Time Zone Configuration

Zaman dilimi yapılandırması:

* **Default Time Zone**: Varsayılan zaman dilimi
* **Job Time Zone**: İş zaman dilimi
* **Daylight Saving**: Yaz saati desteği

**Örnek:**
```
Cron: 0 0 9 * * *
Time Zone: Europe/Istanbul
→ Türkiye saati ile her gün saat 09:00'da
```
