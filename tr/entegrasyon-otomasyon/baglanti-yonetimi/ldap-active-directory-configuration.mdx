LDAP/Active Directory Configuration, API Integrator'da LDAP ve Active Directory bağlantılarını yönetmenizi sağlar.

## Genel Bakış

LDAP/Active Directory Configuration şu özellikleri sunar:

* LDAP ve Active Directory desteği
* Kullanıcı doğrulama
* Grup yönetimi
* Yetkilendirme

## LDAP/AD Bağlantısı Oluşturma

Yeni bir LDAP/AD bağlantısı oluşturmak için:

1. **Connection Management** → **LDAP/Active Directory Configuration** bölümüne gidin
2. **New Connection** butonuna tıklayın
3. LDAP/AD ayarlarını yapılandırın

## Bağlantı Ayarları

### Temel Ayarlar

* **Connection Name**: Bağlantı adı
* **Host**: LDAP/AD sunucu adresi
* **Port**: Port (389 for LDAP, 636 for LDAPS)
* **Base DN**: Base distinguished name
* **Username**: Kullanıcı adı (bind DN)
* **Password**: Şifre

### Güvenlik Ayarları

* **SSL/TLS**: SSL/TLS ayarları
* **Authentication Method**: Kimlik doğrulama yöntemi

## Best Practices

* **SSL/TLS**: Güvenli bağlantı kullanın
* **Credentials**: Kimlik bilgilerini güvenli saklayın
* **Testing**: Bağlantıyı test edin
