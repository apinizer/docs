## LDAP Connection

### Connection Configuration

LDAP bağlantısı oluşturma:

1. **Connection Management** sayfasına gidin
2. **LDAP/Active Directory Configuration** sekmesine tıklayın
3. **New Connection** butonuna tıklayın
4. **LDAP** tipini seçin
5. Bağlantı bilgilerini girin

### Basic Configuration

Temel yapılandırma:

```yaml
Name: ldap-production
Host: ldap.example.com
Port: 389
Base DN: dc=example,dc=com
Bind DN: cn=admin,dc=example,dc=com
Password: ${ldapPassword} (Secret Manager)
SSL: false
```

### SSL/TLS Configuration

SSL/TLS yapılandırması:

```yaml
SSL: true
Port: 636
TLS: true
Certificate Validation: true
```

### LDAP Search Configuration

LDAP arama yapılandırması:

```yaml
Search Base: ou=users,dc=example,dc=com
Search Filter: (objectClass=person)
Search Scope: SUBTREE
Attributes: cn,mail,uid
```

## Active Directory Connection

### Connection Configuration

Active Directory bağlantısı oluşturma:

1. **Connection Management** sayfasına gidin
2. **LDAP/Active Directory Configuration** sekmesine tıklayın
3. **New Connection** butonuna tıklayın
4. **Active Directory** tipini seçin
5. Bağlantı bilgilerini girin

### Basic Configuration

Temel yapılandırma:

```yaml
Name: ad-production
Host: ad.example.com
Port: 389
Base DN: dc=example,dc=com
Domain: example.com
Username: administrator
Password: ${adPassword} (Secret Manager)
SSL: false
```

### Active Directory Specific Configuration

Active Directory özel yapılandırması:

```yaml
Domain Controller: ad.example.com
Global Catalog: gc://ad.example.com:3268
Site: Default-First-Site-Name
```

### Pool Configuration

Havuz yapılandırması:

```yaml

  Min Size: 5
  Max Size: 20
  Connection Timeout: 30000ms
```
