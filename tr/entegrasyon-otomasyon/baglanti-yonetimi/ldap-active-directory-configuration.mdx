LDAP/Active Directory Configuration, LDAP ve Active Directory sunucularına bağlantı oluşturmanızı sağlar.

## Genel Bakış

LDAP/Active Directory Configuration sayesinde:

* LDAP ve Active Directory sunucularına bağlantı oluşturabilirsiniz
* Authentication yapılandırabilirsiniz
* Directory servisleri kullanabilirsiniz

## Bağlantı Oluşturma

### LDAP Yapılandırması

```yaml
Name: ldap-connection
Host: ldap.example.com
Port: 389
Base DN: dc=example,dc=com
Username: cn=admin,dc=example,dc=com
Password: password
SSL: false
```

### Active Directory Yapılandırması

```yaml
Name: ad-connection
Host: ad.example.com
Port: 389
Base DN: dc=example,dc=com
Domain: example.com
Username: administrator
Password: password
SSL: false
```
