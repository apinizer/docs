Bu sayfada RabbitMQ bağlantı ayarlarının nasıl yapıldığı anlatılmaktadır.

**RabbitMQ ayarlarını içeren görsele aşağıda yer verilmiştir:**

![RabbitMQ Queue Connections](/images/rabbitmq-queue-connections.png)

RabbitMQ konfigürasyonu için kullanılan alanlar aşağıdaki tabloda görülmektedir.

| Alan | Açıklama |
|------|----------|
| **Aktif** _(Enabled)_ | Bağlantının aktiflik durumu. |
| **Adı** _(Name)_ | Yapılacak ayarın ad bilgisidir. |
| **Açıklama** _(Description)_ | Açıklama girilebilir. |
| **Host** | Bağlantılar için kullanılacak sunucu |
| **Virtual Host** | Aracıya bağlanırken kullanılacak sanal sunucu |
| **Port** | Bağlantılar için kullanılacak bağlantı noktası |
| **Authentication** | Kimlik Doğrulamayı yapılandırmak için etkinleştirilir |
| **Username** | Broker'a bağlanırken kullanılacak AMQP kullanıcı adı |
| **Password** | Broker'a bağlanırken kullanılacak parola |
| **Connection Timeout** | Milisaniye cinsinden TCP bağlantı zaman aşımı; sonsuz için sıfır giriniz |
| **Requested Channel Max** | Başlangıçta talep edilen maksimum kanal sayısı; sınırsız olması için sıfır giriniz |
| **Requested Frame Max** | Sekizli olarak başlangıçta talep edilen maksimum çerçeve boyutu; sınırsız olması için sıfır giriniz |
| **Requested Heartbeat** | Saniye cinsinden başlangıçta talep edilen kalp atışı aralığı; hiçbiri için sıfır giriniz |
| **Use SSL Protocol** | TLS'yi yapılandırmayı etkinleştirir |
| **SSL Protocol** | TLS'yi yapılandırma yöntemi. Kullanılacak TLS protokolü sürümünü girin, örn. "TLSv1.2" veya "TLSv1.1" |
| **Channel Exchange** | Mesajı yayınlamak için kullanılacak exchange |
| **Channel Routing Key** | Yönlendirme anahtarı |
| **BasicProperties For Message** | Her isteğin meta verilerini ayarlamak için AMQP Basit Özellikleri girilebilir. |
