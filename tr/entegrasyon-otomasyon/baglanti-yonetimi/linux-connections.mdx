## Connection Configuration

### Creating Linux Connection

Linux bağlantısı oluşturma:

1. **Connection Management** sayfasına gidin
2. **Linux Connections** sekmesine tıklayın
3. **New Connection** butonuna tıklayın
4. Bağlantı bilgilerini girin

### Basic Configuration

Temel yapılandırma:

```yaml
Name: linux-production
Host: server.example.com
Port: 22
Username: app_user
Authentication: Password
Password: ${linuxPassword} (Secret Manager)
Working Directory: /home/app_user
```

### SSH Key Authentication

SSH anahtarı kimlik doğrulama:

```yaml
Authentication: SSH Key
SSH Key: ${sshPrivateKey} (Secret Manager)
SSH Key Passphrase: ${keyPassphrase} (Secret Manager)
Key Type: RSA/ED25519
```

### Advanced Configuration

Gelişmiş yapılandırma:

```yaml
Connection Timeout: 30000ms
Command Timeout: 60000ms
Keep Alive: true
Keep Alive Interval: 30000ms
Compression: false
```

## Security Configuration

### SSH Configuration

SSH yapılandırması:

* **Host Key Verification**: Host anahtarı doğrulama
* **Known Hosts**: Bilinen host'lar
* **Strict Host Key Checking**: Sıkı host anahtarı kontrolü

### Authentication Methods

Kimlik doğrulama yöntemleri:

* **Password**: Şifre tabanlı
* **SSH Key**: SSH anahtarı tabanlı
* **Certificate**: Sertifika tabanlı

## Connection Pooling

### Pool Configuration

Havuz yapılandırması:

```yaml
Connection Pool:
  Min Size: 3
  Max Size: 10
  Connection Timeout: 30000ms
  Idle Timeout: 300000ms
```

## Testing

### Test Connection

Bağlantı testi:

1. Bağlantı yapılandırmasında **Test Connection** butonuna tıklayın
2. Bağlantı testi çalıştırılır
3. Test sonucu görüntülenir

### Test Operations

İşlem testi:

* **Command Execution**: Komut çalıştırma
* **File Operations**: Dosya işlemleri
* **Script Execution**: Script çalıştırma
