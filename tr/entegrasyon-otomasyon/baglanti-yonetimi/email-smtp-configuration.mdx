Email SMTP Configuration, API Integrator'da e-posta gönderme için SMTP bağlantılarını yönetmenizi sağlar.

## Genel Bakış

Email SMTP Configuration şu özellikleri sunar:

* SMTP sunucu yapılandırması
* SSL/TLS desteği
* Authentication
* Port yapılandırması

## SMTP Bağlantısı Oluşturma

Yeni bir SMTP bağlantısı oluşturmak için:

1. **Connection Management** → **Email SMTP Configuration** bölümüne gidin
2. **New Connection** butonuna tıklayın
3. SMTP ayarlarını yapılandırın

## SMTP Ayarları

### Temel Ayarlar

* **Connection Name**: Bağlantı adı
* **Host**: SMTP sunucu adresi
* **Port**: Port (25, 465, 587)
* **Username**: Kullanıcı adı
* **Password**: Şifre

### Güvenlik Ayarları

* **SSL/TLS**: SSL/TLS ayarları
* **StartTLS**: StartTLS desteği
* **Authentication**: Kimlik doğrulama yöntemi

## Port Yapılandırması

* **Port 25**: Standart SMTP (genellikle şifrelenmemiş)
* **Port 465**: SSL/TLS ile SMTP
* **Port 587**: StartTLS ile SMTP

## Best Practices

* **SSL/TLS**: Güvenli bağlantı kullanın
* **Credentials**: Kimlik bilgilerini güvenli saklayın
* **Testing**: Bağlantıyı test edin
