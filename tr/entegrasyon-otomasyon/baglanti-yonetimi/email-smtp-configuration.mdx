Email SMTP Configuration, e-posta göndermek için SMTP sunucu bağlantıları oluşturmanızı sağlar.

## Genel Bakış

Email SMTP Configuration sayesinde:

* SMTP sunucularına bağlantı oluşturabilirsiniz
* SSL/TLS güvenli bağlantılar kullanabilirsiniz
* Authentication yapılandırabilirsiniz

## Bağlantı Oluşturma

### Yapılandırma

```yaml
Name: smtp-connection
Host: smtp.example.com
Port: 587
Username: user@example.com
Password: password
SSL/TLS: STARTTLS
Authentication: PLAIN
```
