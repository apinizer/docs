Database Connections, API Integrator'da veritabanı bağlantılarını yönetmenizi sağlar. Bu sayfa, veritabanı bağlantılarını oluşturma, yapılandırma ve yönetme konularını kapsar.

## Genel Bakış

Database Connections şu özellikleri sunar:

* Çoklu veritabanı desteği
* Connection pooling
* Güvenli kimlik doğrulama
* Connection health checks
* Connection monitoring

## Veritabanı Bağlantısı Oluşturma

Yeni bir veritabanı bağlantısı oluşturmak için:

1. **Connection Management** → **Database Connections** bölümüne gidin
2. **New Connection** butonuna tıklayın
3. Bağlantı bilgilerini girin

## Bağlantı Ayarları

### Temel Ayarlar

* **Connection Name**: Bağlantı adı (unique)
* **Description**: Açıklama
* **Database Type**: Veritabanı türü
* **Host**: Sunucu adresi
* **Port**: Port numarası
* **Database Name**: Veritabanı adı

### Kimlik Doğrulama

* **Username**: Kullanıcı adı
* **Password**: Şifre
* **Authentication Method**: Kimlik doğrulama yöntemi

### Connection Pool Ayarları

* **Min Pool Size**: Minimum pool boyutu
* **Max Pool Size**: Maksimum pool boyutu
* **Connection Timeout**: Bağlantı timeout'u
* **Idle Timeout**: Boşta timeout

## Desteklenen Veritabanları

* MySQL
* PostgreSQL
* Oracle
* SQL Server
* MongoDB
* Diğer JDBC uyumlu veritabanları

## Connection Test

Bağlantıyı test etmek için:

1. Connection listesinden bir connection seçin
2. **Test Connection** butonuna tıklayın
3. Test sonucu görüntülenir

## Best Practices

* **Connection Pooling**: Bağlantı havuzlama kullanın
* **Credentials**: Kimlik bilgilerini güvenli saklayın
* **Health Checks**: Sağlık kontrolleri yapın
* **Monitoring**: İzleme yapın
