## Supported Databases

### MySQL

MySQL veritabanı bağlantısı:

* **Driver**: MySQL Connector/J
* **Default Port**: 3306
* **URL Format**: `jdbc:mysql://host:port/database`

### PostgreSQL

PostgreSQL veritabanı bağlantısı:

* **Driver**: PostgreSQL JDBC Driver
* **Default Port**: 5432
* **URL Format**: `jdbc:postgresql://host:port/database`

### Oracle

Oracle veritabanı bağlantısı:

* **Driver**: Oracle JDBC Driver
* **Default Port**: 1521
* **URL Format**: `jdbc:oracle:thin:@host:port:database`

### Microsoft SQL Server

MSSQL veritabanı bağlantısı:

* **Driver**: Microsoft SQL Server JDBC Driver
* **Default Port**: 1433
* **URL Format**: `jdbc:sqlserver://host:port;databaseName=database`

### IBM DB2

DB2 veritabanı bağlantısı:

* **Driver**: IBM DB2 JDBC Driver
* **Default Port**: 50000
* **URL Format**: `jdbc:db2://host:port/database`

## Connection Configuration

### Basic Configuration

Temel yapılandırma:

* **Connection Name**: Bağlantı adı
* **Database Type**: Veritabanı tipi
* **Host**: Sunucu adresi
* **Port**: Port numarası
* **Database Name**: Veritabanı adı
* **Username**: Kullanıcı adı
* **Password**: Şifre (Secret Manager'da saklanır)

### Advanced Configuration

Gelişmiş yapılandırma:

* **Connection URL**: Özel bağlantı URL'i
* **Additional Parameters**: Ek parametreler
* **Schema**: Şema adı
* **Timezone**: Zaman dilimi

### Connection Pool Configuration

Bağlantı havuzu yapılandırması:

* **Min Pool Size**: Minimum havuz boyutu (varsayılan: 5)
* **Max Pool Size**: Maksimum havuz boyutu (varsayılan: 20)
* **Connection Timeout**: Bağlantı zaman aşımı (saniye)
* **Idle Timeout**: Boşta kalma zaman aşımı (saniye)
* **Max Lifetime**: Maksimum yaşam süresi (saniye)
* **Validation Query**: Doğrulama sorgusu

## Creating a Database Connection

### Step 1: Basic Information

Temel bilgileri girin:

1. **Connection Management** sayfasına gidin
2. **Database Connections** sekmesine tıklayın
3. **New Connection** butonuna tıklayın
4. Bağlantı bilgilerini girin:
   * **Name**: `Production MySQL DB`
   * **Database Type**: `MySQL`
   * **Host**: `db.example.com`
   * **Port**: `3306`
   * **Database Name**: `production`

### Step 2: Authentication

Kimlik doğrulama bilgilerini girin:

* **Username**: `dbuser`
* **Password**: Secret Manager'dan seçin veya yeni oluşturun

### Step 3: Connection Pool

Bağlantı havuzu ayarlarını yapın:

* **Min Pool Size**: `5`
* **Max Pool Size**: `20`
* **Connection Timeout**: `30` saniye
* **Idle Timeout**: `600` saniye

### Step 4: Test Connection

Bağlantıyı test edin:

1. **Test Connection** butonuna tıklayın
2. Bağlantı başarılı olursa **Save** ile kaydedin
3. Bağlantı başarısız olursa hataları kontrol edin

## Usage in Task Flow

### Database Connector

Database Connector'da kullanım:

1. Task Flow'a **Database Connector** ekleyin
2. **Connection** alanından bağlantıyı seçin
3. SQL sorgusunu yazın
4. Parametreleri yapılandırın

**Örnek:**
```
Database Connector:
- Connection: Production MySQL DB
- Query: SELECT * FROM customers WHERE customer_id = :customerId
- Input: customerId
- Output: customerData
```
