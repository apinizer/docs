Database Connections, veritabanlarına bağlanmak için kullanılan yapılandırmalardır. MySQL, PostgreSQL, Oracle, MSSQL ve DB2 gibi popüler veritabanlarına bağlantı oluşturabilirsiniz.

## Genel Bakış

Database Connections sayesinde:

* Veritabanlarına merkezi bağlantı yönetimi yapabilirsiniz
* Bağlantı bilgilerini güvenli saklayabilirsiniz
* Connection pooling kullanabilirsiniz
* Bağlantı sağlığını izleyebilirsiniz

## Bağlantı Oluşturma

### Yeni Bağlantı Oluşturma

1. **Connection Management** menüsüne gidin
2. **Database Connections** sekmesine tıklayın
3. **New Connection** butonuna tıklayın
4. Bağlantı bilgilerini girin:
   * **Name**: Bağlantı adı
   * **Database Type**: Veritabanı türü
   * **Host**: Sunucu adresi
   * **Port**: Port numarası
   * **Database Name**: Veritabanı adı
   * **Username**: Kullanıcı adı
   * **Password**: Şifre
   * **SSL**: SSL kullanımı
5. **Test Connection** butonuna tıklayın
6. **Save** butonuna tıklayın

## Veritabanı Türleri

### MySQL

```yaml
Database Type: MySQL
Host: localhost
Port: 3306
Database Name: mydb
Username: root
Password: password
SSL: false
```

### PostgreSQL

```yaml
Database Type: PostgreSQL
Host: localhost
Port: 5432
Database Name: mydb
Username: postgres
Password: password
SSL: false
```

### Oracle

```yaml
Database Type: Oracle
Host: localhost
Port: 1521
Database Name: ORCL
Username: system
Password: password
SSL: false
```

### MSSQL

```yaml
Database Type: MSSQL
Host: localhost
Port: 1433
Database Name: mydb
Username: sa
Password: password
SSL: false
```

### DB2

```yaml
Database Type: DB2
Host: localhost
Port: 50000
Database Name: SAMPLE
Username: db2admin
Password: password
SSL: false
```

## Connection Pooling

Connection pooling yapılandırması:

```yaml
Connection Pool:
  Min Size: 5
  Max Size: 20
  Idle Timeout: 300 seconds
  Connection Timeout: 30 seconds
  Validation Query: SELECT 1
```

## Bağlantı Test Etme

Bağlantıyı test etmek için:

1. Bağlantı listesinde test etmek istediğiniz bağlantıyı seçin
2. **Test Connection** butonuna tıklayın
3. Test sonucunu kontrol edin

## Bağlantı Sağlığı

Bağlantı sağlığını izlemek için:

* **Health Check**: Sağlık kontrolü
* **Connection Monitoring**: Bağlantı izleme
* **Alerting**: Uyarılar

## Güvenlik

* **Credential Encryption**: Kimlik bilgileri şifrelenir
* **SSL/TLS**: Güvenli bağlantılar
* **Access Control**: Erişim kontrolü
