FTP/SFTP Configuration, dosya transferi için FTP ve SFTP sunucu bağlantıları oluşturmanızı sağlar.

## Genel Bakış

FTP/SFTP Configuration sayesinde:

* FTP ve SFTP sunucularına bağlantı oluşturabilirsiniz
* Güvenli SFTP bağlantıları kullanabilirsiniz
* Authentication yapılandırabilirsiniz

## Bağlantı Oluşturma

### FTP Yapılandırması

```yaml
Name: ftp-connection
Protocol: FTP
Host: ftp.example.com
Port: 21
Username: user
Password: password
Passive Mode: true
```

### SFTP Yapılandırması

```yaml
Name: sftp-connection
Protocol: SFTP
Host: sftp.example.com
Port: 22
Username: user
Password: password
SSH Key: (opsiyonel)
```
