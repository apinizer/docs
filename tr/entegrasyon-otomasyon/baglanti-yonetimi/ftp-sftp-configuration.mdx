FTP/SFTP Configuration, API Integrator'da dosya transferi için FTP ve SFTP bağlantılarını yönetmenizi sağlar.

## Genel Bakış

FTP/SFTP Configuration şu özellikleri sunar:

* FTP ve SFTP desteği
* Güvenli bağlantılar (SFTP)
* Pasif/aktif mod
* Port yapılandırması

## FTP/SFTP Bağlantısı Oluşturma

Yeni bir FTP/SFTP bağlantısı oluşturmak için:

1. **Connection Management** → **FTP/SFTP Configuration** bölümüne gidin
2. **New Connection** butonuna tıklayın
3. FTP/SFTP ayarlarını yapılandırın

## Bağlantı Ayarları

### Temel Ayarlar

* **Connection Name**: Bağlantı adı
* **Protocol**: FTP veya SFTP
* **Host**: Sunucu adresi
* **Port**: Port (21 for FTP, 22 for SFTP)
* **Username**: Kullanıcı adı
* **Password**: Şifre

### FTP Ayarları

* **Passive Mode**: Pasif mod
* **Binary Mode**: Binary mod
* **Timeout**: Timeout süresi

### SFTP Ayarları

* **Key File**: SSH key dosyası
* **Key Passphrase**: Key passphrase
* **Host Key Verification**: Host key doğrulama

## Best Practices

* **SFTP**: Mümkünse SFTP kullanın
* **Credentials**: Kimlik bilgilerini güvenli saklayın
* **Testing**: Bağlantıyı test edin
