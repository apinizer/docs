Elasticsearch Configuration, API Integrator'da Elasticsearch bağlantılarını yönetmenizi sağlar.

## Genel Bakış

Elasticsearch Configuration şu özellikleri sunar:

* Elasticsearch cluster bağlantısı
* Authentication
* SSL/TLS desteği
* Connection pooling

## Elasticsearch Bağlantısı Oluşturma

Yeni bir Elasticsearch bağlantısı oluşturmak için:

1. **Connection Management** → **Elasticsearch Configuration** bölümüne gidin
2. **New Connection** butonuna tıklayın
3. Elasticsearch ayarlarını yapılandırın

## Bağlantı Ayarları

### Temel Ayarlar

* **Connection Name**: Bağlantı adı
* **Hosts**: Elasticsearch node'ları (comma-separated)
* **Port**: Port (9200)
* **Username**: Kullanıcı adı (opsiyonel)
* **Password**: Şifre (opsiyonel)

### Güvenlik Ayarları

* **SSL**: SSL ayarları
* **SSL Certificate**: SSL sertifikası
* **Authentication**: Kimlik doğrulama

## Best Practices

* **Connection Pooling**: Bağlantı havuzlama kullanın
* **Health Checks**: Sağlık kontrolleri yapın
* **Monitoring**: İzleme yapın
