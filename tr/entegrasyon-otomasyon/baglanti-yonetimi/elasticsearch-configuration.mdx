Elasticsearch Configuration, Elasticsearch cluster'larına bağlantı oluşturmanızı sağlar.

## Genel Bakış

Elasticsearch Configuration sayesinde:

* Elasticsearch cluster'larına bağlantı oluşturabilirsiniz
* Çoklu node desteği kullanabilirsiniz
* Authentication yapılandırabilirsiniz

## Bağlantı Oluşturma

### Yapılandırma

```yaml
Name: elasticsearch-connection
Hosts:
  - localhost:9200
  - localhost:9201
Authentication:
  Type: Basic
  Username: elastic
  Password: password
SSL: false
```
