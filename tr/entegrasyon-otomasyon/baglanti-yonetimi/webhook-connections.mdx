Webhook Connections, webhook endpoint'lerine bağlantı oluşturmanızı sağlar. Bu sayfa, webhook bağlantı yapılandırmasını detaylı olarak açıklar.

## Genel Bakış

Webhook Connections sayesinde:

* Webhook endpoint'lerine bağlantı oluşturabilirsiniz
* Event-driven bildirimler gönderebilirsiniz
* Özel payload'lar yapılandırabilirsiniz
* Authentication yapılandırabilirsiniz

## Connection Configuration

### Creating Webhook Connection

Webhook bağlantısı oluşturma:

1. **Connection Management** sayfasına gidin
2. **Webhook Connections** sekmesine tıklayın
3. **New Connection** butonuna tıklayın
4. Bağlantı bilgilerini girin

### Basic Configuration

Temel yapılandırma:

```yaml
Name: webhook-production
URL: https://api.example.com/webhooks
Method: POST
Headers:
  Content-Type: application/json
```

### Authentication Configuration

Kimlik doğrulama yapılandırması:

```yaml
Authentication:
  Type: Bearer
  Token: ${webhookToken} (Secret Manager)
```

### Custom Headers

Özel header'lar:

```yaml
Headers:
  Content-Type: application/json
  X-API-Key: ${apiKey} (Secret Manager)
  X-Request-ID: ${requestId}
  X-Signature: ${signature}
```

## Payload Configuration

### Custom Payload

Özel payload:

```yaml
Payload:
  event: ${eventType}
  timestamp: ${currentTimestamp}
  data: ${eventData}
  metadata:
    source: taskflow
    correlationId: ${correlationId}
```

### Template Payload

Şablon payload:

```yaml
Payload Template: webhook-payload-template
Template Variables:
  eventType: ${eventType}
  eventData: ${eventData}
  timestamp: ${currentTimestamp}
```

## Testing

### Test Connection

Bağlantı testi:

1. Bağlantı yapılandırmasında **Test Connection** butonuna tıklayın
2. Test webhook gönderilir
3. Test sonucu görüntülenir
