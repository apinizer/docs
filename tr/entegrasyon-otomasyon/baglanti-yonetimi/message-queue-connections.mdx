Message Queue Connections, API Integrator'da mesaj kuyruğu bağlantılarını yönetmenizi sağlar. Bu sayfa, Kafka, RabbitMQ ve ActiveMQ bağlantılarını oluşturma ve yönetme konularını kapsar.

## Genel Bakış

Message Queue Connections şu sistemleri destekler:

* **Kafka**: Yüksek performanslı mesaj kuyruğu
* **RabbitMQ**: Enterprise mesaj broker
* **ActiveMQ**: Açık kaynak mesaj broker

## Kafka Connections

Kafka bağlantısı oluşturmak için:

**Ayarlar:**
* **Bootstrap Servers**: Kafka broker adresleri
* **Security Protocol**: Güvenlik protokolü (PLAINTEXT, SSL, SASL_PLAINTEXT, SASL_SSL)
* **SASL Mechanism**: SASL mekanizması
* **Username/Password**: Kimlik bilgileri

## RabbitMQ Connections

RabbitMQ bağlantısı oluşturmak için:

**Ayarlar:**
* **Host**: Sunucu adresi
* **Port**: Port (5672)
* **Virtual Host**: Virtual host
* **Username**: Kullanıcı adı
* **Password**: Şifre

## ActiveMQ Connections

ActiveMQ bağlantısı oluşturmak için:

**Ayarlar:**
* **Broker URL**: Broker URL'i
* **Username**: Kullanıcı adı
* **Password**: Şifre

## Best Practices

* **Connection Pooling**: Bağlantı havuzlama kullanın
* **Error Handling**: Hata yönetimi yapın
* **Monitoring**: İzleme yapın
