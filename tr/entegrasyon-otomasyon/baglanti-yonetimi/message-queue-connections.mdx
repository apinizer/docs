## Kafka Connection

### Connection Configuration

Kafka bağlantısı oluşturma:

1. **Connection Management** sayfasına gidin
2. **Message Queue Connections** sekmesine tıklayın
3. **New Connection** butonuna tıklayın
4. **Kafka** tipini seçin
5. Bağlantı bilgilerini girin

### Basic Configuration

Temel yapılandırma:

```yaml
Name: kafka-production
Bootstrap Servers: 
  - kafka1.example.com:9092
  - kafka2.example.com:9092
  - kafka3.example.com:9092
Security Protocol: PLAINTEXT
Authentication: None
```

### Security Configuration

Güvenlik yapılandırması:

```yaml
Security Protocol: SASL_SSL
SASL Mechanism: PLAIN
Username: kafka-user
Password: ${kafkaPassword} (Secret Manager)
SSL Configuration:
  Truststore: /path/to/truststore.jks
  Keystore: /path/to/keystore.jks
```

### Advanced Configuration

Gelişmiş yapılandırma:

```yaml
Consumer Group: taskflow-consumers
Session Timeout: 30000ms
Heartbeat Interval: 3000ms
Max Poll Records: 500
Auto Commit: false
```

## RabbitMQ Connection

### Connection Configuration

RabbitMQ bağlantısı oluşturma:

1. **Connection Management** sayfasına gidin
2. **Message Queue Connections** sekmesine tıklayın
3. **New Connection** butonuna tıklayın
4. **RabbitMQ** tipini seçin
5. Bağlantı bilgilerini girin

### Basic Configuration

Temel yapılandırma:

```yaml
Name: rabbitmq-production
Host: rabbitmq.example.com
Port: 5672
Virtual Host: /
Username: rabbitmq-user
Password: ${rabbitmqPassword} (Secret Manager)
SSL: false
```

### SSL Configuration

SSL yapılandırması:

```yaml
SSL: true
SSL Protocol: TLSv1.2
Truststore: /path/to/truststore.jks
Keystore: /path/to/keystore.jks
```

### Connection Pool Configuration

Bağlantı havuzu yapılandırması:

```yaml
Connection Pool:
  Min Size: 5
  Max Size: 20
  Connection Timeout: 30000ms
  Network Recovery: true
```

## ActiveMQ Connection

### Connection Configuration

ActiveMQ bağlantısı oluşturma:

1. **Connection Management** sayfasına gidin
2. **Message Queue Connections** sekmesine tıklayın
3. **New Connection** butonuna tıklayın
4. **ActiveMQ** tipini seçin
5. Bağlantı bilgilerini girin

### Basic Configuration

Temel yapılandırma:

```yaml
Name: activemq-production
Broker URL: tcp://activemq.example.com:61616
Username: activemq-user
Password: ${activemqPassword} (Secret Manager)
Client ID: taskflow-client
```

### Failover Configuration

Failover yapılandırması:

```yaml
Broker URL: failover:(tcp://activemq1.example.com:61616,tcp://activemq2.example.com:61616)
Failover Options:
  Initial Reconnect Delay: 1000ms
  Max Reconnect Delay: 30000ms
  Use Exponential Backoff: true
```

### SSL Configuration

SSL yapılandırması:

```yaml
Broker URL: ssl://activemq.example.com:61617
SSL Configuration:
  Truststore: /path/to/truststore.jks
  Keystore: /path/to/keystore.jks
  Key Password: ${keyPassword} (Secret Manager)
```
