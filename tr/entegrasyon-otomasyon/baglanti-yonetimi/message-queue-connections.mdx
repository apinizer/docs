Message Queue Connections, mesaj kuyruğu sistemlerine bağlanmak için kullanılan yapılandırmalardır. Kafka, RabbitMQ ve ActiveMQ gibi popüler mesaj kuyruğu sistemlerine bağlantı oluşturabilirsiniz.

## Genel Bakış

Message Queue Connections sayesinde:

* Mesaj kuyruğu sistemlerine merkezi bağlantı yönetimi yapabilirsiniz
* Event-driven entegrasyonlar oluşturabilirsiniz
* Asenkron işlemler gerçekleştirebilirsiniz

## Kafka Connection

### Yapılandırma

```yaml
Name: kafka-connection
Bootstrap Servers: localhost:9092
Security Protocol: PLAINTEXT
Authentication: None
```

## RabbitMQ Connection

### Yapılandırma

```yaml
Name: rabbitmq-connection
Host: localhost
Port: 5672
Virtual Host: /
Username: guest
Password: guest
SSL: false
```

## ActiveMQ Connection

### Yapılandırma

```yaml
Name: activemq-connection
Broker URL: tcp://localhost:61616
Username: admin
Password: admin
```
