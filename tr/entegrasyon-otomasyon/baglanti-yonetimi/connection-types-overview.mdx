Connection Types Overview, API Integrator'da kullanılabilen farklı bağlantı tiplerini ve bunların kullanım alanlarını açıklar. Bu rehber, hangi bağlantı tipinin ne zaman kullanılacağını anlamanıza yardımcı olur.

## Genel Bakış

API Integrator şu bağlantı tiplerini destekler:

* **Database Connections**: Veritabanı bağlantıları
* **Message Queue Connections**: Mesaj kuyruğu bağlantıları
* **Email SMTP Configuration**: E-posta SMTP yapılandırması
* **FTP/SFTP Configuration**: FTP/SFTP yapılandırması
* **Elasticsearch Configuration**: Elasticsearch yapılandırması
* **LDAP/Active Directory Configuration**: LDAP/AD yapılandırması
* **Linux Connections**: Linux bağlantıları
* **Logback Connections**: Logback bağlantıları
* **SNMP Connections**: SNMP bağlantıları
* **Syslog Connections**: Syslog bağlantıları
* **Webhook Connections**: Webhook bağlantıları

## Database Connections

Veritabanı bağlantıları, SQL sorguları ve veritabanı işlemleri için kullanılır.

**Desteklenen Veritabanları:**
* MySQL
* PostgreSQL
* Oracle
* SQL Server
* MongoDB
* Diğer JDBC uyumlu veritabanları

**Kullanım Senaryoları:**
* Veri okuma/yazma
* Stored procedure çağırma
* Transaction yönetimi

**Detaylı Bilgi:** [Database Connections](/tr/entegrasyon-otomasyon/baglanti-yonetimi/database-connections)

## Message Queue Connections

Mesaj kuyruğu bağlantıları, asenkron mesajlaşma için kullanılır.

**Desteklenen Sistemler:**
* Kafka
* RabbitMQ
* ActiveMQ

**Kullanım Senaryoları:**
* Event-driven işlemler
* Asenkron işleme
* Mesaj yayınlama/abone olma

**Detaylı Bilgi:** [Message Queue Connections](/tr/entegrasyon-otomasyon/baglanti-yonetimi/message-queue-connections)

## Email SMTP Configuration

E-posta SMTP yapılandırması, e-posta gönderme için kullanılır.

**Kullanım Senaryoları:**
* Bildirim e-postaları
* Rapor gönderimi
* Toplu e-posta

**Detaylı Bilgi:** [Email SMTP Configuration](/tr/entegrasyon-otomasyon/baglanti-yonetimi/email-smtp-configuration)

## FTP/SFTP Configuration

FTP/SFTP yapılandırması, dosya transferi için kullanılır.

**Kullanım Senaryoları:**
* Dosya yükleme/indirme
* Dosya senkronizasyonu
* Batch dosya işleme

**Detaylı Bilgi:** [FTP/SFTP Configuration](/tr/entegrasyon-otomasyon/baglanti-yonetimi/ftp-sftp-configuration)

## Elasticsearch Configuration

Elasticsearch yapılandırması, arama ve indeksleme için kullanılır.

**Kullanım Senaryoları:**
* Veri indeksleme
* Arama sorguları
* Log analizi

**Detaylı Bilgi:** [Elasticsearch Configuration](/tr/entegrasyon-otomasyon/baglanti-yonetimi/elasticsearch-configuration)

## LDAP/Active Directory Configuration

LDAP/AD yapılandırması, kimlik doğrulama ve yetkilendirme için kullanılır.

**Kullanım Senaryoları:**
* Kullanıcı doğrulama
* Grup yönetimi
* Yetkilendirme

**Detaylı Bilgi:** [LDAP/Active Directory Configuration](/tr/entegrasyon-otomasyon/baglanti-yonetimi/ldap-active-directory-configuration)

## Connection Yönetimi

### Connection Oluşturma

Yeni bir connection oluşturmak için:

1. **Connection Management** sayfasına gidin
2. Connection tipini seçin
3. **New Connection** butonuna tıklayın
4. Bağlantı bilgilerini girin
5. **Save** butonuna tıklayın

### Connection Test

Connection'ı test etmek için:

1. Connection listesinden bir connection seçin
2. **Test Connection** butonuna tıklayın
3. Test sonucu görüntülenir

### Connection Düzenleme

Connection düzenlemek için:

1. Connection listesinden bir connection seçin
2. **Edit** butonuna tıklayın
3. Ayarları değiştirin
4. **Save** butonuna tıklayın

### Connection Silme

Connection silmek için:

1. Connection listesinden bir connection seçin
2. **Delete** butonuna tıklayın
3. Onaylayın

## Best Practices

### Güvenlik

* **Credentials**: Kimlik bilgilerini güvenli saklayın
* **Encryption**: Şifreleme kullanın
* **Access Control**: Erişim kontrolü yapın

### Performance

* **Connection Pooling**: Bağlantı havuzlama kullanın
* **Timeout**: Timeout değerlerini ayarlayın
* **Monitoring**: İzleme yapın

### Reliability

* **Error Handling**: Hata yönetimi yapın
* **Retry Logic**: Yeniden deneme mantığı ekleyin
* **Health Checks**: Sağlık kontrolleri yapın
