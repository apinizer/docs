Bağlantılar (Connections), API Proxy'lerin ve Task Flow'ların dış sistemlere bağlanması için kullanılan yapılandırmalardır. Bu sayfada desteklenen bağlantı türlerine genel bir bakış sunulmaktadır.

## Genel Bakış

Bağlantılar sayesinde:

* Dış sistemlere merkezi bağlantı yönetimi yapabilirsiniz
* Bağlantı bilgilerini güvenli saklayabilirsiniz
* Bağlantıları yeniden kullanabilirsiniz
* Bağlantı sağlığını izleyebilirsiniz

## Bağlantı Türleri

### Database Connections

Veritabanı bağlantıları:

* **MySQL**: MySQL veritabanı bağlantıları
* **PostgreSQL**: PostgreSQL veritabanı bağlantıları
* **Oracle**: Oracle veritabanı bağlantıları
* **MSSQL**: Microsoft SQL Server bağlantıları
* **DB2**: IBM DB2 bağlantıları

### Message Queue Connections

Mesaj kuyruğu bağlantıları:

* **Kafka**: Apache Kafka bağlantıları
* **RabbitMQ**: RabbitMQ bağlantıları
* **ActiveMQ**: Apache ActiveMQ bağlantıları

### Email SMTP Configuration

E-posta sunucu bağlantıları:

* **SMTP**: SMTP sunucu bağlantıları
* **SSL/TLS**: SSL/TLS desteği
* **Authentication**: Kimlik doğrulama

### FTP/SFTP Configuration

Dosya transfer bağlantıları:

* **FTP**: FTP sunucu bağlantıları
* **SFTP**: SFTP sunucu bağlantıları
* **Passive Mode**: Pasif mod desteği

### Elasticsearch Configuration

Elasticsearch bağlantıları:

* **Cluster Connection**: Cluster bağlantıları
* **Multiple Nodes**: Çoklu node desteği
* **Authentication**: Kimlik doğrulama

### LDAP/Active Directory Configuration

LDAP ve Active Directory bağlantıları:

* **LDAP**: LDAP sunucu bağlantıları
* **Active Directory**: Active Directory bağlantıları
* **Authentication**: Kimlik doğrulama

### Linux Connections

Linux sunucu bağlantıları:

* **SSH**: SSH bağlantıları
* **Command Execution**: Komut çalıştırma

### Logback Connections

Logback bağlantıları:

* **Logback Configuration**: Logback yapılandırması

### SNMP Connections

SNMP bağlantıları:

* **SNMP Configuration**: SNMP yapılandırması

### Syslog Connections

Syslog bağlantıları:

* **Syslog Configuration**: Syslog yapılandırması

### Webhook Connections

Webhook bağlantıları:

* **Webhook Configuration**: Webhook yapılandırması

## Bağlantı Yönetimi

### Bağlantı Oluşturma

1. **Connection Management** menüsüne gidin
2. İlgili bağlantı türünü seçin
3. **New Connection** butonuna tıklayın
4. Bağlantı bilgilerini girin
5. **Test Connection** butonuna tıklayın
6. **Save** butonuna tıklayın

### Bağlantı Test Etme

Bağlantıları test etmek için:

* **Test Connection**: Bağlantı testi
* **Connection Health**: Bağlantı sağlığı
* **Connection Monitoring**: Bağlantı izleme

### Bağlantı Güvenliği

* **Credential Encryption**: Kimlik bilgisi şifreleme
* **SSL/TLS**: SSL/TLS kullanımı
* **Access Control**: Erişim kontrolü

## Bağlantı Yeniden Kullanılabilirliği

### Bağlantıları Yeniden Kullanma

Bağlantılar yeniden kullanılabilir:

* **Multiple Connectors**: Aynı bağlantıyı birden fazla connector'da kullanın
* **Multiple Task Flows**: Aynı bağlantıyı Task Flow'lar arasında kullanın
* **Centralized Management**: Bağlantıları merkezi olarak yönetin
