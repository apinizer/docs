## Connection Types

### Database Connections

Veritabanı bağlantıları:

* **MySQL**: MySQL veritabanları
* **PostgreSQL**: PostgreSQL veritabanları
* **Oracle**: Oracle veritabanları
* **Microsoft SQL Server**: MSSQL veritabanları
* **IBM DB2**: DB2 veritabanları

Daha fazla bilgi için [Database Connections](/tr/entegrasyon-otomasyon/baglanti-yonetimi/database-connections) sayfasına bakın.

### Message Queue Connections

Mesaj kuyruğu bağlantıları:

* **Kafka**: Apache Kafka bağlantıları
* **RabbitMQ**: RabbitMQ bağlantıları
* **ActiveMQ**: Apache ActiveMQ bağlantıları

Daha fazla bilgi için [Message Queue Connections](/tr/entegrasyon-otomasyon/baglanti-yonetimi/message-queue-connections) sayfasına bakın.

### Email SMTP Configuration

E-posta SMTP bağlantıları:

Daha fazla bilgi için [Email SMTP Configuration](/tr/entegrasyon-otomasyon/baglanti-yonetimi/email-smtp-configuration) sayfasına bakın.

### LDAP/Active Directory Configuration

LDAP/Active Directory bağlantıları:

### Linux Connections

Linux bağlantıları:

* **SSH Host**: SSH sunucu adresi
* **SSH Port**: SSH port numarası
* **Authentication**: Kimlik doğrulama
* **Command Execution**: Komut çalıştırma

### Logback Connections

Logback bağlantıları:

* **Logback Configuration**: Logback yapılandırması
* **Appender Configuration**: Appender yapılandırması

### SNMP Connections

SNMP bağlantıları:

* **SNMP Host**: SNMP sunucu adresi
* **SNMP Port**: SNMP port numarası
* **Community**: SNMP community
* **Version**: SNMP versiyonu

### Syslog Connections

Syslog bağlantıları:

* **Syslog Host**: Syslog sunucu adresi
* **Syslog Port**: Syslog port numarası
* **Protocol**: UDP/TCP
* **Format**: Syslog formatı

### Webhook Connections

Webhook bağlantıları:

* **Webhook URL**: Webhook URL'i
* **Authentication**: Kimlik doğrulama
* **Headers**: HTTP header'ları

## Connection Management

### Creating Connections

Bağlantı oluşturma:

1. **Connection Management** sayfasına gidin
2. İlgili bağlantı tipini seçin
3. **New Connection** butonuna tıklayın
4. Bağlantı bilgilerini girin
5. **Test Connection** ile bağlantıyı test edin
6. **Save** ile kaydedin

### Editing Connections

Bağlantı düzenleme:

1. Bağlantı listesinden ilgili bağlantıyı seçin
2. **Edit** butonuna tıklayın
3. Bilgileri güncelleyin
4. **Save** ile kaydedin

### Testing Connections

Bağlantı testi:

* **Test Connection**: Bağlantıyı test etme
* **Connection Status**: Bağlantı durumu
* **Health Check**: Sağlık kontrolü

### Deleting Connections

Bağlantı silme:

* **Delete**: Bağlantıyı silme
* **Dependency Check**: Bağımlılık kontrolü
* **Confirmation**: Onaylama

## Security

### Credential Management

Kimlik bilgisi yönetimi:

* **Secret Manager**: Şifreleri Secret Manager'da saklama
* **Encryption**: Şifreleme
* **Access Control**: Erişim kontrolü

### Connection Security

Bağlantı güvenliği:

* **TLS/SSL**: TLS/SSL desteği
* **Encrypted Connections**: Şifreli bağlantılar
* **Certificate Management**: Sertifika yönetimi

## Connection Pooling

### Pool Configuration

Havuz yapılandırması:

* **Min Pool Size**: Minimum havuz boyutu
* **Max Pool Size**: Maksimum havuz boyutu
* **Connection Timeout**: Bağlantı zaman aşımı
* **Idle Timeout**: Boşta kalma zaman aşımı

### Benefits

Havuzlama avantajları:

* **Performance**: Performans artışı
* **Resource Efficiency**: Kaynak verimliliği
* **Connection Reuse**: Bağlantı yeniden kullanımı

## Monitoring

### Connection Monitoring

Bağlantı izleme:

* **Connection Status**: Bağlantı durumu
* **Health Checks**: Sağlık kontrolleri
* **Usage Statistics**: Kullanım istatistikleri
* **Error Tracking**: Hata takibi

### Alerts
