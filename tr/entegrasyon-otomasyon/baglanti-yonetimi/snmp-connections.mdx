## Connection Configuration

### Creating SNMP Connection

SNMP bağlantısı oluşturma:

1. **Connection Management** sayfasına gidin
2. **SNMP Connections** sekmesine tıklayın
3. **New Connection** butonuna tıklayın
4. Bağlantı bilgilerini girin

### Basic Configuration

Temel yapılandırma:

```yaml
Name: snmp-production
Host: device.example.com
Port: 162
Community: public
Version: v2c
```

### SNMP v3 Configuration

SNMP v3 yapılandırması:

```yaml
Version: v3
Username: snmp-user
Auth Protocol: SHA
Auth Password: ${authPassword} (Secret Manager)
Priv Protocol: AES
Priv Password: ${privPassword} (Secret Manager)
```

## SNMP Versions

### SNMP v1

SNMP v1 desteği:

* **Community-based**: Community tabanlı
* **Basic Security**: Temel güvenlik

### SNMP v2c

SNMP v2c desteği:

* **Community-based**: Community tabanlı
* **Enhanced Features**: Gelişmiş özellikler

### SNMP v3

SNMP v3 desteği:

* **User-based**: Kullanıcı tabanlı
* **Encryption**: Şifreleme
* **Enhanced Security**: Gelişmiş güvenlik

## Testing

### Test Connection

Bağlantı testi:

1. Bağlantı yapılandırmasında **Test Connection** butonuna tıklayın
2. Bağlantı testi çalıştırılır
3. Test sonucu görüntülenir

### Test Operations

* **Trap Test**: Trap testi
