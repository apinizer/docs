Bu sayfada JiraOps bağlantı ayarlarının nasıl yapıldığı anlatılmaktadır.

## Connection Parametreleri

| Alan | Açıklama |
|------|----------|
| Aktif (Enabled) | Bağlantının aktiflik durumu |
| Ad (Name) | Oluşturulan bağlantının ad bilgisidir |
| Açıklama (Description) | Oluşturulan konfigürasyon ile ilgili yönetimi kolaylaştırmak için açıklama yazılabilir |
| URL | Jira Ops API Url.|
| API Key | Jira Ops API'sine client olmak için gerekli olan API Anahtarı |

<Steps>
    <Step title="Connection Sayfasına Gitme">
        - Sol menüden **Connection → JiraOps** bölümüne gidin.
        - Sağ üstteki **[+ New]** butonuna tıklayın.
    </Step>

    <Step title="Temel Bilgileri Girme">
        **Enabled (Aktif):**
        - Toggle ile bağlantıyı aktif/pasif yapabilirsiniz.

        **Name (Ad):**
        - Bağlantı için benzersiz bir isim girin.
        - Örnek: `OpsGenie_API`

        **Description (Açıklama):**
        - Bağlantının amacını açıklayın.
        - Örnek: "Opsgenie alert integration"
    </Step>

    <Step title="API Bilgilerini Girme">
        **URL:**
        - Jira Ops API endpoint'ini girin.

        **API Key:**
        - Jira Ops platformundan aldığınız API anahtarını girin.
        - Bu bilgi şifrelenmiş olarak saklanır.
    </Step>

    <Step title="Kaydetme">
        - Tüm bilgileri girdikten sonra **[Save and Deploy]** butonuna tıklayın.
        - Bağlantı Integration Flow ve Connector adımlarında kullanılabilir hale gelir.
    </Step>
</Steps>

## Kullanım Alanları

JiraOps connection aşağıdaki senaryolarda kullanılır:

- Apinizer üzerinden Opsgenie'ye alert gönderme
- Incident yönetimi ve otomasyonu
- On-call rotasyon entegrasyonu
- Alert eskalasyonu ve bildirim yönetimi

## Best Practices

| Kategori | Öneri |
|----------|-------|
| API Key Güvenliği | API anahtarlarını asla kod içinde saklamayın. Secret manager kullanın |
| Test Ortamı | Production API key'ini test ortamında kullanmayın. Ayrı key'ler oluşturun |
| Monitoring | API çağrı limitlerini takip edin ve rate limit uyarıları için alert kurun |

## Sık Sorulan Sorular

| Soru | Cevap |
|------|-------|
| API Key nereden alınır? | Jira Ops/Opsgenie platformunda Settings → Integrations bölümünden API key oluşturabilirsiniz |
| Rate limit var mı? | Evet, Opsgenie API'sinin rate limitleri vardır. Detaylar için Opsgenie API dokümantasyonuna bakın |
| Hangi API versiyonu destekleniyor? | v2 API endpoint'leri desteklenmektedir |
