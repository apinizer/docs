## Connection Yapılandırma

### Yeni jiraOps Connection Oluşturma

    src="/images/yonetici/baglantilar/jira.png"

<Steps>
    <Step title="Connection Sayfasına Gitme">
        - Sol menüden **Connection → JiraOps** bölümüne gidin.
        - Sağ üstteki **[+ New]** butonuna tıklayın.
    </Step>

    <Step title="Temel Bilgileri Girme">
        **Enabled (Aktif):**
        - Toggle ile bağlantıyı aktif/pasif yapabilirsiniz.

        **Name (Ad):**
        - Bağlantı için benzersiz bir isim girin.
        - Örnek: `OpsGenie_API`

        **Description (Açıklama):**
        - Bağlantının amacını açıklayın.
        - Örnek: "Opsgenie alert integration"
    </Step>

    <Step title="API Bilgilerini Girme">
        **URL:**
        - Jira Ops API endpoint'ini girin.

        **API Key:**
        - Jira Ops platformundan aldığınız API anahtarını girin.
        - Bu bilgi şifrelenmiş olarak saklanır.
    </Step>

    <Step title="Kaydetme">
        - Tüm bilgileri girdikten sonra **[Save and Deploy]** butonuna tıklayın.
        - Bağlantı Integration Flow ve Connector adımlarında kullanılabilir hale gelir.
    </Step>
</Steps>

## Sık Sorulan Sorular

<AccordionGroup>
  <Accordion title="API Key nereden alınır?">
    Jira Ops/Opsgenie platformunda Settings → Integrations bölümünden API key oluşturabilirsiniz
  </Accordion>

  <Accordion title="Rate limit var mı?">
    Evet, Opsgenie API'sinin rate limitleri vardır. Detaylar için Opsgenie API dokümantasyonuna bakın
  </Accordion>

  <Accordion title="Hangi API versiyonu destekleniyor?">
    v2 API endpoint'leri desteklenmektedir
  </Accordion>
</AccordionGroup>
