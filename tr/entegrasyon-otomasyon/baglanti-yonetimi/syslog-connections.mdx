Syslog Connections, syslog sunucularına bağlantı oluşturmanızı sağlar. Bu sayfa, syslog bağlantı yapılandırmasını detaylı olarak açıklar.

## Genel Bakış

Syslog Connections sayesinde:

* Syslog sunucularına bağlantı oluşturabilirsiniz
* RFC 3164 veya RFC 5424 standartlarına uygun log mesajları gönderebilirsiniz
* Merkezi log yönetimi yapabilirsiniz
* UDP veya TCP protokollerini kullanabilirsiniz

## Connection Configuration

### Creating Syslog Connection

Syslog bağlantısı oluşturma:

1. **Connection Management** sayfasına gidin
2. **Syslog Connections** sekmesine tıklayın
3. **New Connection** butonuna tıklayın
4. Bağlantı bilgilerini girin

### Basic Configuration

Temel yapılandırma:

```yaml
Name: syslog-production
Host: syslog.example.com
Port: 514
Protocol: UDP
Format: RFC5424
```

### RFC 3164 Configuration

RFC 3164 yapılandırması:

```yaml
Format: RFC3164
Facility: 16 (Local0)
Severity: 6 (Informational)
Hostname: ${hostname}
Tag: taskflow
```

### RFC 5424 Configuration

RFC 5424 yapılandırması:

```yaml
Format: RFC5424
Facility: 16 (Local0)
Severity: 6 (Informational)
Hostname: ${hostname}
App Name: taskflow
Proc ID: ${processId}
Msg ID: ${messageId}
```

## Protocol Configuration

### UDP Protocol

UDP protokolü:

* **Faster**: Daha hızlı
* **No Guaranteed Delivery**: Garantili teslimat yok
* **Lower Overhead**: Daha düşük overhead

### TCP Protocol

TCP protokolü:

* **Reliable**: Güvenilir teslimat
* **Guaranteed Delivery**: Garantili teslimat
* **Higher Overhead**: Daha yüksek overhead

## Testing

### Test Connection

Bağlantı testi:

1. Bağlantı yapılandırmasında **Test Connection** butonuna tıklayın
2. Test syslog mesajı gönderilir
3. Test sonucu görüntülenir

## Best Practices

### Security

* **Network Security**: Ağ güvenliği
* **Message Encryption**: Mesaj şifreleme
* **Access Control**: Erişim kontrolü
* **Audit Logging**: Denetim loglama

### Performance

* **UDP for High Volume**: Yüksek hacim için UDP
* **TCP for Critical**: Kritik loglar için TCP
* **Batch Logging**: Toplu loglama
* **Async Processing**: Asenkron işleme
