Logback Connections, Logback logging framework'üne bağlantı oluşturmanızı sağlar. Bu sayfa, Logback bağlantı yapılandırmasını detaylı olarak açıklar.

## Genel Bakış

Logback Connections sayesinde:

* Logback logging framework'üne bağlantı oluşturabilirsiniz
* Yapılandırılmış loglama yapabilirsiniz
* Logback appender'larına log mesajları gönderebilirsiniz
* Merkezi log yönetimi yapabilirsiniz

## Connection Configuration

### Creating Logback Connection

Logback bağlantısı oluşturma:

1. **Connection Management** sayfasına gidin
2. **Logback Connections** sekmesine tıklayın
3. **New Connection** butonuna tıklayın
4. Bağlantı bilgilerini girin

### Basic Configuration

Temel yapılandırma:

```yaml
Name: logback-production
Appender: fileAppender
Logger: apinizer
Level: INFO
```

### Appender Configuration

Appender yapılandırması:

* **File Appender**: Dosya appender'ı
* **Console Appender**: Konsol appender'ı
* **Socket Appender**: Soket appender'ı
* **SMTP Appender**: SMTP appender'ı

### Logger Configuration

Logger yapılandırması:

```yaml
Logger Name: apinizer
Logger Level: INFO
Additivity: true
```

## Appender Types

### File Appender

Dosya appender:

```yaml
Appender Type: File
File Path: /var/log/apinizer/app.log
Pattern: %d{yyyy-MM-dd HH:mm:ss.SSS} [%thread] %-5level %logger{36} - %msg%n
Rolling Policy:
  Max File Size: 10MB
  Max History: 30
```

### Console Appender

Konsol appender:

```yaml
Appender Type: Console
Target: System.out
Pattern: %d{HH:mm:ss.SSS} [%thread] %-5level %logger{36} - %msg%n
```

### Socket Appender

Soket appender:

```yaml
Appender Type: Socket
Host: logserver.example.com
Port: 4560
Pattern: %d{yyyy-MM-dd HH:mm:ss.SSS} [%thread] %-5level %logger{36} - %msg%n
```

## Testing

### Test Connection

Bağlantı testi:

1. Bağlantı yapılandırmasında **Test Connection** butonuna tıklayın
2. Test log mesajı gönderilir
3. Test sonucu görüntülenir
