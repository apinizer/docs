# Prometheus ve Grafana Entegrasyonu

Bu sayfa, Apinizer metriklerinin Prometheus ve Grafana'ya entegre edilmesi işlemini açıklamaktadır.

## Genel Bakış

Prometheus ve Grafana entegrasyonu ile Apinizer metriklerini izleyebilir ve görselleştirebilirsiniz.

## Yapılandırma Adımları

### 1. Prometheus Kurulumu

Prometheus'u kurun ve yapılandırın:

- Prometheus deployment'ı
- ServiceMonitor yapılandırması
- Scraping ayarları

### 2. Grafana Kurulumu

Grafana'yı kurun ve yapılandırın:

- Grafana deployment'ı
- Persistent storage yapılandırması
- Admin kullanıcı ayarları

### 3. Prometheus Data Source

Grafana'da Prometheus data source ekleyin:

- Prometheus URL'ini girin
- Authentication ayarlarını yapın
- Test connection yapın

### 4. Dashboard Oluşturma

Grafana dashboard'ları oluşturun:

- Apinizer metrikleri için dashboard'lar
- Custom panel'ler
- Alert kuralları
