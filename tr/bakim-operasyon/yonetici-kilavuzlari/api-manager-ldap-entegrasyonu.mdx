# API Manager LDAP Entegrasyonu

Bu sayfa, API Manager'ın LDAP (Lightweight Directory Access Protocol) ile entegrasyonu işlemini açıklamaktadır.

## Genel Bakış

API Manager'ı LDAP ile entegre ederek kurumsal kimlik yönetimi sisteminizle kullanıcı doğrulaması yapabilirsiniz.

## Yapılandırma Adımları

### 1. LDAP Sunucu Bilgileri

LDAP sunucu bilgilerini hazırlayın:

- **Host**: LDAP sunucu adresi
- **Port**: LDAP port numarası (389 veya 636)
- **Base DN**: LDAP base distinguished name
- **Bind DN**: LDAP bağlantı kullanıcı adı
- **Bind Password**: LDAP bağlantı parolası

### 2. API Manager Yapılandırması

API Manager'da LDAP yapılandırması:

```yaml
spring:
  ldap:
    urls: ldap://ldap.example.com:389
    base: dc=example,dc=com
    username: cn=admin,dc=example,dc=com
    password: <ldap-password>
```

### 3. User Search Yapılandırması

Kullanıcı arama yapılandırması:

- **User Search Base**: Kullanıcı arama base'i
- **User Search Filter**: Kullanıcı arama filtresi
- **Group Search Base**: Grup arama base'i

### 4. Test ve Doğrulama

LDAP entegrasyonunu test edin:

- Kullanıcı girişini test edin
- Grup eşleştirmesini test edin
- Log'ları kontrol edin
