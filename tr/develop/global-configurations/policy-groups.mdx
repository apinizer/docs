Politika grubu ekranında, birden fazla politika gruplandırılarak kullanılması için kullanılır. Gruplanan politikalar API Proxy/API Proxy Grup sayfasında politika eklenir gibi grup halinde eklenebilir ya da oluşturulurken belirli bir işlem sırasına göre API Proxy ve API Proxy Gruplarına bağlanabilir. Kullanıcılar, 'Add Policy' butonu ile grup içine istediği yerel veya global politikaları ekleyebilir ve sırasını düzenleyebilir, isterse politikanın durumunu aktif/pasif hale getirilebilir.

<Info>
Politikalar detayı için [Politikalar (Policies)](/tr/develop/politikalar/api-bazli-daraltma.mdx) sayfasını ziyaret edebilirsiniz.
</Info>

## Politika Grubu Deploy İşlemi

Politika grubu üzerinde herhangi bir değişiklik yapıldığında, bu grubu kullanan API Proxy'lerin ve API Proxy Grupların durumu "Redeploy Required" olarak değişir. Bunları tek tek deploy etmek yerine, "Politika Grup Deploy" özelliği ile bu politika grubunu kullanan tüm API Proxy'lerin ve API Proxy Grupların belirli bir çalışma ortamına *(environment)* toplu olarak deploy edilmesini sağlar.

Deploy işlemi başlatıldığında, bir onay dialogu açılır. Bu dialog'da aşağıdaki bilgiler ve seçenekler bulunur:

- **Politika/Ayar Grubu Adı**: Deploy edilecek politika grubunun adı gösterilir. Bu bilgi, toplu işlem olduğundan onay mekanizmasında görüntülenir.
- **API Proxy ve API Proxy Group Seçimi**: Deploy işleminin API Proxy'lere mi, API Proxy Gruplarına mı uygulanacağını seçebilirsiniz. Her iki seçenek de işaretlenebilir.
- **Filtreleme Seçeneği**: Deploy edilecek API Proxy ve API Proxy Gruplarını filtrelemek için aşağıdaki seçeneklerden birini seçebilirsiniz:
  - **Hepsine**: Bu politika grubunu kullanan tüm API Proxy ve API Proxy Grupları deploy edilir.
  - **Sadece deployed durumunda olanlara**: Sadece seçili ortamda deployed durumunda olan ve redeploy gerektirmeyen API Proxy ve API Proxy Grupları deploy edilir.
  - **Sadece deployed ve redeploy required durumda olanlara**: Sadece seçili ortamda deployed durumunda olan ve redeploy gerektiren API Proxy ve API Proxy Grupları deploy edilir.
- **Deploy Edilecek Nesne Sayıları**: Seçilen filtreleme seçeneğine göre deploy edilecek API Proxy ve API Proxy Group sayıları gösterilir.

Filtreleme seçeneği değiştirildiğinde, deploy edilecek nesne sayıları otomatik olarak güncellenir. Deploy işlemi başlatıldığında, seçilen filtreleme seçeneğine göre belirlenen API Proxy ve API Proxy Grupları deploy edilir.

Deploy işlemi tamamlandığında, kullanıcıya deploy işleminin sonucunu gösteren **Deployment Results** adında bir sonuç ekranı sunulur.
