JSON to XML konfigürasyonu için kullanılan alanlar aşağıdaki tabloda görülmektedir.

| Alan | Açıklama |
|------|----------|
| **NULL değerleri görmezden gel** <em>(Ignore fields with NULL values)</em> | Değeri NULL olan elemanların JSON mesajının içine koyulmaması için işaretlenir. |
| **Boş değerleri görmezden gel** <em>(Ignore EMPTY fields)</em> | Değeri boş olan elemanların JSON mesajının içine koyulmaması için işaretlenir. |
| **NULL değerler için xsi:nil="true" kullan** <em>(Use xsi:nil="true" for NULL values)</em> | İşaretlenirse, Backend API'ye gönderilecek XML mesajın içine, istekte gelen JSON mesajdaki NULL değerleri için xsi:nil="true" niteliği kullanılır. Aksi taktirde XML mesajın içine ilgili değer için boş eleman eklenir. |
| **Elemanı Çöz** <em>(Unwrap Element)</em> | İşaretlenirse, JSON kök elemanı olduğu gibi dönüştürür, işaretlenmezse (öntanımlı olarak) ObjectNode veya ArrayNode elemanı ile sarmalar. |
