# Belirli Api Proxy'lere ait logları dump alma

Aşağıdaki komut ile Elasticsearch'teki indexler --searchBody alanındaki değere sorguya göre filtrelenip _"/tmp/es_backup"_ klasörüne dump edilir.
[code] 
    ./multielasticdump --direction=dump --match='apilogs*' 
    --input=http://<ELASTICSEARCH_IP>:9200 
    --ignoreType='mapping,settings,template' 
    --output=/tmp/es_backup --searchBody="{\"query\": {\"terms\": { \"apiGatewayId\": [33,38,96,97,99,100,101,82,80]}}}"  
    --overwrite 
    --ignoreChildError=true 
    --ignoreMapping=true 
    --ignoreSettings=true 
    --ignoreTemplate=true 
    --ignoreAlias=true
    
[/code]

Kurulum:
[code] 
    yum install npm
    npm install elasticdump
    
    npm install  --no-bin-links elasticdump
[/code]
