## Genel Bakış

Client Route Raporu, API Proxy ve Proxy Group'larınızın client route yapılandırmalarını analiz ederek potansiyel çakışmaları tespit etmenizi sağlar. Bu rapor sayesinde route çakışmalarını önceden görebilir ve yapılandırma sorunlarını giderebilirsiniz.

<img src="/images/yonetici/client-route.png" alt="Senaryo Diyagramı" width="1000" />


## Rapor Tablosu

Rapor sonuçları aşağıdaki kolonlarda gösterilir:

| Alan | Açıklama |
|------|----------|
| Proje (*Project*) | Route'un ait olduğu proje adı |
| Tip (*Type*) | Route'un tipi (API Proxy veya Proxy Group) |
| İsim (*Name*) | API Proxy veya Proxy Group adı |
| Path'ler (*Paths*) | Route'un tanımlı path'leri |
| Host'lar (*Hosts*) | Route'un tanımlı host'ları |
| Header'lar (*Headers*) | Route'un tanımlı header'ları (name:value formatında) |
| Method'lar (*Methods*) | Route'un desteklediği HTTP method'ları |
| Deployment'lar (*Deployments*) | Deploy edilen/Total ortam sayısı (örn: 2/3) |
| Çakışma (*Conflict*) | Route'un çakışma durumu (Çakışma Yok, Tam Eşleşme, Önek Çakışması) |
| İşlemler (*Actions*) | Detayları görüntüleme butonu |

## Detay Görüntüleme

Route detaylarını görüntülemek için **Detayları Görüntüle** butonuna tıklayabilirsiniz. Detay diyaloğunda aşağıdaki bilgiler gösterilir:

<CardGroup cols={2}>
  <Card title="Route Bilgileri" icon="info-circle">
    * Proje adı
    * Tip (API Proxy/Proxy Group)
    * İsim
    * Path'ler
    * Host'lar
    * Header'lar
    * Method'lar
    * Çakışma durumu
  </Card>
  <Card title="Deployment Detayları" icon="server">
    * Ortam adı
    * Erişim URL'i
    * Deploy durumu
    * Yeniden deploy gereksinimi
  </Card>
  <Card title="Çakışan Route'lar" icon="exclamation-triangle">
    Çakışma durumunda olan diğer route'ların bilgileri:
    * Proje adı
    * Tip
    * İsim
    * Path
    * Çakışma nedeni
  </Card>
</CardGroup>

## Çakışma Durumları

Route'lar üç farklı çakışma durumuna sahip olabilir:

<CardGroup cols={3}>
  <Card title="Çakışma Yok" icon="check-circle">
    Route'un başka bir route ile çakışması yoktur. Bu route güvenle kullanılabilir.
  </Card>
  <Card title="Tam Eşleşme" icon="exclamation-circle">
    Route'un path, host, header ve method yapılandırması başka bir route ile tam olarak eşleşmektedir. Bu durum ciddi bir çakışmadır.
  </Card>
  <Card title="Önek Çakışması" icon="exclamation-triangle">
    Route'un path'i başka bir route'un path'inin öneki ile çakışmaktadır. Bu durum potansiyel bir sorun yaratabilir.
  </Card>
</CardGroup>

## Excel Export

Rapor sonuçlarını Excel formatında dışa aktarabilirsiniz. **Export** butonuna tıklayarak mevcut filtreleme kriterlerine göre oluşturulmuş raporu ZIP formatında indirebilirsiniz.

<Info>
Excel export işlemi mevcut filtreleme kriterlerine göre tüm sonuçları içerir.
</Info>

## Sıralama

Rapor tablosunda **Proje**, **Tip** ve **İsim** kolonlarına göre artan veya azalan şekilde sıralama yapabilirsiniz. Sıralama için kolon başlığına tıklayabilirsiniz.

<Note>
Bu rapor tüm route'ları analiz etmek için tasarlanmıştır. Pagination olmadan tüm sonuçlar gösterilir.
</Note>
