---

İstemci Trafiği & Zaman Metrikleri *(Client Traffic & Time Metrics)*, projedeki API Proxy'lerin **kullanıcı adı (credential username)** bilgisine göre gruplayarak kullanım özetlerine ait metrikleri sunmaktadır.

Kullanıcıya ait loglar sırasıyla, API Proxy ve API Proxy metotlarına göre de gruplandırılmaktadır. Böylelikle hangi kullanıcıların hangi API Proxy'leri kullandıkları, hangi uç noktaya ne kadar istek gönderdikleri gibi bilgilere ulaşılabilir.

Eğer kullanıcı adı olmadan istek gönderilirse bu isteğin kullanıcı adı **anonymous** olarak loglanır.

Sağ üst köşede bulunan excel tuşu ile verinin excel formatında dışa aktarılabilir.
