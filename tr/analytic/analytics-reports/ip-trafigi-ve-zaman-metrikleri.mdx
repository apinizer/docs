---


IP Trafiği & Zaman Metrikleri *(IP Traffic & Time Metrics)*, projedeki API Proxy'lerin IP adresi bilgisine göre gruplayarak kullanım özetlerine ait metriklerin yer aldığı rapordur.

IP adresine ait loglar sırasıyla API Proxy ve API Proxy metotlarına göre de gruplandırılmaktadır. Böylelikle hangi IP adresinin hangi API Proxy'leri kullandığı, hangi uç noktaya ne kadar istek gönderdiği gibi bilgilere ulaşılabilir.

Sağ üst köşede bulunan excel butonu ile veri excel formatında dışa aktarılabilir.

Bir API Proxy birden fazla ortama yüklenebildiği için metrikleri de yüklendiği ortama göre değerlendirmek gerekir. Bu sebeple tüm analitik ekranlarında öncelikle ortam bilgisi seçilerek metrikler filtrelenir.
