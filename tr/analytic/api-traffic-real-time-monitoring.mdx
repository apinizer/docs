## Genel Bakış

API Traffic Tab, API Proxy'nize gelen istekleri gerçek zamanlı olarak izlemenizi sağlar. Bu sekme özellikle:

* Gerçek zamanlı trafik izleme
* İstek/yanıt detaylarını inceleme
* Performans analizi
* Hata tespiti

için kullanılır.

## API Traffic Tab Kullanımı

### API Traffic Tab'a Erişim

API Traffic Tab'a erişmek için:

1. API Proxy detay sayfasına gidin
2. **API Traffic** sekmesine tıklayın
3. Trafik görüntülenir

### Gerçek Zamanlı İzleme

API Traffic Tab, gerçek zamanlı olarak gelen istekleri gösterir.

#### Real-time Updates

Gerçek zamanlı güncellemeler:

* **Auto Refresh**: Otomatik yenileme (varsayılan: 5 saniye)
* **Manual Refresh**: Manuel yenileme
* **Pause**: İzlemeyi duraklatma

#### Traffic List

Trafik listesi şu bilgileri içerir:

* **Timestamp**: İsteğin zamanı
* **Method**: HTTP metodu
* **Path**: İstek path'i
* **Status**: Yanıt durum kodu
* **Response Time**: Yanıt süresi
* **Size**: Yanıt boyutu
* **Client IP**: İstemci IP adresi

## Settings

API Traffic Tab ayarları, trafik görüntüleme davranışını kontrol eder.

### Display Settings

Görüntüleme ayarları:

* **Auto Refresh Interval**: Otomatik yenileme aralığı (saniye)
* **Max Visible Requests**: Görüntülenen maksimum istek sayısı
* **Column Visibility**: Kolon görünürlüğü
* **Row Height**: Satır yüksekliği

### Capture Settings

Yakalama ayarları:

* **Capture Request Body**: İstek gövdesini yakala
* **Capture Response Body**: Yanıt gövdesini yakala
* **Capture Headers**: Header'ları yakala
* **Max Body Size**: Maksimum gövde boyutu

### Filter Settings

Filtre ayarları:

* **Default Filters**: Varsayılan filtreler
* **Saved Filters**: Kaydedilmiş filtreler
* **Filter Presets**: Filtre ön ayarları

## Real-time Request/Response View

API Traffic Tab, her isteğin detaylı bilgilerini görüntüler.

### Request Details

İstek detayları:

#### Request Headers

İstek header'ları:

* **Standard Headers**: Standart HTTP header'ları
* **Custom Headers**: Özel header'lar
* **Authorization**: Kimlik doğrulama bilgileri

#### Request Query Parameters

Query parametreleri:

* **Parameter Name**: Parametre adı
* **Parameter Value**: Parametre değeri
* **URL Encoding**: URL encoding durumu

#### Request Body

İstek gövdesi:

* **Content Type**: İçerik tipi
* **Body Format**: Gövde formatı (JSON, XML, Form, Raw)
* **Body Content**: Gövde içeriği
* **Body Size**: Gövde boyutu

#### Request Metadata

İstek metadata'sı:

* **Request ID**: İstek ID'si
* **Client IP**: İstemci IP adresi
* **User Agent**: Kullanıcı aracısı
* **Timestamp**: Zaman damgası

### Response Details

Yanıt detayları:

#### Response Headers

Yanıt header'ları:

* **Status Code**: Durum kodu
* **Content Type**: İçerik tipi
* **Content Length**: İçerik uzunluğu
* **Custom Headers**: Özel header'lar

#### Response Body

Yanıt gövdesi:

* **Content Type**: İçerik tipi
* **Body Format**: Gövde formatı (JSON, XML, Text, Raw)
* **Body Content**: Gövde içeriği
* **Body Size**: Gövde boyutu

#### Response Metadata

Yanıt metadata'sı:

* **Response Time**: Yanıt süresi
* **Backend Time**: Backend süresi
* **Total Time**: Toplam süre
* **Status**: Durum

## Filtering & Search

API Traffic Tab, gelişmiş filtreleme ve arama özellikleri sunar.

### Filter Options

Filtre seçenekleri:

#### Method Filter

HTTP metodu filtresi:

* **GET**: GET istekleri
* **POST**: POST istekleri
* **PUT**: PUT istekleri
* **DELETE**: DELETE istekleri
* **All Methods**: Tüm metodlar

#### Status Filter

Durum kodu filtresi:

* **2xx**: Başarılı istekler
* **4xx**: İstemci hataları
* **5xx**: Sunucu hataları
* **Specific Status**: Belirli durum kodu

#### Path Filter

Path filtresi:

* **Exact Match**: Tam eşleşme
* **Prefix Match**: Önek eşleşmesi
* **Regex Match**: Regex eşleşmesi

#### Time Filter

Zaman filtresi:

* **Last 5 Minutes**: Son 5 dakika
* **Last 15 Minutes**: Son 15 dakika
* **Last Hour**: Son 1 saat
* **Custom Range**: Özel aralık

#### Client IP Filter

İstemci IP filtresi:

* **Specific IP**: Belirli IP adresi
* **IP Range**: IP aralığı
* **IP Pattern**: IP pattern'i

### Search

Arama özellikleri:

* **Full Text Search**: Tam metin arama
* **Header Search**: Header araması
* **Body Search**: Gövde araması
* **Advanced Search**: Gelişmiş arama

### Saved Filters

Kaydedilmiş filtreler:

* **Save Filter**: Filtreyi kaydet
* **Load Filter**: Filtreyi yükle
* **Delete Filter**: Filtreyi sil
* **Share Filter**: Filtreyi paylaş

## Performance Analysis

API Traffic Tab, performans analizi için çeşitli metrikler sağlar.

### Response Time Analysis

Yanıt süresi analizi:

* **Average Response Time**: Ortalama yanıt süresi
* **Median Response Time**: Medyan yanıt süresi
* **P95 Response Time**: 95. percentile yanıt süresi
* **P99 Response Time**: 99. percentile yanıt süresi
* **Min/Max Response Time**: Min/Max yanıt süresi

### Request Volume Analysis

İstek hacmi analizi:

* **Requests per Second**: Saniye başına istek sayısı
* **Requests per Minute**: Dakika başına istek sayısı
* **Peak Traffic**: Zirve trafik
* **Traffic Pattern**: Trafik deseni

### Error Analysis

Hata analizi:

* **Error Rate**: Hata oranı
* **Error Types**: Hata türleri
* **Error Endpoints**: Hata oluşan endpoint'ler
* **Error Trends**: Hata trendleri

### Backend Performance

Backend performans analizi:

* **Backend Response Time**: Backend yanıt süresi
* **Backend Error Rate**: Backend hata oranı
* **Backend Availability**: Backend kullanılabilirliği

## Request Details View

Bir isteğe tıklayarak detaylı bilgileri görüntüleyebilirsiniz.

### Request/Response Tabs

İstek/yanıt sekmeleri:

* **Request**: İstek detayları
* **Response**: Yanıt detayları
* **Headers**: Header detayları
* **Timing**: Zamanlama bilgileri
* **Policies**: Politika çalıştırma detayları

### Request Flow

İstek akışı:

* **Pre-flow Policies**: Pre-flow politikaları
* **Conditional Policies**: Koşullu politikalar
* **Route Step**: Route adımı
* **Post-flow Policies**: Post-flow politikaları
* **Fault Handler**: Hata yakalayıcı

### Policy Execution

Politika çalıştırma detayları:

* **Policy Name**: Politika adı
* **Execution Time**: Çalıştırma süresi
* **Status**: Durum
* **Changes**: Yapılan değişiklikler

## Export & Sharing

API Traffic Tab, trafik verilerini export etmenizi ve paylaşmanızı sağlar.

### Export Options

Export seçenekleri:

* **Export to CSV**: CSV formatında export
* **Export to JSON**: JSON formatında export
* **Export Selected**: Seçili istekleri export et
* **Export All**: Tüm istekleri export et

### Sharing

Paylaşım seçenekleri:

* **Share Link**: Paylaşım linki oluştur
* **Export Report**: Rapor export et
* **Email Report**: Raporu e-posta ile gönder

## Kullanım Senaryoları

### Senaryo 1: Gerçek Zamanlı İzleme

Gerçek zamanlı trafik izleme:

1. API Traffic Tab'ı açın
2. Auto Refresh'i aktif edin
3. Gelen istekleri izleyin
4. Anormal durumları tespit edin

### Senaryo 2: Hata Analizi

Hata analizi:

1. Status filtresini 4xx/5xx olarak ayarlayın
2. Hatalı istekleri inceleyin
3. Hata detaylarını analiz edin
4. Hata nedenlerini belirleyin

### Senaryo 3: Performans İzleme

Performans izleme:

1. Response time filtresini yüksek değerlere ayarlayın
2. Yavaş istekleri belirleyin
3. Performans metriklerini analiz edin
4. Optimizasyon yapın

## En İyi Uygulamalar

### Monitoring

* **Regular Monitoring**: Düzenli izleme yapın
* **Alerting**: Anormal durumlar için alert kurun
* **Baseline**: Performans baseline'ı oluşturun
* **Documentation**: Sonuçları dokümante edin

### Filtering

* **Relevant Filters**: İlgili filtreleri kullanın
* **Saved Filters**: Sık kullanılan filtreleri kaydedin
* **Performance**: Filtreleme performansını optimize edin

### Analysis

* **Trend Analysis**: Trend analizi yapın
* **Comparison**: Zaman içinde karşılaştırma yapın
* **Root Cause**: Kök neden analizi yapın
