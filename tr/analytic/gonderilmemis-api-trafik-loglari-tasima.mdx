---
title: "Gönderilmemiş API Trafik Logları Taşıma"
description: "Apinizer'da failover konnektörde biriken API trafik loglarını esas konnektörlere taşıma ve veri kaybını önleme hakkında bilgi edinin."
---

API Trafik Logları gönderildikleri ortamda tanımlanan konnektöre gönderilemediğinde, eğer konnektör ayarında failover konnektör kullanılsın seçeneği aktifse, failover konnektöre gönderilir.

Failover konnektöre gönderilen verilerin esas konnektör düzeltildikten sonra, esas konnektördeki hedefe taşınması ihtiyacı vardır.

<Note>
Bu ekran failover konnektör olarak **Apinizer konfigürasyon veri tabanı seçilmiş ise** Apinizer konfigürasyon veri tabanında biriken verileri esas gitmesi gereken konnektörlere taşımak için kullanılır.
</Note>

Bu işlem manuel olarak başlatılır ve eğer esas konnektörde hata alınırsa işlem veri kaybı olmayacak şekilde yarıda kesilir. Daha sonra yine bu ekrandan tekrar başlatılabilir.
