---
title: "Gönderilmemiş API Trafik Logları Taşıma"
description: "Failover konnektörde biriken API trafik loglarını esas konnektörlere taşıyabilirsiniz ve veri kaybını önleyebilirsiniz. Apinizer konfigürasyon veri tabanında biriken verileri esas konnektörlere taşıyabilirsiniz ve manuel olarak taşıma işlemini başlatabilirsiniz."
---


<Note>
Bu ekran failover konnektör olarak **Apinizer konfigürasyon veri tabanı seçilmiş ise** Apinizer konfigürasyon veri tabanında biriken verileri esas gitmesi gereken konnektörlere taşımak için kullanılır.
</Note>

Bu işlem manuel olarak başlatılır ve eğer esas konnektörde hata alınırsa işlem veri kaybı olmayacak şekilde yarıda kesilir. Daha sonra yine bu ekrandan tekrar başlatılabilir.


API Trafik Logları gönderildikleri ortamda tanımlanan konnektöre gönderilemediğinde, eğer konnektör ayarında failover konnektör kullanılsın seçeneği aktifse, failover konnektöre gönderilir. Failover konnektöre gönderilen verilerin esas konnektör düzeltildikten sonra, esas konnektördeki hedefe taşınması ihtiyacı vardır.
