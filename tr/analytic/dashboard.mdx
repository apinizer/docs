Analytics Dashboard, API Proxy'lerinizin performansını ve kullanımını görselleştirmenizi sağlayan bir analiz aracıdır. Bu dashboard sayesinde, API trafiği, hata oranları, yanıt süreleri ve diğer önemli metrikleri takip edebilirsiniz.

## Genel Bakış

Analytics Dashboard, API Proxy'lerinizin performans metriklerini görselleştirmek için kullanılan bir araçtır. Bu dashboard özellikle:

* API trafiğini izlemek
* Performans metriklerini analiz etmek
* Hata oranlarını takip etmek
* Kullanım istatistiklerini görüntülemek

için kullanılır.

## Dashboard Widget'ları

Dashboard, çeşitli widget'lardan oluşur. Her widget belirli bir metriği görüntüler.

### Widget Türleri

#### Line Chart

Zaman içindeki değişimi gösteren çizgi grafik:

* **Request Count**: İstek sayısı zaman içinde
* **Response Time**: Yanıt süresi zaman içinde
* **Error Rate**: Hata oranı zaman içinde

#### Bar Chart

Kategorilere göre karşılaştırma yapan çubuk grafik:

* **Requests by Method**: Metoda göre istek sayısı
* **Requests by Status**: Durum koduna göre istek sayısı
* **Top Endpoints**: En çok kullanılan endpoint'ler

#### Pie Chart

Dağılımı gösteren pasta grafik:

* **Status Code Distribution**: Durum kodu dağılımı
* **Method Distribution**: Metod dağılımı
* **Error Type Distribution**: Hata türü dağılımı

#### Metric Card

Tek bir metriği gösteren kart:

* **Total Requests**: Toplam istek sayısı
* **Average Response Time**: Ortalama yanıt süresi
* **Error Rate**: Hata oranı
* **Success Rate**: Başarı oranı

#### Table

Detaylı veri tablosu:

* **Top Endpoints**: En çok kullanılan endpoint'ler
* **Error Details**: Hata detayları
* **Client Details**: İstemci detayları

### Widget Yapılandırması

Widget'ları yapılandırmak için:

1. Widget'a tıklayın
2. **Configure** butonuna tıklayın
3. Metrik türünü seçin
4. Zaman aralığını ayarlayın
5. Filtreleri uygulayın
6. **Save** butonuna tıklayın

### Widget Düzenleme

Dashboard'u düzenlemek için:

* **Add Widget**: Yeni widget ekle
* **Remove Widget**: Widget'ı kaldır
* **Resize Widget**: Widget boyutunu değiştir
* **Move Widget**: Widget'ı taşı
* **Configure Widget**: Widget'ı yapılandır

## Metric Türleri

Dashboard'da görüntülenebilen çeşitli metrik türleri bulunur.

### Trafik Metrikleri

#### Request Count

Toplam istek sayısı:

* **Total Requests**: Toplam istek sayısı
* **Requests per Second**: Saniye başına istek sayısı
* **Requests per Minute**: Dakika başına istek sayısı
* **Requests per Hour**: Saat başına istek sayısı

#### Request Volume

İstek hacmi:

* **Request Size**: İstek boyutu
* **Response Size**: Yanıt boyutu
* **Total Data Transfer**: Toplam veri transferi

### Performans Metrikleri

#### Response Time

Yanıt süresi metrikleri:

* **Average Response Time**: Ortalama yanıt süresi
* **Median Response Time**: Medyan yanıt süresi
* **P95 Response Time**: 95. percentile yanıt süresi
* **P99 Response Time**: 99. percentile yanıt süresi
* **Min Response Time**: Minimum yanıt süresi
* **Max Response Time**: Maksimum yanıt süresi

#### Backend Performance

Backend performans metrikleri:

* **Backend Response Time**: Backend yanıt süresi
* **Backend Connection Time**: Backend bağlantı süresi
* **Backend Error Rate**: Backend hata oranı

### Hata Metrikleri

#### Error Rate

Hata oranı metrikleri:

* **Total Errors**: Toplam hata sayısı
* **Error Rate**: Hata oranı (%)
* **4xx Errors**: İstemci hataları
* **5xx Errors**: Sunucu hataları

#### Error Details

Hata detayları:

* **Error Types**: Hata türleri
* **Error Messages**: Hata mesajları
* **Error Endpoints**: Hata oluşan endpoint'ler

### Kullanım Metrikleri

#### Endpoint Usage

Endpoint kullanım metrikleri:

* **Top Endpoints**: En çok kullanılan endpoint'ler
* **Endpoint Request Count**: Endpoint istek sayısı
* **Endpoint Success Rate**: Endpoint başarı oranı

#### Client Usage

İstemci kullanım metrikleri:

* **Top Clients**: En çok istek gönderen istemciler
* **Client Request Count**: İstemci istek sayısı
* **Client Error Rate**: İstemci hata oranı

## Zaman Aralığı Seçimi

Dashboard'da farklı zaman aralıkları seçebilirsiniz.

### Önceden Tanımlı Aralıklar

* **Last 15 Minutes**: Son 15 dakika
* **Last Hour**: Son 1 saat
* **Last 24 Hours**: Son 24 saat
* **Last 7 Days**: Son 7 gün
* **Last 30 Days**: Son 30 gün
* **Last 90 Days**: Son 90 gün

### Özel Zaman Aralığı

Özel bir zaman aralığı seçmek için:

1. **Custom Range** seçeneğini seçin
2. Başlangıç tarihini seçin
3. Bitiş tarihini seçin
4. **Apply** butonuna tıklayın

### Zaman Granularity

Zaman granularity seçenekleri:

* **1 Minute**: 1 dakika
* **5 Minutes**: 5 dakika
* **15 Minutes**: 15 dakika
* **1 Hour**: 1 saat
* **1 Day**: 1 gün

## Karşılaştırmalı Analiz

Dashboard, farklı zaman aralıklarını karşılaştırmanızı sağlar.

### Zaman İçinde Karşılaştırma

İki zaman aralığını karşılaştırma:

* **Current Period**: Mevcut dönem
* **Previous Period**: Önceki dönem
* **Comparison**: Karşılaştırma sonuçları

### Trend Analizi

Trend analizi:

* **Increasing**: Artış trendi
* **Decreasing**: Azalış trendi
* **Stable**: Stabil trend
* **Fluctuating**: Dalgalı trend

## Custom Dashboard Oluşturma

Özel dashboard'lar oluşturabilirsiniz.

### Dashboard Oluşturma

Yeni bir dashboard oluşturmak için:

1. **Create Dashboard** butonuna tıklayın
2. Dashboard adını girin
3. Açıklama ekleyin
4. Widget'ları ekleyin
5. **Save** butonuna tıklayın

### Dashboard Yönetimi

Dashboard'ları yönetmek için:

* **List**: Tüm dashboard'ları listele
* **Edit**: Dashboard'u düzenle
* **Delete**: Dashboard'u sil
* **Duplicate**: Dashboard'u kopyala
* **Share**: Dashboard'u paylaş

## Kullanım Senaryoları

### Senaryo 1: Günlük Performans İzleme

Günlük performans izleme için dashboard:

**Widget'lar:**
* Total Requests (Metric Card)
* Average Response Time (Line Chart)
* Error Rate (Line Chart)
* Top Endpoints (Bar Chart)

**Zaman Aralığı:** Last 24 Hours

### Senaryo 2: Hata Analizi

Hata analizi için dashboard:

**Widget'lar:**
* Error Rate (Line Chart)
* Error Types (Pie Chart)
* Error Details (Table)
* Error Endpoints (Bar Chart)

**Zaman Aralığı:** Last 7 Days

### Senaryo 3: Endpoint Performansı

Endpoint performansı için dashboard:

**Widget'lar:**
* Top Endpoints (Bar Chart)
* Endpoint Response Time (Line Chart)
* Endpoint Success Rate (Line Chart)
* Endpoint Details (Table)

**Zaman Aralığı:** Last 30 Days

## En İyi Uygulamalar

### Dashboard Tasarımı

* **Relevance**: İlgili metrikleri gösterin
* **Clarity**: Açık ve anlaşılır widget'lar kullanın
* **Organization**: Widget'ları mantıklı şekilde düzenleyin
* **Performance**: Performansı etkilemeyen widget sayısı

### Metric Seçimi

* **Key Metrics**: Ana metrikleri seçin
* **Balance**: Farklı metrik türlerini dengeleyin
* **Actionable**: Aksiyon alınabilir metrikleri seçin

### Zaman Aralığı

* **Context**: Bağlama uygun zaman aralığı seçin
* **Comparison**: Karşılaştırma için uygun aralık seçin
* **Granularity**: Uygun granularity seçin
