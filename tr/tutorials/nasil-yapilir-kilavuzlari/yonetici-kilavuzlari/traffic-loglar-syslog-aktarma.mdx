# Trafik Loglarının Syslog'a Aktarılması

Apinizer trafik loglarını syslog sunucusuna aktararak merkezi log yönetimi sağlayabilirsiniz.

## Genel Bakış

Apinizer trafik loglarının syslog'a aktarılması için Gateway Environment ayarlarından Syslog Connector yapılandırması yapılmalıdır.

## Yapılandırma

### 1. Syslog Bağlantı Tanımı

Administration > Bağlantı Konfigürasyonlar > Syslog sayfasına gidilir ve bir Syslog bağlantı tanımı oluşturulur.

### 2. Gateway Environment Ayarları

Gateway Environment ayarlarına gidilir ve trafik logları için Syslog Connector seçilir.

Daha önce oluşturulmuş olan Syslog bağlantı tanımı seçilir.

## Log Formatı

Gönderilecek trafik logları için veri yapısı JSON formatında olacaktır ve aşağıdaki alanları içerir:

- **id**: Log kaydının benzersiz kimliği
- **timestamp**: İsteğin gerçekleştiği zaman damgası
- **apinizerCorrelationId**: İstek için korelasyon ID'si
- **environmentId**: Ortam kimliği
- **projectId**: Proje kimliği
- **apiProxyId**: API Proxy kimliği
- **apiProxyName**: API Proxy adı
- **httpRequestMethod**: HTTP metodu
- **httpRequestUrl**: İstek URL'si
- **httpRequestRemoteAddress**: İstemci IP adresi
- **httpResponseStatusCode**: Yanıt durum kodu
- **timeTotal**: Toplam işlem süresi (ms)
