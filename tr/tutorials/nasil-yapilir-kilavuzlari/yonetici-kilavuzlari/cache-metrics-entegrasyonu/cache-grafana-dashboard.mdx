# Apinizer Cache için Grafana Üzerinde Örnek Dashboardlar

Grafana Kurulumu ve Veri Kaynağı Olarak Prometheus Entegrasyonu için [ilgili dökümanımıza göz atabilirsiniz](/apinizer-dokumantasyonu/grafana-kurulumu-ve-veri-kaynagi-olarak-prometheus-entegrasyonu-126910830.html).

## Panel 1: Saniye Başına Önbellek Aramaları

**PromQL Sorguları:**

- A: `topk(5, sum by (cache) (rate(cache_gets_total{cache=~".*"}[2h])))`

**Görselleştirme:** Gauge

## Panel 2: En Yüksek Bellek Kullanımına Sahip Önbellek Özellikleri

**PromQL Sorguları:**

- A: `topk(5, sum by (cache) (cache_entry_memory_bytes))`

**Görselleştirme:** Bar gauge veya Bar chart

## Panel 3: Toplam API İsteği (1 saat için)

**PromQL Sorguları:**

- A: `sum(increase(apinizer_cache_api_requests_total[1h]))`

**Görselleştirme:** Bar chart veya stat
