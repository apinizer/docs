# pfx Dosyasının jks Dosyasına Çevrilmesi

Apinizer Worker uygulamalarının HTTPS ile açılması istendiğinde pfx ve jks sertifikalarını destekler. Ancak yinede kurumlar pfx sertifikalarını jks formatında yüklemek isteyebilirler.

Bu doküman bu duruma yardımcı olması için "Keystore Explorer" programı yardımıyla bu dönüşümün nasıl yapılabileceğini anlatır.

## Keystore Explorer (KSE) ile Sertifika Dosyalarını jks Formatına Çevirmek

### Adım 1: PFX Dosyasını Açma

pfx dosyası KSE ile açılır.

### Adım 2: Keypair Export Etme

İçindeki dosya sağ tık **export key pair** seçilir.

### Adım 3: PKCS12 Formatında Export

Keypair **pkcs12** formatında **p12** uzantısı ile export edilir.

### Adım 4: Yeni JKS Dosyası Oluşturma

Yeni dosya oluştur seçeneği ile **JKS** tipi seçilir.

### Adım 5: Keypair Import Etme

Oluşturulan dosyaya import keypair seçeneği ile az önce export edilen .p12 dosyası import edilir.

**Not:** İsteğe bağlı alias girilebilir. Dikkat edilmesi gereken aynı alias ile başka sertifika varsa **üzerine yazılmamalıdır**.

### Adım 6: Şifre Belirleme

Yeni şifre verilir, dikkat edilmesi gereken nokta verilen şifre ile jks şifresinin aynı olması eğer şifre yoksa jks'nin de şifresinin olmaması gerekliliğidir.

### Adım 7: Certificate Chain Export

Sonrasında import edilen keypair'e sağ tıklanır export certificate chain seçeneği seçilir.

Export edilirken dikkat edilmesi gereken **Entire Chain** ve **X.509** seçeneğinin seçilmesi gerekliliğidir.

### Adım 8: Certificate Chain Import

Sonrasında export edilen bu sertifika jksye import edilir. İsteğe bağlı alias girilebilir. Dikkat edilmesi gereken aynı alias ile sertifika varsa **üzerine yazılmamalıdır**.

### Adım 9: Dosyayı Kaydetme

Dosya keypair'e verilen şifre ile kaydedilerek kapatılır.
