## Kullanıma Başlama

MCP bağlantınızı yaptıktan ve çalışır bir token'ınız olduğuna emin olduktan sonra, direkt kullanıma başlayabilirsiniz.

### 1. API Keşfi

Örneğin şunu yazmayı deneyin:

<CodeGroup>
```bash
Apinizer API portaline mcp_xxx tokenı ile mcp bağını kullanarak hangi apilere erişebilirim kontrol edebilir misin?
```
</CodeGroup>

### 2. API Detaylarını İnceleme

Cevapları inceledikten sonra şu şekilde ilerleyebilirsiniz:

<CodeGroup>
```bash
1. api'nin detaylarını paylaşır mısın?
```
</CodeGroup>

Burada servisin detaylarını inceliyoruz, uygun görüyorsak istediğimiz şekilde devam etmesini söylüyoruz.

### 3. API Test Etme

<CodeGroup>
```bash
Evet bu servisi test edebilir misin? Ücretsiz plana kaydolalım ve toplama işlemi ile 3 ve 5i toplatalım.
```
</CodeGroup>

Burada servisi test ediyor, bir hata buluyor, hatayı anlamlandırıyor ve düzeltmek için alternatif bir endpoint'e istek atıyor.

Doğru adresi bulduktan sonra servisin güvenlik yöntemine uygun http header'ını da ekleyerek isteği tekrarlıyor ve başarılı cevap alıyor.
