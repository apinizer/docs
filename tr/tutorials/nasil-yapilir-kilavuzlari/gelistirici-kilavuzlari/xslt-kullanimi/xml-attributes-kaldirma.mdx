---
title: "Bir XML Dosyasından Tüm Attributes'ları Kaldırmak İçin Gereken XSLT"
description: "XSLT kullanarak XML dosyasındaki tüm attribute'ları kaldırabilir. XML elemanlarının tüm özelliklerini (attributes) temizleyerek sadece içerik ve yapıyı koruyabilir."
---

```xml
<xsl:stylesheet xmlns:xsl="http://www.w3.org/1999/XSL/Transform" version="1.0">
  <xsl:template match="*/@*" />
  <xsl:template match="@*|node()">
    <xsl:copy>
      <xsl:apply-templates select="@*|node()" />
    </xsl:copy>
  </xsl:template>
</xsl:stylesheet>
```
