# Bir XML Dosyasından Tüm "attributes"ları Kaldırmak İçin Gereken XSLT

Bu sayfa, XSLT kullanarak bir XML dosyasındaki tüm attribute'ları kaldırma işlemini açıklamaktadır.

```xml
<xsl:stylesheet xmlns:xsl="http://www.w3.org/1999/XSL/Transform" version="1.0">
  <xsl:template match="*/@*" />
  <xsl:template match="@*|node()">
    <xsl:copy>
      <xsl:apply-templates select="@*|node()" />
    </xsl:copy>
  </xsl:template>
</xsl:stylesheet>
```
