# Veri Tipi Olarak String Belirtilen Key Değerlerini Numerik Veri Tipine Çevirme

Bu örnekte **kisiId** key ifadesi ve **uyeId** key ifadesinin veri tipinin numerik bir veri tipi olan **Long** veri tipine dönüştürülmesine yer verilmiştir.

**Örnek Giriş Değeri**

```json
{
  "kisiId": "123",
  "uyeId": "456",
  "adi": "xxx",
  "soyadi": "yyy"
}
```

**JOLT Dönüşüm Kodu**

```json
[
  {
    "operation": "modify-overwrite-beta",
    "spec": {
      "kisiId": "=toLong",
      "uyeId": "=toLong"
    }
  }
]
```

**Örnek Çıkış Değeri**

```json
{
  "kisiId": 123,
  "uyeId": 456,
  "adi": "xxx",
  "soyadi": "yyy"
}
```
