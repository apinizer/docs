# İstenilen Alanı Silme

Bu örnekte JSON verisi içerisinde yer alan "columnNames" isimli key değerinin nasıl silineceği gösterilmiştir.

**Örnek Giriş Değeri**

```json
{ "statusCode":200, "statusDescription":"OK", "columnNames":[ "Id", "UstBirimId", "Adi", "TeskilatId", "Yol", "SehirId", "BirimGrupId", "no" ], "numRows":1201 }
```

**JOLT Dönüşüm Kodu**

```json
[ { "operation" : "remove", "spec" : { "columnNames" : "" } } ]
```

**Örnek Çıkış Değeri**

```json
{ "statusCode" : 200, "statusDescription" : "OK", "numRows" : 1201 }
```
