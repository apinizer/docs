## Data Elementinin İsmini Değiştirme

"data" elementinin ismini değiştiren ve diğer elemanlardan temizleyen JOLT örneği:

```json
[ { "operation": "shift", "spec": { "data":"sonuç" } } ]
```

## Country Alanını Üst Seviyeye Çıkarma

"data" elementinin içindeki "country" elemanlarını bir üst seviyeye çıkartan ve diğer elemanlardan temizleyen JOLT örneği:

```json
[ { "operation": "shift", "spec": { "data": { "*": { "country": "&" } } } } ]
```

**Çıktı:**

```json
{ "country": [ "France", "France" ] }
```

## Country Değerlerini Dizi Olarak Çıkarma

"data" elementinin içindeki "country" elemanlarının değerini bir üst seviyeye çıkartan ve diğer elemanlardan temizleyen JOLT örneği:

```json
[ { "operation": "shift", "spec": { "data": { "*": { "country": "[&1]." } } } } ]
```

**Çıktı:**

```json
[ "France", "France" ]
```
