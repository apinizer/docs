# DB2API'de Procedure Sonucunun Düzenlenmesi

DB2API'de Procedure çağrımı yapıldığında dönen result değeri:

```json
{
  "statusCode": 200,
  "statusDescription": "OK",
  "numRows": 0,
  "data": [ {} ],
  "result": [
    {
      "data": {
        "columnNames": [ "xxx", "yyy" ],
        "numRows": 2,
        "result": [
          { "xxx":"aaa", "yyy":"bbb" },
          { "xxx":"ccc", "yyy":"ddd" }
        ]
      }
    }
  ],
  "elapsedTime": 6,
  "errors": []
}
```

**JOLT: result değerini select sonucuna benzetme (sadece bir sonuç kümesi dönüyorsa bu işlem anlamlıdır)**

```json
[
  {
    "operation": "shift",
    "spec": {
      "statusCode":"statusCode",
      "statusDescription":"statusDescription",
      "result": {
        "*": {
          "data": {
            "columnNames":"columnNames",
            "numRows":"numRows",
            "result": "data"
          }
        }
      },
      "elapsedTime":"elapsedTime",
      "errors":"errors"
    }
  }
]
```
