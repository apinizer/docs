---
title: "Belli Bir Alan Dizi Türüne Dönüştürülmesi"
description: "JSON verisi içindeki belirli alanları dizi türüne dönüştürebilir. Bu sayfa, cardinality operasyonu kullanarak string değerleri dizi formatına çevirme örneğini içerir."
---

**Giriş Değeri**

```json
{ "statusCode" : 200, "statusDescription" : "OK", "numRows" : 1, "data" : [ { "data" : { "Alan1" : "içerik1", "Alan2" : "içerik2", "Alan3" : 1, "Alan4" : "içerik3", "Alan5" : "içerik4", "NewField": "test" } } ], "elapsedTime" : 1, "errors" : [ ] }
```

**JOLT Dönüşüm Kodu**

```json
[ { "operation": "cardinality", "spec": { "data": { "*": { "data": { "Alan1": "MANY" } } } } }, { "operation": "cardinality", "spec": { "data": { "*": { "data": { "Alan2": "MANY" } } } } } ]
```

**Çıkış Değeri**

```json
{ "statusCode" : 200, "statusDescription" : "OK", "numRows" : 1, "data" : [ { "data" : { "Alan1" : [ "içerik1" ], "Alan2" : [ "içerik2" ], "Alan3" : 1, "Alan4" : "içerik3", "Alan5" : "içerik4", "NewField" : "test" } } ], "elapsedTime" : 1, "errors" : [ ] }
```
