# Yeni Bir Key İfadesi Ekleme ve Var Olan Bir Key İfadesini Silme

Bu örnekte var olan JSON verisi içerisine yeni bir **key ifadesinin eklenmesi** ve var olan bir **key ifadesinin silinmesi** gösterilecektir.

**Örnek Giriş Değeri**

```json
{
  "Rating": 1,
  "SecondaryRatings": {
    "Design": 4,
    "Price": 2,
    "RatingDimension3": 1
  }
}
```

**JOLT Dönüşüm Kodu**

```json
[
  {
    "operation": "remove",
    "spec": {
      "Rating": ""
    }
  },
  {
    "operation": "default",
    "spec": {
      "NewlyAddedField": "123"
    }
  }
]
```

**Örnek Çıkış Değeri**

```json
{
  "SecondaryRatings" : {
    "Design" : 4,
    "Price" : 2,
    "RatingDimension3" : 1
  },
  "NewlyAddedField" : "123"
}
```
