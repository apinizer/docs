---
title: "String Tipindeki Key Değerlerini Numerik Veri Tipine Çevirme"
description: "String tipindeki key değerlerini numerik veri tipine dönüştürebilir. Bu sayfa, modify-overwrite-beta operasyonu ile toLong fonksiyonunu kullanarak string değerleri Long tipine çevirme örneğini içerir."
---

**Giriş Değeri**

```json
{
  "kisiId": "123",
  "uyeId": "456",
  "adi": "xxx",
  "soyadi": "yyy"
}
```

**JOLT Dönüşüm Kodu**

<Tip>
  `=toLong` ifadesi, string değerleri Long tipine dönüştürür. Eğer değer geçerli bir sayı değilse hata oluşabilir.
</Tip>

```json
[
  {
    "operation": "modify-overwrite-beta",
    "spec": {
      "kisiId": "=toLong",
      "uyeId": "=toLong"
    }
  }
]
```

**Çıkış Değeri**

```json
{
  "kisiId": 123,
  "uyeId": 456,
  "adi": "xxx",
  "soyadi": "yyy"
}
```
