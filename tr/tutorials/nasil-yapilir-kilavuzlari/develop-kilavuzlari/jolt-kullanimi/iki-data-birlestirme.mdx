  "statusCode": 200,

  "numRows": 5521,
  "data": [
    {
      "xTur": "BASİT 1",
      "xLoc": "TERMAL 1"
    },

      "xTur": "BASİT 2",

    {

### JOLT Spesifikasyonu

```json
[
  {
    "operation": "shift",
    "spec": {
      "data": {
        "*": {
          "@": "data.[&1]"
        }
      },
      "xServis": {
        "*": {
          "@": "data.[&1]"
        }
      }
    }
  }
]
```
