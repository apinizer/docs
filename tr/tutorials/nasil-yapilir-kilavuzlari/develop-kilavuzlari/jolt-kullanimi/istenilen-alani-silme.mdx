---
title: "İstenilen Alanı Silme"
description: "JSON verisi içindeki belirli alanları silebilir. Bu sayfa, remove operasyonu kullanarak gereksiz alanları temizleme örneğini içerir."
---

Bu örnekte JSON verisi içerisinde yer alan "columnNames" isimli key değerinin nasıl silineceği gösterilmiştir.

### Giriş Değeri

```json
{
  "statusCode": 200,
  "statusDescription": "OK",
  "columnNames": [
    "Id",
    "UstBirimId",
    "Adi",
    "TeskilatId",
    "Yol",
    "SehirId",
    "BirimGrupId",
    "no"
  ],
  "numRows": 1201
}
```

### JOLT Spesifikasyonu

```json
[
  {
    "operation": "remove",
    "spec": {
      "columnNames": ""
    }
  }
]
```

### Çıkış Değeri

```json
{
  "statusCode": 200,
  "statusDescription": "OK",
  "numRows": 1201
}
```
