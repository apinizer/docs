---
title: 'Application Logs'
description: 'API Proxy için gelen ve dönen mesajlar için akışın işletilmesi sırasında oluşan yazılımsal loglara ait kayıtlar'
---

Bu sekmede, ilgili API Proxy'e gelen ve dönen mesajlar için akışın işletilmesi sırasında oluşan yazılımsal loglara ait kayıtlar görüntülenir.

Bu sayfanın daha çok yazılımsal bir hata olduğundan şüphe edildiğinde veya uygulamanın arka planda nasıl davrandığı araştırılmak istenildiğinde kullanılması uygundur.

**Uygulama Logları sekmesini içeren görsele aşağıda yer verilmiştir:**

<img src="https://docs.apinizer.com/apinizer-dokumantasyonu/files/26347973/154697776/1/1763453817147/image-2025-11-18_11-16-56.png" alt="Uygulama Logları Sekmesi" width="1000" />

Uygulama Logları üzerinde yapılan işlemler ve filtreleme için kullanılan alanlar aşağıdaki tabloda görülmektedir.

| Alan | Açıklama |
| --- | --- |
| Başlat *(Start)* | Uygulama loglarının API Proxy bazında kaydının tutulmasını sağlar. |
| Ortam Filtresi *(Environment)* | Uygulama loglarının belirtilen ortam için filtrelenmesini sağlar. |
| Zaman Filtresi *(From, To)* | Uygulama loglarının belirtilen tarih aralığı için filtrelenmesini sağlar. |
| Log Seviyesi *(Search by Log Levels)* | Uygulama loglarında "Log Seviyesi" ile belirtilen seviyede olan logların filtrelenmesini sağlar. |
| Correlation ID'ye göre ara *(Search in Correlation ID)* | API Proxy'ye bir istek geldiği anda, isteğin takip edilebilmesi için Apinizer tarafından bu isteğe bir korelasyon id'si tanımlanır. API Proxy Trafik loglarından bulunacak olan korelasyon id ile uygulama loglarında detaylı arama yapılarak sadece bir isteğin uygulam loglarına erişilebilir. |
| Detaya göre ara *(Search in Detail)* | Uygulama loglarının metin alanlarında arama yapılmasını sağlar. |
| Stack Trace'e göre ara *(Search in Stack Trace)* | Uygulama loglarında exception oluşmuş ise exception metni üzerinde arama yapılmasını sağlar. |
| İndir Tuşu *(Download)* | Uygulama loglarının tek bir dosya olarak indirilmesini sağlar. Eğer "Hepsi" seçeneği seçili ise bu API Proxy ile ilgili tüm loglar indirilir. Eğer "Arama Sonuçları" seçeneği seçili ise filtrelenen tüm loglar indirilir. |
| Sil Tuşu *(Delete)* | Bu API Proxy'e ait uygulama log kayıtlarının silinmesini sağlar. |
