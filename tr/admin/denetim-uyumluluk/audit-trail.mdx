# Denetim Kayıtları

Denetim Kayıtları kullanıcıların sistem üzerinde yapılan değişiklikleri görmelerine olanak sağlar. Gerçekleştirilen işlemler tarih, operasyon tipi, kullanıcı, nesne id, nesne adı ve denetim objesinin tipi parametreleriyle loglanır. Sayfanın üst tarafında filtre bölümünden kullanıcılar kriterler belirleyerek sonucları log tablosunda görüntüleyebilirler. Silme butonu ile filtre içerikleri temizlenir.

**İlgili sayfayı içeren görsele aşağıda yer verilmiştir:**

![Denetim Kayıtları](https://docs.apinizer.com/apinizer-dokumantasyonu/files/1180203/39747107/1/1682492569042/image2023-4-26_10-2-59.png)

"Değişiklik Detayı" ekranı verilerin sağındaki göz simgeli detay butonuna tıklanarak görüntülenir. Sayfa sağladığı diff editor aracılığıyla kullanıcının sistem üzerindeki yapılan değişikleri eski versiyon ve yeni versiyon şeklinde sırasıyla görmesine olanak sağlar.
