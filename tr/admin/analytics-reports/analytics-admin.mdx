Apinizer, API Proxy çağrıları hakkında birçok metrik verisini loglamaktadır. Analitik modülünde;

- Log verileri ile API'lerin yükünü ve performansını izleyebilir, durumu hakkında hızlıca durum tespiti yapılabilir,
- Loglara gelişmiş filtreler uygulanabilir,
- Sistemleri detaylı izleyebilmek için özel raporlar ve özel panolar (dashboards) oluşturulabilir.

Bu bölümde proje bazlı analitikler anlatılmaktadır. Herhangi bir analitiğe ait detaylı bilgi için kendi sayfasına bakılabilir.

## Analitik Modülleri

- [Pano (Dashboard)](/pano)
- [API Trafiği (API Traffic)](/api-trafigi)
- [Sorgu Editörü (Query Editor)](/sorgu-editoru)
  - [Sorgular (Queries)](/sorgular)
  - [Filtreler (Filters)](/filtreler)
- [Gönderilmemiş API Trafik Loglarını Taşı](/gonderilmemis-api-trafik-loglarini-tasi)

<Warning>
- Apinizer Yönetim Konsolunda ilgili API Trafiğinin sorgulanması ve analitik olarak görselleştirilmesi ancak ilgili ortama Elasticsearch Konnektörü eklenmesi ile mümkün olmaktadır.
- Eğer birden fazla Elasticsearch Konnektörü tanımlanması durumunda Apinizer Yönetim Konsolunda **ilk** eklenen Elasticsearch Konnektöründeki veriler görüntülenebilir.
</Warning>
