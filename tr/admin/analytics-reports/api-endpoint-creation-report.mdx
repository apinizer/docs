API Endpoint Oluşturma Raporu ekranında, oluşturulan API Endpoint'lerin belirlenen filtre alanlarına göre grafik oluşturulur ya da listelenir. Bu ekranın iki farklı görünümü bulunmaktadır; **grafik ve liste.**

Grafik görünümü filtrelerden belirlenen tarih aralığında aylık kaç adet API Endpoint oluşturulduğu bilgisini grafik olarak verir.
