Endpoint Trafiği & Zaman Metrikleri *(Endpoint Traffic & Time Metrics)*, projedeki API Proxy'lerin **metot/uç nokta** bilgisine göre gruplayarak kullanım özetlerine ait metrikler yer almaktadır.

**Ekran detaylarını içeren görsele aşağıda yer verilmiştir:**

![Endpoint Traffic Time Metrics](/images/endpoint-traffic-time-metrics.png)

<Info>
Bir API Proxy birden fazla ortama yüklenebildiği için metrikleri de yüklendiği ortama göre değerlendirmek gerekir. Bu sebeple tüm analitik ekranlarında öncelikle ortam bilgisi seçilerek metrikler filtrelenir.
</Info>
