## Filtreleme Seçenekleri

Gelişmiş filtreleme konfigürasyonu için kullanılan alanlar aşağıdaki tabloda görülmektedir.

| Alan | Açıklama |
|------|----------|
| **Ortam** _(Environment)_ | Ortam değişkeni bu filtre alanından seçilen değeri içeren Token Kayıtları gelir. |
| **Proje** _(Project)_ | Proje değeri bu filtre alanından seçilen değer(ler)i içeren Token Kayıtları gelir. |
| **Tarih Aralığı** _(Date Range)_ | Token Kayıtları'nı seçilen tarih aralığına göre filtreler. |
| **Proxy** _(API Proxy)_ | API Proxy bilgisi bu filtre alanından seçilen değeri içeren Token Kayıtları gelir. |
| **Kullanıcı Adı** _(Username or Key)_ | Kullanıcı adı bu filtre alanına girilen değeri içeren Token Kayıtları gelir. |
| **İstemciye Gönderilen Yanıtın Gövdesi** _(To Client Body)_ | İstemciye dönen yanıt gövdesi içinde bu filtre alanına girilen değeri içeren Token Kayıtları gelir. |
| **Durum Kodu Büyüktür** _(Status Code Bigger Than)_ | Durum kodu bu filtre alanına girilen değerden büyük değerleri içeren Token Kayıtları gelir. |
| **Durum Kodu Küçüktür** _(Status Code Smaller Than)_ | Durum kodu bu filtre alanına girilen değerden küçük değerleri içeren Token Kayıtları gelir. |
