<Info>
Sunucu Yönetimi, Apinizer platformunda API Proxy'lerin çalışacağı ortamları oluşturmak ve yönetmek, Kubernetes kaynaklarını yönetmek, Gateway Engine ve Cache sunucu yapılandırması yapmak, SSL/TLS sertifikaları yapılandırmak ve API trafik log konnektörlerini yönetmek için gerekli tüm araçları sunar.
</Info>

## Modülün Temel Bileşenleri

Sunucu Yönetimi modülü, Apinizer platformunun altyapı ve ortam yönetimini sağlar. Bu modül ile:

- API Proxy'lerin çalışacağı ortamlar oluşturulur ve yönetilir
- Kubernetes kaynakları ve deployment'lar yönetilir
- Gateway Engine ve Cache sunucu yapılandırmaları yapılır
- SSL/TLS sertifikaları yapılandırılır
- API trafik log konnektörleri yönetilir
