<Info>
**Yönetim → Sistem Ayarları → Sistem Özellikleri** yolunu izleyerek Sistem Özellikleri ekranına erişilir. Değişikliklerinizi kaydetmek için **Değişiklikleri Uygula (Apply Changes)** butonuna tıklayınız.
</Info>

Sistem Özellikleri ekranı, API Gateway üzerinde yeni bir API Proxy oluşturulurken veya yapılandırılırken kullanılan varsayılan değerleri tek bir merkezden yönetmenizi sağlar. Bu sayede her API Proxy için ayrı ayrı yapılandırma yapmak yerine, platform genelinde geçerli olacak standart değerleri belirleyebilirsiniz.

## Ekran Özellikleri

| Özellik | Açıklama |
|---------|----------|
| Kategorize Edilmiş Yapı | Ayarlar sekme ve kategori bazında gruplandırılmıştır. İlgili parametrelere hızlıca ulaşabilirsiniz. |
| Arama | Tüm özellikler arasında isim veya açıklamaya göre arama yapabilirsiniz. |
| Varsayılana Sıfırlama | Değiştirilen herhangi bir özelliği tek tıkla fabrika varsayılan değerine geri döndürebilirsiniz. |
| Değişiklik Takibi | Kaydedilmemiş değişiklikler ekran üzerinde görsel olarak işaretlenir. |

<Tip>
Burada yapılan değişiklikler, yeni oluşturulan API Proxy yapılandırmalarında varsayılan olarak uygulanır. Mevcut API Proxy'lerin zaten özelleştirilmiş ayarları bu değişiklikten etkilenmez.
</Tip>

<Warning>
Değişiklikler kaydedildikten sonra ilgili ortamlara otomatik olarak dağıtılır. Üretim ortamlarında değişiklik yapmadan önce değerlerin etkisini değerlendirmeniz önerilir.
</Warning>
