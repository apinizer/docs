# Sunucu Yönetimi

Sunucu Yönetimi, Apinizer'ın kubernetesde çalışan altyapısını oluşturmayı, yönetmeyi, altyapı dağıtımını gerçekleştirmeyi sağlar. Bunu yaparken kubernetes bilmenize gerek yoktur. Sadece ihtiyaç duyulan bilgilerin ekran üzerinden girilmesi ile Apinizer Kubernetes API'leri ile iletişime geçerek tüm işlemlerin otomatik olarak yapılmasını sağlar.

Bu bölümde ayrıca Elasticsearch sunucularının tanımlanması, API Integration ortamının yapılandırılması ve yönetimleri de yapılır.

Sunucu Yönetimi menüsüne **Admin** rolüne sahip kullanıcılar erişebilir.

Sunucu Yönetimi aşağıdaki tanımlar yapılır:

- [Ortam (Environment)](/tr/admin/gateway-ortamlari/environment-creation)
- [Kubernetes Resources](/tr/admin/gateway-ortamlari/kubernetes-resources)
