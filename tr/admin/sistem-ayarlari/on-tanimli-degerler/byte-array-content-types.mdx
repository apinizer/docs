# Byte Array Content Types

Normalde gelen (request) ve dönen (response) mesajların gövdeleri (body) text olarak değerlendirilir. Bu sayfa Content-Type header değerine göre mesajların Byte Array olarak kullanılmasını sağlayacak değer kümesini oluşturmak için kullanılır.

Eğer Content-Type değeri bu liste içinde ise içerik Byte Array değeri olarak ele alınır.

## Yeni Bir Byte Array Content Type Oluşturma

**Yönetim → Sistem Ayarları → Ön Tanımlı Değerler → Byte Array Content Types** ekranından **+Yeni** butonuna tıklayınız.

**Byte Array Content Type oluşturma ayarlarını içeren görsele aşağıda yer verilmiştir:**

![Byte Array Content Type Oluşturma](https://docs.apinizer.com/apinizer-dokumantasyonu/files/39725348/39725352/1/1653565498708/image2022-5-26_14-44-59.png)

Byte Array Content Type oluşturma konfigürasyonu için kullanılan alanlar aşağıdaki tabloda görülmektedir.

| Alan | Açıklama |
| --- | --- |
| Değer *(Value)* | Byte Array Content Type Değerinin girildiği alandır. |
