# İstek Başlık Adları

Bu bölümde, global olarak istek başlıkları tanımlayarak, API Geçitlerinde, Test Konsolda hazır ve seçilebilir olması için tanımlamalar yapılır.

İstek başlığı, sunucunun yanıtı uyarlayabilmesi için istek bağlamı hakkında bilgi sağlamak için HTTP isteğinde kullanılabilen bir HTTP başlığıdır (Http Header). Örneğin, Accept- * üstbilgileri yanıtın izin verilen ve tercih edilen biçimlerini gösterir. Diğer başlıklar, kimlik doğrulama bilgileri sağlamak için kullanılabilir. (Yetkilendirme, önbelleğe almayı kontrol etmek veya kullanıcı aracısı veya yönlendiren hakkında bilgi almak için vb.)

Apinizer Platformu'nda varsayılan olarak 33 adet İstek Başlık Adı bulunmaktadır:

- Accept
- Content-MD5
- Content-Type
- Cookie
- Date
- Expect
- Forwarded
- From
- Host
- If-Match
- If-Modified-Since
- Accept-Charset
- If-None-Match
- If-Range
- If-Unmodified-Since
- Max-Forwards
- Origin
- Pragma
- Proxy-Authorization
- Range
- Referer
- TE
- Accept-Datetime
- Upgrade
- User-Agent
- Via
- Warning
- Accept-Encoding
- Accept-Language
- Authorization
- Cache-Control
- Connection
- Content-Length

Apinizer tarafından varsayılan olarak eklenenleri düzenleyemez ve silemezsiniz.

Yukarıdakilere ek olarak kendi **İstek Başlık** larınızı ekleyebilirsiniz.

## Yeni Bir İstek Başlık Adı Oluşturma

**Yönetim → Sistem Ayarları → Ön Tanımlı Değerler → İstek Başlık Adları** ekranından **+Yeni** butonuna tıklayınız.

**İstek Başlık Adı oluşturma ayarlarını içeren görsele aşağıda yer verilmiştir:**
