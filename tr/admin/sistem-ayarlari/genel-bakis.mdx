# Sistem Ayarları

Sistem Ayarları bölümünde, Apinizer Platformuna ait sistemsel ayarlar yapılır.

Sistem Ayarları menüsüne yalnızca **Yönetici Rolü** sahip kullanıcılar tarafından erişilir.

Sistem Ayarları menüsünde yönetilen ekranlar aşağıdaki gibidir:

- [Genel Ayarlar (General Settings)](/tr/admin/sistem-ayarlari/global-configuration)
- [Ön Tanımlı Değerler](/tr/admin/sistem-ayarlari/predefined-values)
  - [CORS Başlıkları](/tr/admin/sistem-ayarlari/predefined-values/cors-headers)
  - [CORS Orijin Değerleri](/tr/admin/sistem-ayarlari/predefined-values/cors-origin-values)
  - [Filtre Kuralları](/tr/admin/sistem-ayarlari/predefined-values/filter-rules)
  - [İstek Başlık Adları](/tr/admin/sistem-ayarlari/predefined-values/http-request-header-names)
  - [İstek Başlık Değerleri](/tr/admin/sistem-ayarlari/predefined-values/http-request-header-values)
  - [Byte Array Content Types](/tr/admin/sistem-ayarlari/predefined-values/byte-array-content-types)
- [LDAP ile Manager Giriş Ayarları](/tr/admin/kullanici-erisim-yonetimi/ldap-active-directory-integration)
- [Lisans Yönetimi](/tr/admin/sistem-ayarlari/license-management)
