# Ön Tanımlı Değerler

Ön Tanımlı Değerler, Apinizer genelinde kullanıma hazır bulunan değerlerdir. Sistem ilk yüklendiğinden varsayılan (default) olarak hazır değerler yüklenmektedir.

İstediğiniz zaman yeni değerler ekleyebilirsiniz. Bu girilen değerler **Filtre Kuralları**, **CORS Başlıkları**, **İstek Başlıkları** sayfalarıdanki alanlarda **auto complete** olarak kullanılır.

Apinizer tarafından varsayılan (default) olarak gelen değerleri değiştiremez ve silemezsiniz.

Apinizer üzerinde tanımlayabileceğiniz Ön Tanımlı Değerler aşağıdaki gibidir:

- [CORS Başlıkları](/tr/admin/sistem-ayarlari/predefined-values/cors-headers)
- [CORS Orijin Değerleri](/tr/admin/sistem-ayarlari/predefined-values/cors-origin-values)
- [Filtre Kuralları](/tr/admin/sistem-ayarlari/predefined-values/filter-rules)
- [İstek Başlık Adları](/tr/admin/sistem-ayarlari/predefined-values/http-request-header-names)
- [İstek Başlık Değerleri](/tr/admin/sistem-ayarlari/predefined-values/http-request-header-values)
- [Değişkenler](/tr/admin/sistem-ayarlari/predefined-values/common-variables)
- [Byte Array Content Types](/tr/admin/sistem-ayarlari/predefined-values/byte-array-content-types)
