# CORS Başlıkları

Bu ekranda, CORS ayarlarının yapıldığı ekranlarda ortak olarak kullanılan başlık tanımları yapılır.

Apinizer konsolunda kullanım kolaylığı sağlamak amacıyla varsayılan olarak bazı başlıklar tanımlamaktadır:

- Accept
- Accept-Language
- Content-Language
- Content-Type

## Yeni Bir CORS Başlık Değeri Oluşturma

**Yönetim → Sistem Ayarları → Ön Tanımlı Değerler → CORS Başlıkları** ekranından **+Yeni** butonuna tıklayınız.

**CORS Başlık Değeri oluşturma ayarlarını içeren görsele aşağıda yer verilmiştir:**

![CORS Başlık Değeri Oluşturma](/images/cors-header-creation.png)

## CORS Başlık Değeri Oluşturma Konfigürasyonu

CORS Başlık Değeri oluşturma konfigürasyonu için kullanılan alanlar aşağıdaki tabloda görülmektedir:

| Alan | Açıklama |
|------|----------|
| Değer (Value) | CORS Başlık Değerinin girildiği alandır. |
