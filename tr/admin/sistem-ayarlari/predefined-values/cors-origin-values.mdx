# CORS Orijin Değerleri

Apinizer Platformu'nda CORS Orijin Değeri kullanılarak izin verilen domainler eklenebilir. Varsayılan CORS Orijin Değeri "*" olarak mevcuttur ve tüm orijinlere izin verir.

## Yeni Bir CORS Orijin Değeri Oluşturma

**Yönetim → Sistem Ayarları → Ön Tanımlı Değerler → CORS Orijin Değerleri** ekranından **+Yeni** butonuna tıklayınız.

**CORS Orijin Değerleri oluşturma ayarlarını içeren görsele aşağıda yer verilmiştir:**

![CORS Orijin Değerleri Oluşturma](/images/cors-origin-values-creation.png)

## Cors Orijin Değerleri Oluşturma Konfigürasyonu

Cors Orijin Değerleri oluşturma konfigürasyonu için kullanılan alanlar aşağıdaki tabloda görülmektedir:

| Alan | Açıklama |
|------|----------|
| Origin | CORS Orijin değerinin girildiği alandır. Örnek: **http://domain-a.com**. |
