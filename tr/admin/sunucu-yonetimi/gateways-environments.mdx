# Ağ Geçidi Ortamları (Gateways Environments)

Ortam (Environment), bir kuruluştaki API Proxy'leri (API Proxy) için bir çalışma zamanı yürütme bağlamıdır (Runtime Execution Context).

Bu bölümde, ortam oluşturma ve ilgili işlemler anlatılmaktadır.

Ortamlar, API Proxy'lerin farklı ortamlarda (örneğin, geliştirme, test, üretim) çalıştırılmasını ve yönetilmesini sağlar. Ortam oluşturma ve yönetimi hakkında detaylı bilgi için [Manuel Gateway Yönetimi](/tr/admin/sunucu-yonetimi/manual-gateway-management) sayfasına bakabilirsiniz.
