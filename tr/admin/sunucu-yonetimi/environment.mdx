# Ortam (Environment)

Ortam (Environment), bir kuruluştaki API Proxy'leri (API Proxy) için bir çalışma zamanı yürütme bağlamıdır (Runtime Execution Context).

Bir ortam, API Proxy'lerin çalıştırılacağı fiziksel veya mantıksal bir ortamı temsil eder. Her ortam, kendi konfigürasyon ayarlarına, erişim URL'lerine ve kaynak yönetim ayarlarına sahiptir.

Ortamlar, Apinizer platformunda API Proxy'lerin farklı ortamlarda (örneğin, geliştirme, test, üretim) çalıştırılmasını ve yönetilmesini sağlar.

Ortam oluşturma ve yönetimi hakkında detaylı bilgi için [Manuel Gateway Yönetimi](/tr/admin/sunucu-yonetimi/manual-gateway-management) sayfasına bakabilirsiniz.
