<Warning>
Kimlik bilgisi erişim izni, API Proxy Grup'a erişim için tek başına yeterli değildir.

Kimlik bilgisindeki ve erişim iznindeki ayarların geçerli olması için API Proxy Grup üzerinde kimlik doğrulama politikalarından birisi eklenmiş olmalı ve bu politikada kimlik doğrulama şekli olarak **Security Manager** seçeneği seçili olmalıdır.
</Warning>

Bu sayfada yetkili uygulama kullanıcısının içinde yer aldığı projeye ait olan tüm API Proxy Grup'lar listelenir, isteğe bağlı olarak filtrelenebilir.

## API Proxy Grup Detay Ekranı

Bir API Proxy Grup üzerinde işlem yapmak için listeden ilgili grup seçilerek detay ekranına gidilir.

Bu ekranda API Proxy Grup için daha önceden erişim izni verilmiş olan **Kimlik Bilgilerinin listesi** yer alır.

Yeni bir Kimlik Bilgisini eklemek için **Ekle** tuşuna basılır.

  src="/images/yonetici/apg-ekle-buton.png"

  width="1000"
  height="600"

## Kimlik Bilgisi Ekleme

**Ekle (Add)** tuşuna basıldığında proje içerisinde tanımlı veya yönetim menüsünden tanımlanmış olan kimlik bilgileri listelenir.

<img
  src="/images/yonetici/apg-ekle.png"
  alt="Kimlik Bilgisi Ekleme"

  height="600"
/>

## Konfigürasyon Alanları

| Alan | Açıklama |
|------|----------|
| Geçerliliğini Yitirme Zamanı (Expires On) | Bu tarih değerinin girilmesi durumunda Kimlik Bilgisi bu tarih geldiği zamandan itibaren (gece 00.00'dan itibaren geçerli olmak üzere) API Proxy'e erişemez hale gelir. Boş kalması durumunda pasif hale gelmediği sürece API Proxy Grup'a erişmeye devam eder. Bu tarihin gelmesi sadece bu API Proxy Grup'a erişimi kısıtlar, diğer API ProxyGrup'lara erişimde bir değişiklik yapmaz. |
| Ortam Listesi (Environment List) | API Proxy Grup'un yüklendiği ortama özel olarak Kota ve Daraltma değerlerinin girilebilmesini sağlar. |
| Ortam Kotası (Quota) | API Proxy Grup'un belirtilen ortama özel kota değeridir. |
| Ortam Daraltması (Throttling) | API Proxy Grup'un belirtilen ortama özel daraltma değeridir. |
| Mesaj Sayısı (Message Count) | Daraltma Aralığı ile verilen süre içinde Backend API'ye gönderilebilecek olan maksimum mesaj sayısıdır. |
| Daraltma Zaman Miktarı (Interval Time Amount) | Seçilen zaman birimi cinsinden, sınırlama penceresinin süresini belirten sayısal değer. |
| Daraltma Zaman Birimi (Interval Time Unit) | API istek sınırlaması için kullanılan zaman aralığı birimi (örneğin, saniye, dakika). |
| Periyot Tipi (Interval Window Type) | API istek sınırlaması için kullanılan zaman aralığı yöntemi (sabit veya kayan). |
| Cache Bağlantısı Zaman Aşım Süresi (Cache Connection Timeout (Second)) | Cache bağlantısı için zaman aşımı süresi belirtilir. |
| Cache Bağlantı Hatası Eylemi (Action for Cache Connection Error) | Eğer politika cache sunucusuna bağlantı sorunu yaşarsa uygulanacak eylem belirtilir. |
| İzin Verilmeyen API Proxy'ler (Disallowed API Proxies) | Kimlik bilgisinin sahip olduğu rollerinden bağımsız olarak API Proxy Grup'un herhangi bir API Proxy'sine erişmemesi isteniyorsa, burada API Proxy Grup'un erişime kapatılmak istenen API Proxy'lere seçilir. Ön tanımlı olarak kimlik bilgisi ile API Proxy Grup'nin tüm API Proxy'lerine erişim sağlanabilir. |
| Kaydet ve Yükle Tuşu (Save and Deploy) | Ayarların/değişikliklerin tamamlanması sonrasında **Kaydet ve Yükle (Save and Deploy)** tuşuna basılarak ayarların aktifleşmesi sağlanır. |
