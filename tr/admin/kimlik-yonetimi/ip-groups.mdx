IP Grubu, bir grup IP adresini bir arada yönetmek için tanımlanır. Tanımlı IP grupları, İzin Verilen IP Listesi ile Yasaklı IP Listesi politikalarında kullanılmaktadır.

**IP Grubu ayarlarını içeren görsele aşağıda yer verilmiştir:**

  className="mr-auto"
/>


IP Grubu konfigürasyonu için kullanılan alanlar aşağıdaki tabloda görülmektedir.

| Alan | Açıklama |
|------|----------|
| Ad<br/>*(Name)* | Oluşturulan IP Grubunun ad bilgisidir. |
| Açıklama<br/>*(Description)* | IP Grubuna ilişkin açıklama/tanım bilgisidir. |
| IP Adresi<br/>*(IP Address)* | Gruba eklenmek istenen IP Adresleri girilir. IP aralığı belirtmek için **'*'** ve **'-'** karakterleri kullanılabilir.<br/><br/>Bir seferde birçok IP eklemek için tüm IP'ler virgül ile ayrılarak yazılabilir.<br/><br/>IP'ler otomatik olarak ayrıştırılarak eklenecektir.<br/><br/>- **'10.3.10.*'** değeri **'10.3.10.0' ile '10.3.10.255'** arasındaki IP'leri belirtir.<br/>- **'10.3.10.4-18'** değeri **'10.3.10.4' ile '10.3.10.18'** arasındaki IP'leri ifade eder (4 ve 18 dahil). |

<Info>
Tüm IP grupları "Globalleştir" seçeneği ile Admin sayfasına taşınarak görünümü bu listeden kalkar, tüm projelerde kullanılabilir hale gelir ve yönetimi Admin kullanıcısına bırakılır.
</Info>
