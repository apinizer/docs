ACL Rapor ekranında **API Proxy bazlı** ya da **Kurum/Kimlik Bilgisi bazlı** olarak iki seçenekte sorgu çalıştırılabilir.

API Proxy bazlı sorguda, hangi API Proxy'ye hangi kullanıcıların erişebildiğinin bilgisi listelenir.

Kurum/Kimlik bazlı sorguda ise hangi kurumun hangi API Proxy'lere erişebildiğinin bilgisi listelenir.

Filtre alanlarıyla sorgular özelleştirilebilir.

<Tabs>
  <Tab title="API Proxy Bazlı Sorgu">
    API Proxy bazlı sorgu ekranında, seçilen API Proxy'ye hangi kullanıcıların erişim yetkisi olduğu görüntülenir.
    
    ![API Proxy bazlı ACL raporu](/images/acl-report-api-proxy.png)
  </Tab>
  
  <Tab title="Kurum/Kimlik Bazlı Sorgu">
    Kurum/Kimlik bazlı sorgu ekranında, seçilen kurumun hangi API Proxy'lere erişim yetkisi olduğu görüntülenir.
    
    ![Kurum/Kimlik bazlı ACL raporu](/images/acl-report-organization.png)
  </Tab>
</Tabs>

<Note>
Filtre alanlarını kullanarak daha spesifik sorgular oluşturabilir ve raporları özelleştirebilirsiniz.
</Note>
