## Yeni Bir Key Store Oluşturma

<Steps>
  <Step title="Key Store Tanımı Oluştur">
    **Yönetim → Gizlilik Yönetimi → Key Stores** ekranından **Yeni (Create)** butonuna tıklayınız. **Tanım (Definition)** altındaki bilgileri doldurunuz. Ardından **Yeni Key Store Tanımı (New Key Store Definition)** butonuna tıklayınız.

<img
  src="/images/yonetici/key-store1.png"
  alt="Key Store Tanımı Oluşturma"
  width="1000"
  height="600"
/>
    Key Store oluşturma konfigürasyonu için kullanılan alanlar:

    | Alan | Açıklama |
    |------|----------|
    | Ad  | Oluşturulan Key store'un ad bilgisidir. |
    | Açıklama  | Oluşturulan Key Store ile ilgili yönetimi kolaylaştırmak için açıklama yazılabilir. |
  </Step>

  <Step title="Key Store Konfigürasyonu">
    Açılan pencerede Key Store için konfigürasyon bilgilerini doldurunuz. Ardından **Kaydet** butonuna tıklayınız. Kaydedilen **Key Store Tanımı**, **Konfigürasyon** altında açılan tabloda listelenir.

<img
  src="/images/yonetici/key-store2.png"
  alt="Key Store Konfigürasyonu"
  width="1000"
  height="600"
/>
    Key Store tanımlama konfigürasyonu için kullanılan alanlar:

    | Alan | Açıklama |
    |------|----------|
    | Ortamlar (Environments) | Key Store'un kullanılacağı önceden tanımlı ortam seçilir. |
    | Key Store Tipi (Key Store Type) | Key Store'un tipi seçimidir. İki seçenek bulunur:<br/>• JKS<br/>• PKCS12 |
    | Dosya (File) | "Dosya Seç" butonuna tıklanarak Key Store'u içeren dosya seçilir.<br/>• İzin verilen formatlar: .jks, .pkcs12 |
    | Alias | Key Store'un alias bilgisi girilir. |
    | Parola (Password) | Key Store için parola girilen alandır. |
  </Step>

  <Step title="Kaydet ve Yükle">
    Tanımlama ve konfigürasyon bilgilerini tamamladıktan sonra **Kaydet ve Yükle (Save and Deploy)** butonuna tıklayarak yükleyiniz.

<img
  src="/images/yonetici/key-store3.png"
  alt="Key Store Kaydet ve Yükle"
  width="1000"
  height="600"
/>  </Step>
</Steps>

## Key Store'u Görüntüleme (View)

Key Store'ları görüntülemek için iki yöntem bulunmaktadır:

<CardGroup cols={2}>
  <Card title="Yöntem 1: Liste Ekranından" icon="list">
    - Key Store listesinde görüntülemek istediğiniz Key Store'un **Ad** alanına tıklayın.
    - View ekranına yönlendirilirsiniz.
  </Card>
  
  <Card title="Yöntem 2: Menüden" icon="ellipsis-vertical">
    - Key Store listesinde satır sonundaki **⋮** menüsünden **View** seçeneğini tıklayın.
    - View ekranına yönlendirilirsiniz.
  </Card>
</CardGroup>

### View Ekranı Görseli

<img
    src="/images/yonetici/key-store-view.png"

### View Ekranından Yapılabilecek İşlemler

View ekranının sağ üst köşesinde bulunan butonlarla aşağıdaki işlemleri gerçekleştirebilirsiniz:

<CardGroup cols={2}>
  <Card title="Edit (Düzenle)" icon="pencil">
    **[Edit]** butonuna tıklayarak Key Store'u düzenleme ekranına yönlendirilirsiniz. Tüm parametreleri güncelleyebilirsiniz.
  </Card>
  
  <Card title="Back to List (Listeye Dön)" icon="list">
    **[Back to List]** butonuna tıklayarak Key Store listesine geri dönebilirsiniz.
  </Card>
</CardGroup>

### View Ekranında Görüntülenen Bilgiler

View ekranında aşağıdaki bilgiler görüntülenir:

<CardGroup cols={2}>
  <Card title="Tanım Bilgileri" icon="file-lines">
    - **Ad:** Key Store'un adı
    - **Açıklama:** Key Store hakkında açıklama
  </Card>
  
  <Card title="Ortam Bilgileri" icon="server">
    - Key Store'un tanımlı olduğu ortamlar, keystore tipi ve alias bilgileri tablo halinde görüntülenir.
    - Her ortam için ortam adı, keystore tipi (JKS/PKCS12) ve alias bilgisi listelenir.
  </Card>
</CardGroup>

<Info>
View ekranından Key Store bilgilerini görüntüleyebilir ve **[Edit]** butonu ile düzenleme ekranına geçebilirsiniz.
</Info>

## Key Store Düzenleme

Key Store'u düzenlemek için:

<CardGroup cols={1}>
  <Card title="View Ekranından Düzenleme" icon="eye">
    - Key Store'un view ekranına gidin.
    - Sağ üstteki **[Edit]** butonuna tıklayın.
    - Düzenleme ekranına yönlendirilirsiniz.
  </Card>
</CardGroup>

<Steps>
  <Step title="Düzenleme İşlemini Başlatma">
    Varolan bir Key Store'u düzenlemek için view ekranındaki **[Edit]** butonuna tıklayınız.

<img
  src="/images/yonetici/key-store-view2.png"
  alt="Key Store Düzenleme İşlemi"
  width="1000"
  height="600"
/>  </Step>

  <Step title="Konfigürasyon Düzenleme">
    Key Store'a ait konfigürasyon bilgilerini düzenlemek için konfigürasyon listesi altındaki menüden **Düzenle (Edit)** butonuna tıklayınız.

<img
  src="/images/yonetici/key-store5.png"
  alt="Key Store Konfigürasyon Düzenleme"
  width="1000"
  height="600"
/>
    <Note>Key Store'u düzenlemek için "Düzenle" butonuna tıklandıktan sonra oluştururken izlenen adımlar izlenir.</Note>
  </Step>
</Steps>

## Key Store Silme

Key Store'u silmek için:

<CardGroup cols={2}>
  <Card title="Liste Ekranından" icon="list">
    - Key Store listesinde satır sonundaki **⋮** menüsünden **Sil (Delete)** seçeneğini tıklayın.
    - Onay dialogunda silme işlemini onaylayın.
  </Card>
  
  <Card title="Silme İpuçları" icon="info-circle">
    **Silmeden Önce Kontrol Edin:**
    - Key Store politikalar veya bağlantılarda kullanılıyor olabilir.
    - Gerekirse alternatif bir Key Store atayın.
  </Card>
</CardGroup>

<Steps>
  <Step title="Silme İşlemini Başlat">
    Varolan bir Key Store'u silmek için o Key Store'un menüsünden **Sil (Delete)** butonuna tıklayınız.

<img
  src="/images/yonetici/key-store6.png"
  alt="Key Store Silme İşlemi"
  width="1000"
  height="600"
/>  </Step>

  <Step title="Silme İşlemini Onayla">
    Açılan pencerede tekrar **Sil (Delete)** butonuna tıklayarak onaylayınız.

<img
  src="/images/yonetici/key-store7.png"
  alt="Key Store Silme Onayı"
  width="1000"
  height="600"
/>  </Step>
</Steps>
