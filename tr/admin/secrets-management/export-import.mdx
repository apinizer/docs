<Info>
Dışa/İçe Aktarma Sihirbazı (Import/Export Wizard) ile proje içindeki API Proxy'ler dışa/içe aktarılabilir, global politikalar veya proje tüm varlıklarıyla dışarı aktarılabilir. Dışa/İçe Aktarmanın başlıca amacı konfigürasyonu başka bir projeye ya da başka bir ortamdaki Apinizer'a entegre edilmesini sağlamaktır.
</Info>

## Kullanım Senaryoları

### Ortamlar Arası Transfer

Erişim kısıtlamasından dolayı fiziksel olarak, production ve test adlı iki farklı ortamda kurulu iki tane Apinizer uygulaması olduğunu varsayalım. Production ortamı istemcilere, test ortamı ise yazılım geliştiricilere hizmet etmektedir. Test ortamındaki API Proxy ve uygulanan politikalar ve konfigürasyonlar, production ortamına da uygulanması istenmektedir. Bu durumda test ortamındaki API Proxy dışa aktarılır. Sonrasında oluşturulan konfigürasyon dosyaları production ortamına uygulanır.

### Konfigürasyon İzleme

Konfigürasyon değişiklikleri izlenebilir. Kurum, güvenlik kuralları nedeniyle politika ayarlarındaki farklılıkları izlemek isteyebilir. Bunun için CI sürecine, dışa aktarılan politika, API Proxy ya da projeye ait konfigürasyon dosyası eklenmelidir.
