Bu sayfada LDAP/Active Directory bağlantı ayarlarının nasıl yapıldığı anlatılmaktadır.

**LDAP Bağlantı ayarlarını içeren görsele aşağıda yer verilmiştir:**

  src="/images/yonetici/ldap.png"

/>
LDAP bağlantısı konfigürasyonu için kullanılan alanlar aşağıdaki tabloda görülmektedir.

| Alan | Açıklama |
|------|----------|
| Ad<br/>*(Name)* | Oluşturulan bağlantının ad bilgisidir. |
| Açıklama<br/>*(Description)* | Oluşturulan konfigürasyon ile ilgili yönetimi kolaylaştırmak için açıklama yazılabilir. |
| Sunucu Adresi<br/>*(Server Address)* | LDAP Sunucusunun bağlantı adresidir. Örnek: ldap://localhost:389 |
| Sertifika Durumu<br/>*(Certificate Status)* | LDAP bağlantısı için sertifika gerekliliğini, gerekli ise kullanıcı bilgilerinin sertifikanın neresinden alınacağı belirtilir.<br/>• Not Required<br/>• Trust All Certificates<br/>• Certificate Required |
| Sertifika<br/>*(Certificate)* | Sertifika Durumu "Sertifika Gerekli" olarak ayarlanmışsa, LDAP bağlantısı için gerekli sertifika seçilir. Sertifika için ilgili belgelere bakın..<br/>Sertifika için [bakınız.](/tr/admin/gizlilik-yonetimi/sertifikalar)
|
| Kullanıcı Adı<br/>*(Username)* | Oluşturulan istemci kullanıcının kullanıcı adı bilgisidir. Kullanıcı adı için DN (Distinguished Name) girilmesi gereklidir. Environment Variable kullanılabilir. İsteğe bağlı alandır.<br/>Kullanıcı adı için DN girilmesi gereklidir. |
| Şifre<br/>*(Password)* | Oluşturulan istemci kullanıcının şifre bilgisidir. Environment Variable kullanılabilir ve mutlaka secret olarak işaretlenmelidir. İsteğe bağlı alandır. |
| Custom Filter<br/>*(Custom Filter)* | Bu alan ile aramaya ek kısıtlama/filtre getirebilir. |
| Search Scope<br/>*(Search Scope)* | Arama işleminin base dn üzerinde hangi seviyede yapılacağını belirtir. |
| Base DN<br/>*(Base DN)* | LDAP aramalarının başlatılacağı kök girilir. |

<Info>
Environment Variable'ların nasıl oluşturulduğu ve kullanılabileceği hakkında detaylı bilgi için [Ortam Değişkenleri (Environment Variables)](/ortam-degiskenleri-environment-variables) bölümüne bakabilirsiniz.
</Info>
