## Konfigürasyon

    src="/images/yonetici/konnektor/api-call.png"

API Çağrısı yapılandırması konfigürasyonu için kullanılan alanlar:


  <Accordion title="Temel Ayarlar">
    <CardGroup cols={2}>
      <Card title="Metot" icon="list">
        **Alan:** Metot (Method)
        
        API çağrısı yapılacak endpoint'in HTTP metodu seçilir. Zorunludur.
      </Card>
      
      <Card title="URL" icon="link">
        **Alan:** URL
        
        API çağrısı yapılacak endpoint bilgisi girilir. Zorunludur.
      </Card>
      
      <Card title="Zaman Aşımı" icon="clock">
        **Alan:** Zaman Aşımı (Timeout)
        
        İsteğin ne kadar süre sonra zaman aşımına uğrayacağı bilgisidir. Zorunludur.
      </Card>
    </CardGroup>
  </Accordion>
  
  <Accordion title="Güvenlik Ayarları">
    <CardGroup cols={2}>
      <Card title="Sertifikayı Aktive Et" icon="shield">
        **Alan:** Sertifikayı Aktive Et (Enable Certificate)
        
        İstekle birlikte sertifikanın gidip gitmeyeceği yönetilir.
      </Card>
      
      <Card title="Sertifika" icon="certificate">
        **Alan:** Sertifika (Certificate)
        
        Sertifika aktif edildiyse bu alan zorunludur. Eğer sertifika listesinde istenilen değer yoksa (Sertifika Ekle) Add Certificate linkinden yeni bir tane oluşturulabilir.
      </Card>
    </CardGroup>
  </Accordion>
  
  <Accordion title="İstek İçeriği">
    <CardGroup cols={2}>
      <Card title="Başlıklar" icon="heading">
        **Alan:** Başlıklar (Headers)
        
        İsteğin başlık bilgileri girilir.
        
        :::info

Başlıklar alanında değişkenler kullanılabilir.

:::
      </Card>
      
      <Card title="Gövde" icon="file-lines">
        **Alan:** Gövde (Body)
        
        İsteğin gövde değeri girilir.
        
        :::info

Gövde alanında değişkenler kullanılabilir.

:::
      </Card>
    </CardGroup>
  </Accordion>
