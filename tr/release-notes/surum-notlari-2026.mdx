- API Ops ile JWK oluşturma özelliği eklendi.
- General Settings'ten log details gösterimini kapatma özelliği eklendi.
- Keys ve Keystores için API Ops desteği eklendi.
- API Proxy Traffic Log Settings işlemleri için audit log desteği eklendi.
- API Management API'leriyle yapılan işlemler için audit log desteği eklendi.
- FTP Read ve List işlemlerine ends_with ve wildcard arama seçenekleri eklendi.
- Client Route Report özelliği eklendi.
- Portal Overview sayfası eklendi.

- API Proxy politika ekleme ekranında Security and Authentication ve Message Validation & Crypto bölümlerinin alt açıklamaları güncellendi.
- WS Security politikalarında politika ekleme ekranındaki isimlendirmeler Türkçe dile uygun hale getirildi.
- General Settings'te API Root Context değiştirme kısmına restart uyarısı eklendi.
- API Traffic'te Absolute tarih alanında "Please Wait" yazısı tarih alanının önünde görünmemesi sağlandı.
- Proxy Group'ta disabled edilmiş metodları belirtme özelliği eklendi.
- Test konsol ve data operations gibi alanlardaki büyük text alanlarında yazılar satır başından başlatıldı.
- Host alias girdikten sonra save&republish işlemi için onay popup'ı eklendi.
- Variables sayfasında parameter cinsi listede gösterilmeye başlandı.
- Kubernetes Resources sayfası probe ekranında görsel düzeltmeler yapıldı.
- Integration Overview sayfasına trafik yenileme butonu eklendi.
- API Portal Kullanım İstatistikleri aylık-günlük raporlama özelliği eklendi.
- Portal Accounts/Developers sayfasından account'lar Excel olarak indirilebilmeye başlandı.
- Management API ile export/import işlemlerinde DB2API tanımlarına "-imported" kelimesi eklenmemesi sağlandı.
- Portal için Landing Page özelliği eklendi.
- XML Schema Validation politikasında oluşan hatalarda ilgili element ve path bilgisi hata mesajına eklendi.
- es client versiyonu 7.9.2'den 7.17.29 a çıkarıldı, es7 ve es8 tamamen desteklenir hale getirildi.
