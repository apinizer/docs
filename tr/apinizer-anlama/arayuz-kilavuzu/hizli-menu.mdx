Hızlı Menü (Quick Menu), Apinizer platformunun sağ üst köşesinde yer alan ve kullanıcıların hızlı erişim ihtiyaçlarını karşılayan menü bileşenidir.

## Hızlı Menü Genel Bakış

Bir kullanıcı platforma giriş yaptığı zaman, arayüzün sağ üst kısmında Hızlı Menü görüntülenir. Bu menü, kullanıcının en sık kullandığı işlevlere hızlı erişim sağlar.

<CardGroup cols={2}>
  <Card title="Proje Seçimi" icon="folder">
    Aktif projeyi görüntüleme ve değiştirme
  </Card>
  <Card title="Test Araçları" icon="flask">
    API test araçlarına hızlı erişim
  </Card>
  <Card title="Yönetim" icon="gear">
    Yönetim işlemlerine erişim
  </Card>
  <Card title="Bildirimler" icon="bell">
    Bildirimleri görüntüleme ve yönetme
  </Card>
  <Card title="Yardım" icon="question-circle">
    Dokümantasyon ve destek linkleri
  </Card>
  <Card title="Kişisel Ayarlar" icon="user">
    Profil ve ayar yönetimi
  </Card>
</CardGroup>

## Proje Seçimi

Hızlı Menünün en solunda seçilmiş olan projenin adı görünür.

### Proje Değiştirme

<AccordionGroup>
  <Accordion title="Proje Listesi">
    * Proje adına veya yanındaki ok simgesine tıklanarak proje listesi açılır
    * Tüm erişim yetkisi olan projeler listelenir
    * Proje adına tıklanarak proje seçilir
  </Accordion>
  
  <Accordion title="Proje Seçimi Sonrası">
    * Seçilen proje aktif hale gelir
    * Ana Menü Geliştirici Menüsüne dönüşür
    * Çalışma alanı proje içeriğini gösterir
  </Accordion>
</AccordionGroup>

<Info>
  Proje seçimi, kullanıcının hangi proje bağlamında çalışacağını belirler. Proje seçilmeden bazı işlemler yapılamaz.
</Info>

## Test Araçları

Proje adının sağındaki simge Test Araçları menüsünü açar. Kullanıcılar çalışmalarının herhangi bir anında bu menüyü kullanarak:

<CardGroup cols={2}>
  <Card title="API Test" icon="code">
    * Bir API'ye hızlıca istek gönderme
    * Request/Response görüntüleme
    * Authentication test etme
  </Card>
  <Card title="XPath/JSONPath Test" icon="code">
    * XPath ifadelerinin doğruluğunu sınama
    * JSONPath ifadelerinin test edilmesi
    * Expression sonuçlarını görüntüleme
  </Card>
  <Card title="Dönüşüm Test" icon="wand-magic-sparkles">
    * Dönüşüm kodlarının çıktısını kontrol etme
    * Transformation test
    * Script test
  </Card>
</CardGroup>

## Yönetim (Administration)

Yönetim işlemlerinin bir alt kümesini içeren bir menü açar. Bu öğelerden herhangi biri seçilirse, ana menü alanında Yönetim Menüsü açılır.

<Warning>
  **Önemli**: Giriş yapan kullanıcı Yönetici (Admin) yetkisinde değilse Yönetim (Administration) öğesi görünmez.
</Warning>

### Yönetim Menü Öğeleri

<AccordionGroup>
  <Accordion title="Genel Bakış (Overview)">
    * Platform genel bakış
    * Sistem durumu
    * İstatistikler
  </Accordion>
  
  <Accordion title="Kullanıcı Yönetimi">
    * Kullanıcı oluşturma ve yönetimi
    * Rol yönetimi
    * ACL yönetimi
  </Accordion>
  
  <Accordion title="Sistem Ayarları">
    * Sistem konfigürasyonu
    * Ortam yönetimi
    * Gateway yönetimi
  </Accordion>
</AccordionGroup>

## Bildirimler (Notifications)

Apinizer üzerinde 3 farklı durum sonucunda bildirim oluşturulmaktadır:

<CardGroup cols={3}>
  <Card title="Eylem Sonuçları" icon="check-circle">
    * Çalıştırılmış eylem sonucunda üretilen bildirimler
    * Deployment sonuçları
    * İşlem durumları
  </Card>
  <Card title="Hesap/Geliştirici İstekleri" icon="user-plus">
    * Yeni kullanıcı oluşturma istekleri
    * Geliştirici hesap istekleri
    * Onay gerektiren işlemler
  </Card>
  <Card title="API Ürünü Abonelik İstekleri" icon="bell">
    * API ürününe abonelik istekleri
    * Onay bekleyen abonelikler
  </Card>
</CardGroup>

### Bildirim Yönetimi

Tüm bildirimler, uygulamanın Hızlı Menü kısmındaki ikondan görüntülenebilir veya ilgili bildirim kategorisine tıklanarak bildirim yönetimi yapılabilir.

<AccordionGroup>
  <Accordion title="Bildirim Görüntüleme">
    * Bildirim ikonuna tıklanarak bildirim listesi açılır
    * Okunmamış bildirim sayısı ikon üzerinde gösterilir
    * Bildirim detayları görüntülenir
  </Accordion>
  
  <Accordion title="Bildirim Yönetimi">
    * Bildirimler sayfasına gitmek için "Bildirimler" veya "Okunmamış Bildirimler" linkine tıklanır
    * Sadece çalıştırılmış eylem sonucunda üretilenler, Bildirimler sayfasından yönetilmektedir
    * Bildirimleri okundu olarak işaretleme
    * Bildirimleri silme
  </Accordion>
</AccordionGroup>

## Kişisel Ayarlar (Personal Settings)

Hızlı menünün en sağında, aktif kullanıcının kendi profilini, parolasını ya da arayüz dilini güncelleyebileceği öğelere ek olarak oturumunu sonlandırabileceği **Çıkış** bağlantısı bulunur.

### Kişisel Ayarlar Menüsü

<AccordionGroup>
  <Accordion title="Profil">
    * Kullanıcı profil bilgileri
    * İsim, e-posta güncelleme
    * Profil fotoğrafı
  </Accordion>
  
  <Accordion title="Şifre Değiştirme">
    * Mevcut şifre ile yeni şifre belirleme
    * Şifre güvenlik kuralları
  </Accordion>
  
  <Accordion title="Dil Seçimi">
    * Arayüz dili değiştirme
    * Türkçe / İngilizce
    * Dil tercihi kaydetme
  </Accordion>
  
  <Accordion title="Çıkış">
    * Oturumu sonlandırma
    * Güvenli çıkış
  </Accordion>
</AccordionGroup>

## Klavye Kısayolları

Hızlı Menü ve genel platform kullanımı için klavye kısayolları:

<CardGroup cols={2}>
  <Card title="Hızlı Arama" icon="magnifying-glass">
    * `Ctrl+K` veya `Cmd+K`: Hızlı arama menüsünü açar
    * API Proxy, Policy, Connection arama
    * Son kullanılan öğeler
  </Card>
  <Card title="Navigasyon" icon="route">
    * `Esc`: Modal/dialog kapatma
    * `Tab`: Form alanları arası geçiş
    * `Enter`: Form gönderme
  </Card>
  <Card title="İşlemler" icon="keyboard">
    * `Ctrl+S` veya `Cmd+S`: Kaydet
    * `Ctrl+Z` veya `Cmd+Z`: Geri al
    * `Ctrl+Y` veya `Cmd+Y`: Yinele
  </Card>
  <Card title="Menü Erişimi" icon="bars">
    * `Alt+P`: Proje seçimi
    * `Alt+T`: Test araçları
    * `Alt+N`: Bildirimler
  </Card>
</CardGroup>

## Hızlı Arama (Ctrl+K)

<AccordionGroup>
  <Accordion title="Arama Kapsamı">

    * Bağlantılar

  <Accordion title="Son Kullanılan Öğeler">

  </Accordion>
  
  <Accordion title="Favori Sayfalar">

## Son Kullanılan API Proxy'ler

Hızlı Menü, son kullanılan API Proxy'leri hatırlar ve hızlı erişim sağlar:

* **Son Açılanlar**: En son açılan API Proxy'ler
* **Sık Kullanılanlar**: En çok kullanılan API Proxy'ler
* **Favoriler**: Favori olarak işaretlenen API Proxy'ler

## Favori Sayfalar

Kullanıcılar sık kullandıkları sayfaları favori olarak işaretleyebilir:

<AccordionGroup>
  <Accordion title="Favori Ekleme">
    * Sayfa üzerindeki yıldız ikonuna tıklama
    * Favori olarak işaretleme
  </Accordion>
  
  <Accordion title="Favori Erişimi">
    * Hızlı Menüden favori sayfalara erişim
    * Hızlı arama içinde favoriler
  </Accordion>
</AccordionGroup>
