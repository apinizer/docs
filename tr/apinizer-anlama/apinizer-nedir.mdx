---
title: "Apinizer Nedir?"
description: "Apinizer, kurumsal şirketlerin iç ve dış sistemlerindeki API'leri güvenli, performanslı ve yönetilebilir hale getiren **Full Lifecycle API Management** platformudur. Kod yazmadan, görsel arayüzler ve basit konfigürasyonlar ile API Gateway, API Management, Developer Portal ve Analytics ihtiyaçlarınızı tek bir entegre platformda karşılar. Aşağıdaki ürün ailesi ile API yaşam döngüsünün tüm aşamalarını yönetebilir, API'lerinizi güvenli bir şekilde yayınlayabilir ve izleyebilirsiniz."
---



## Apinizer Ürün Ailesi

<CardGroup cols={2}>
  <Card title="Apinizer API Gateway" icon="shield">
    API/Web Servisler ile ilgili güvenlik, trafik yönetimi, yük dengeleme, loglama, mesaj içeriği dönüştürme ve zenginleştirme, doğrulama, test etme ve daha birçok işi **kod yazmadan**, form tabanlı arayüzlerde konfigürasyon yaparak gerçekleştirme.
  </Card>
  <Card title="Apinizer API Creator → DB-2-API" icon="database">
    Veri tabanı işlemleri için işletilecek SQL dışında **kod yazmadan** ayrı bir sunucuya kurulum ya da yükleme ihtiyacı olmaksızın ve anında (on-the-fly) **API/Web Servis açma**.
  </Card>
  <Card title="Apinizer API Creator → Script-2-API" icon="code">
    JavaScript ya da Groovy kodlarını ayrı bir sunucuya kurulum ya da yükleme ihtiyacı olmaksızın ve anında (on-the-fly) **API/Web Servis** olarak **açma**.
  </Card>
  <Card title="Apinizer API Creator → Mock API Creator" icon="wand-magic-sparkles">
    Herhangi bir sunucu ya da kod yazma ihtiyacı olmadan, anında **Mock API** oluşturup kullanıma açma.
  </Card>
  <Card title="Apinizer API Monitor" icon="chart-line">
    API/Web Servislerin ya da dış sistemlerin sağlıklı bir şekilde çalışıp çalışmadığını **otomatik** olarak **sürekli izleme** ve **sorunlardan** anında **haberdar olma**.
  </Card>
  <Card title="Apinizer API Analytics" icon="chart-bar">
    API/Web Servislerin performansını, kullanım detaylarını ve hata metriklerini gerçek zamanlı olarak izleme ve analiz etme. Hazır dashboard'lar ile sistemin genel durumunu görüntüleme, özelleştirilmiş sorgular ve raporlar oluşturma, anomali tespiti ile anormal durumları yakalama ve API bazlı, endpoint bazlı, istemci bazlı detaylı analitik veriler toplama imkanı sunar.
  </Card>
  <Card title="Apinizer API Portal" icon="globe">
    API Tüketicilere API Ürünlerini görme, dokümantasyona erişim, kayıt olma, API'leri deneme ve kullanma imkanı sunan portal. Apinizer ile entegre olarak çalışabileceği gibi, bağımsız olarak da kullanılabilir.
  </Card>
  <Card title="Apinizer Identity Provider" icon="key">
    JWT ve OAuth2 token üretimi ve validasyonu sağlayan kimlik sağlayıcı servisi. API'leriniz için güvenli token tabanlı kimlik doğrulama ve yetkilendirme çözümleri sunar. OAuth2 protokolü ile uyumlu token üretimi, JWT token oluşturma ve doğrulama, token yenileme ve yönetimi gibi özelliklerle API güvenliğinizi güçlendirir.
  </Card>
  <Card title="Apinizer Integrator" icon="puzzle-piece">
    Entegrasyon gereksinimlerini **kod yazmadan** görsel iş akışı tasarımcısı ile basit iş akışları tasarlayarak karşılama. Zamanlanmış görevler (schedule) ile belirli zamanlarda otomatik olarak çalıştırma, API çağrıları, veri tabanı işlemleri, dosya işlemleri ve e-posta gönderme gibi çeşitli görevleri birleştirerek karmaşık entegrasyon senaryolarını gerçekleştirme. İstenirse tasarlanan iş akışlarını **REST API** olarak açarak dış sistemlerden de tetiklenebilir hale getirme.
  </Card>
  <Card title="Apinizer Platformu" icon="layer-group">
    API ekiplerine Yazılım Geliştirme Yaşam Döngüsü (SDLC) süreçlerine paralel bir **API Geliştirme Yaşam Döngüsü** ortamı sağlama. Böylece kullanıcıların farklı roller üstlenerek işbirliği içinde çalışabilecekleri ve gereksinim belirleme, belgeleme, tasarım, geliştirme, test etme, yükleme, sürüm oluşturma, izleme, analiz etme, raporlama, portal üzerinden yayına alma ve fiyatlandırma gibi API yaşam döngüsü adımlarını işletebilecekleri tek ve entegre bir platform sunma.
  </Card>
</CardGroup>
