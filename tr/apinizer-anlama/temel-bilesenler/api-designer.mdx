API Designer (API Tasarımcısı), Apinizer platformunda API'lerin tasarımı, dokümantasyonu ve spesifikasyonu için araçlar sağlayan bileşendir.

## API Designer Kavramı

API Designer, API geliştiricilerinin API'lerini tasarlaması, dokümante etmesi ve spesifikasyonlarını oluşturması için kullanılan araç setidir.

## API Designer Kullanım Senaryoları

### Senaryo 1: Yeni API Tasarımı

```
1. API Designer'da yeni API oluşturma
2. Endpoint'leri tasarlama
3. Schema'ları tanımlama
4. Dokümantasyon oluşturma
5. Mock API oluşturma
6. Test etme
```

### Senaryo 2: Mevcut API Dokümantasyonu

```
1. Mevcut API'yi import etme
2. Dokümantasyonu güncelleme
3. Örnekler ekleme
4. Portal'da yayınlama
```

### Senaryo 3: API Versiyonlama

```
1. Mevcut API'den yeni versiyon oluşturma
2. Değişiklikleri yapma
3. Breaking changes'i belirleme
4. Dokümantasyonu güncelleme
```

## API Designer ve API Manager İlişkisi

API Designer, API Manager ile entegre çalışır:

```
API Designer
   │
   │ API Tasarımı
   │ OpenAPI Spec
   │
   ▼
API Manager
   │
   │ API Proxy Oluşturma
   │ Konfigürasyon
   │
   ▼
API Gateway
```
