---
title: "API Creator"
description: "API Creator (API Oluşturucu), Apinizer platformunda kod yazmadan veritabanı, script veya mock API'lerden anında API oluşturma imkânı sağlayan bileşendir. Üç ana modülü vardır: DB-2-API (veritabanından API oluşturma), Script-2-API (JavaScript/Groovy script'lerinden API oluşturma) ve Mock API Creator (Mock API oluşturma). Form tabanlı arayüzlerle hızlı bir şekilde API oluşturulmasını sağlar."
---

## API Creator ve API Gateway İlişkisi

API Creator ile oluşturulan API'ler otomatik olarak API Gateway üzerinden yayınlanır. İşleyiş akışı:

```
API Creator
   │
   │ API Oluşturma
   │ (DB-2-API / Script-2-API / Mock API)
   │
   ▼
API Gateway
   │
   │ Otomatik Yayınlama
   │ Trafik Yönetimi
   │
   ▼
Client Requests
```

<CardGroup cols={3}>
  <Card title="1. API Oluşturma" icon="plus-circle">
    API Creator ile API oluşturma
    
    DB-2-API, Script-2-API veya Mock API oluşturma.
  </Card>
  <Card title="2. Otomatik Yayınlama" icon="rocket">
    API Gateway üzerinden otomatik yayınlama
    
    Oluşturulan API'ler otomatik olarak gateway'e eklenir.
  </Card>
  <Card title="3. Trafik Yönetimi" icon="route">
    API Gateway trafik yönetimi
    
    Güvenlik, rate limiting ve routing uygulanır.
  </Card>
</CardGroup>

<Info>
API Creator ile oluşturulan API'ler, API Gateway'in tüm özelliklerinden (güvenlik, rate limiting, monitoring, analytics) otomatik olarak yararlanır.
</Info>
