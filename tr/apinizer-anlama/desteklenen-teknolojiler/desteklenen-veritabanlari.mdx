# Veritabanı Bağlantısı

Veritabanı Bağlantısı, Apinizer'da harici veritabanı sistemlerine bağlanmak ve veri işlemleri yapmak için kullanılan bağlantı konfigürasyonudur.

## Genel Bakış

Veritabanı Bağlantısı konfigürasyonu ile JDBC ve Mongo URI üzerinden çoklu veritabanı tiplerini destekleyen havuzlu bağlantı yönetimi sağlanır.

## Desteklenen Veritabanları

Apinizer aşağıdaki veritabanı sistemlerini destekler:

- **Oracle Database**
- **PostgreSQL**
- **MySQL**
- **Microsoft SQL Server**
- **MongoDB**
- **IBM DB2**
- **MariaDB**
- **H2 Database**
- **SQLite**
- **Apache Derby**

## Bağlantı Havuzu

Veritabanı bağlantıları havuz yönetimi ile yönetilir. Bu sayede:

- Bağlantı performansı optimize edilir
- Kaynak kullanımı kontrol altında tutulur
- Bağlantı sayısı sınırlandırılabilir
- Bağlantı zaman aşımı ayarları yapılabilir

## Konfigürasyon

Veritabanı bağlantısı oluştururken aşağıdaki bilgiler gereklidir:

- **Bağlantı Adı**: Bağlantıyı tanımlayan benzersiz isim
- **Veritabanı Tipi**: Kullanılacak veritabanı sistemi
- **Bağlantı URL'si**: Veritabanına erişim için JDBC URL veya MongoDB URI
- **Kullanıcı Adı**: Veritabanı kullanıcı adı
- **Şifre**: Veritabanı şifresi
- **Havuz Ayarları**: Minimum ve maksimum bağlantı sayıları, timeout değerleri

## Kullanım

Veritabanı bağlantıları şu durumlarda kullanılır:

- **DB-2-API**: Veritabanı üzerinden API oluşturma
- **Politikalar**: Veritabanı sorguları çalıştıran politikalar
- **API Integrator**: Veritabanı işlemleri içeren görev akışları
- **Connector'lar**: Veritabanı bağlantısı gerektiren connector'lar

## Güvenlik

Veritabanı bağlantı bilgileri Apinizer'ın Secret Manager'ında güvenli bir şekilde saklanır ve şifrelenir.
