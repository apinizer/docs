Variable (Değişken), Apinizer platformunda API Proxy'lerde ve Task Flow'larda kullanılan dinamik değerleri saklayan yapılardır. Değişkenler, mesaj içeriğinden, header'lardan, sistem özelliklerinden veya manuel olarak tanımlanan değerlerden alınabilir.

## Variable Kavramı

Değişkenler, API Proxy ve Task Flow'larda dinamik değerleri temsil eder. Bu değerler mesaj işleme sırasında kullanılır.

## Variable Türleri

### Mesaj Değişkenleri

Mesaj içeriğinden alınan değerler:

* **request.body**: İstek gövdesi
* **response.body**: Yanıt gövdesi
* **request.header**: İstek header'ları
* **response.header**: Yanıt header'ları
* **request.query**: Query parametreleri
* **request.path**: Path parametreleri

### Sistem Değişkenleri

Sistem özelliklerinden alınan değerler:

* **system.timestamp**: Zaman damgası
* **system.uuid**: Benzersiz tanımlayıcı
* **system.environment**: Ortam adı
* **system.project**: Proje adı

### Kullanıcı Tanımlı Değişkenler

Manuel olarak tanımlanan değişkenler:

* **Custom Variables**: Kullanıcı tarafından tanımlanan değişkenler
* **Environment Variables**: Ortam değişkenleri
* **Project Variables**: Proje değişkenleri

## Variable Kullanımı

Değişkenler şu durumlarda kullanılır:

* **Politika Yapılandırması**: Politika ayarlarında dinamik değerler
* **Routing**: Yönlendirme kurallarında dinamik hedef seçimi
* **Dönüştürme**: Mesaj dönüştürme işlemlerinde dinamik değerler
* **Koşullu İfadeler**: Koşullu politika çalıştırmada değer kontrolü
* **Script**: Groovy/JavaScript script'lerinde değişken kullanımı

## Variable Syntax

Değişkenler farklı syntax'larla kullanılabilir:

### JSONPath (JSON için)

```json
$.user.name
$.products[0].price
$.items[*].id
```

### XPath (XML için)

```xpath
/user/name
/products/product[1]/price
//item[@id='123']
```

### Header Değişkenleri

```
request.header["Content-Type"]
response.header["X-Custom-Header"]
```

### Query Parametreleri

```
request.query["page"]
request.query["limit"]
```

## Variable Dönüştürme

Değişkenler dönüştürülebilir:

* **String Operations**: Uppercase, lowercase, substring
* **Number Operations**: Math operations, formatting
* **Date Operations**: Date formatting, timezone conversion
* **Custom Transformations**: Script ile özel dönüştürmeler
