API Proxy, Apinizer platformunun temel bileşenidir. Backend API'lerin önünde duran ve güvenlik, trafik yönetimi, mesaj dönüştürme, doğrulama ve zenginleştirme gibi işlevleri sağlayan bir katmandır.

## API Proxy Nedir?

API Proxy, istemcilerden gelen istekleri alan, bu isteklere çeşitli politikalar uygulayan ve ardından backend API'lere yönlendiren bir ara katmandır. Backend API'lerin önünde durarak, API yönetimi ve güvenlik işlevlerini merkezi bir noktadan sağlar.

## API Proxy Nasıl Çalışır?

### İstek Akışı

```
İstemci
   │
   ▼ (HTTPS Request)
┌─────────────┐
│ API Proxy   │ ← Politika Uygulama Noktası (Policy Enforcement Point)
│             │
│ 1. Güvenlik │ (Kimlik doğrulama, yetkilendirme)
│ 2. Doğrulama│ (Mesaj doğrulama, şema kontrolü)
│ 3. Dönüştürme│ (Format dönüştürme)
│ 4. Zenginleştirme│ (Header ekleme, veri ekleme)
│ 5. Yönlendirme│ (Backend'e yönlendirme)
└──────┬──────┘
       │
       ▼ (Backend API Request)
   Backend API
```

### Yanıt Akışı

```
Backend API
   │
   ▼ (Response)
┌─────────────┐
│ API Proxy   │
│             │
│ 1. Dönüştürme│ (Format dönüştürme)
│ 2. Maskeleme│ (Hassas veri maskeleme)
│ 3. Loglama │ (İşlem kaydı)
│ 4. Yanıt │ (İstemciye yanıt)
└──────┬──────┘
       │
       ▼ (HTTPS Response)
   İstemci
```

## API Proxy Oluşturma

API Proxy oluştururken şu adımlar izlenir:

1. **API Tanımı**: OpenAPI/Swagger veya WSDL dosyasından import
2. **Routing Yapılandırması**: Client Route ve Upstream Target tanımlama
3. **Politika Ekleme**: Güvenlik, doğrulama, dönüştürme politikaları
4. **Test**: Test Console ile test etme
5. **Deployment**: Ortam(lar)a yükleme

## API Proxy Bileşenleri

### Client Route

İstemcilerden gelen isteklerin karşılandığı endpoint. API Proxy'nin dış dünyaya açılan kapısıdır.

### Upstream Target

Backend API'nin adresi. API Proxy'nin istekleri yönlendirdiği hedeftir.

### Policies

API Proxy'ye uygulanan politikalar:
* Güvenlik politikaları (OAuth2, JWT, mTLS)
* Trafik yönetimi politikaları (Rate Limiting, Quota)
* Dönüştürme politikaları (JSON Transform, XML Transform)
* Doğrulama politikaları (JSON Schema, XML Schema)
