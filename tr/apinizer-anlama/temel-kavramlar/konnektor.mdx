Konnektör (Connector), Apinizer API Integrator'da farklı kaynaklara bağlanabilmeyi, bu kaynaklara veri gönderebilmeyi veya bu kaynaklardan veri çekebilmeyi sağlayan bileşenlerdir.

## Konnektör Kavramı

Konnektörler, API Integrator'ın Task Flow'larında kullanılan, belirli bir sistem veya protokol ile iletişim kuran bileşenlerdir. Her konnektör, belirli bir kaynak türü için özelleştirilmiştir.

## Konnektör Türleri

### Veritabanı Konnektörleri

* **Database Connector**: SQL veritabanlarına bağlantı (MySQL, PostgreSQL, Oracle, MSSQL, DB2)

### Mesaj Kuyruğu Konnektörleri

* **Kafka Connector**: Apache Kafka entegrasyonu
* **RabbitMQ Connector**: RabbitMQ entegrasyonu
* **ActiveMQ Connector**: Apache ActiveMQ entegrasyonu

### İletişim Konnektörleri

* **Email Connector**: E-posta gönderme
* **HTTP/REST Connector**: HTTP/REST API çağrıları
* **Webhook Connector**: Webhook gönderme

### Dosya ve Veri Konnektörleri

* **FTP Connector**: FTP/SFTP dosya transferi
* **Elasticsearch Connector**: Elasticsearch sorguları
* **Script Connector**: JavaScript/Groovy script çalıştırma

### Monitoring ve Bildirim Konnektörleri

* **Notification Connector**: Bildirim gönderme
* **SNMP Connector**: SNMP trap gönderme
* **Syslog Connector**: Syslog mesajları gönderme
* **Logback Connector**: Log kayıtları gönderme

### Sistem Konnektörleri

* **Linux Script Connector**: Linux komutları çalıştırma
* **JiraOps (Opsgenie) Connector**: Jira/Opsgenie entegrasyonu

## Konnektör Kullanımı

Konnektörler, API Integrator'ın Task Flow'larında kullanılır:

1. **Task Flow Oluşturma**: Yeni bir Task Flow oluşturulur
2. **Konnektör Ekleme**: İhtiyaç duyulan konnektörler eklenir
3. **Yapılandırma**: Konnektörler yapılandırılır (bağlantı bilgileri, parametreler)
4. **Bağlantı**: Konnektörler birbirine bağlanarak akış oluşturulur
5. **Çalıştırma**: Task Flow çalıştırılır

## Custom Konnektör Geliştirme

Gerekirse custom konnektörler geliştirilebilir:

* **Java API**: Java ile konnektör geliştirme
* **Plugin Sistemi**: Konnektör plugin'leri oluşturma
* **Yeniden Kullanım**: Custom konnektörlerin paylaşılması
