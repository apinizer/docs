## Apinizer Kurulum Dosyalarının İndirilmesi

**Apinizer İmajları**

* İndirilebilir son versiyona ait indirme linki için lütfen bizimle [support@apinizer.com](mailto:support@apinizer.com) mail adresinden iletişime geçiniz.

* Ya da belirli bir versiyonu kendiniz indirmek ve ilgili sunucuya aktarmak için aşağıdaki komutları kullanabilirsiniz (manager, worker ve cache zorunlu deployment isimleridir. Lisansınıza bağlı olarak integration ve portal deployment'ları da gerekebilecek imajlardır).

### Docker Kullanıyorsanız

```bash
sudo docker pull apinizercloud/manager:<APINIZER_VERSION>
sudo docker save apinizercloud/manager:<APINIZER_VERSION> > manager_<APINIZER_VERSION>.tar
sudo docker load < manager_<APINIZER_VERSION>.tar
```

### Containerd Kullanıyorsanız

```bash
sudo ctr -n k8s.io images list | grep apinizer
sudo ctr --namespace k8s.io images pull docker.io/apinizercloud/manager:<APINIZER_VERSION>
sudo ctr -n k8s.io images export manager_<APINIZER_VERSION>.tar docker.io/apinizercloud/manager:<APINIZER_VERSION>
sudo ctr -n=k8s.io images import manager_<APINIZER_VERSION>.tar
```

<Info>
  Paket isimleri ve indirilebilecek versiyonlar için: [https://hub.docker.com/u/apinizercloud](https://hub.docker.com/u/apinizercloud)
</Info>
