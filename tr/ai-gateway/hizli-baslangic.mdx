## Genel Bakış

Bu sayfa, Apinizer AI Gateway üzerinden ilk AI proxy'nizi uçtan uca kurmanız için gereken dört adımı gösterir.

<Steps>
  <Step title="LLM Sağlayıcı Bağlantısı Ekleyin">
    İstek göndereceğiniz LLM sağlayıcısına (OpenAI, Anthropic, Azure vb.) ait bir bağlantı tanımlayın. Bağlantı, sağlayıcının API kimlik bilgilerini şifreli olarak saklar ve dağıtım türünü (bulut / şirket içi) belirler.

    Detaylı adımlar için [LLM Sağlayıcıları ve Bağlantılar](/tr/ai-gateway/llm-saglayicilari) sayfasına bakın.
  </Step>

  <Step title="Sanal API Anahtarı Oluşturun">
    İstemcilerinizin kullanacağı bir **sanal API anahtarı** oluşturun. Sanal anahtar gerçek sağlayıcı anahtarını istemciden gizler; kapsamını (kullanıcı, rol, proje veya takım) siz belirlersiniz.

    Detaylar için [Sanal API Anahtarları](/tr/ai-gateway/sanal-api-anahtarlari) sayfasına bakın.
  </Step>

  <Step title="OpenAI SDK base_url Yönlendirmesi">
    İstemci kodunuzda `base_url`'i Apinizer AI Gateway adresinize, `api_key`'i ise oluşturduğunuz sanal anahtara yönlendirin:

    ```python
    from openai import OpenAI

    client = OpenAI(
        api_key="sanal-api-anahtarınız",
        base_url="https://apinizer-gateway-adresiniz.com/api/ai/v1"
    )
    ```

    Bunun dışında istemci kodunuzda herhangi bir değişiklik gerekmez.
  </Step>

  <Step title="İlk İsteği Gönderin (Streaming)">
    Sohbet tamamlama isteğinizi akış (streaming) modunda gönderin:

    ```python
    response = client.chat.completions.create(
        model="gpt-4o",
        messages=[{"role": "user", "content": "Merhaba"}],
        stream=True
    )

    for chunk in response:
        print(chunk.choices[0].delta.content, end="")
    ```

    İstek; korumalar, semantik önbellek, hız sınırlaması ve yönlendirme aşamalarından geçtikten sonra seçilen LLM sağlayıcısına iletilir ve yanıt chunk başına geri akar.
  </Step>
</Steps>

:::tip
İsteğinizin hangi aşamalardan geçtiğini görmek için [AI Gateway Genel Bakış](/tr/ai-gateway/genel-bakis) sayfasındaki istek akışı diyagramına bakabilirsiniz.
:::
