# Yasaklı IP Listesi (Blocked IP List)

Yasaklı IP Listesi Politikası ile bir IP ya da IP aralığı listesi tanımlanarak, bu listedeki IP'lerden gelen isteklerin kabul edilmesi engellenir.

Apinizer'da IP kontrolü yapılırken eğer varsa öncelikle X-Forwarded-For politikasında ayarlanan İstek Başlık değerine bakılır. Bu politika yok ise istemcinin Remote Address değerine bakılır.

Apinizer'da IP kontrolü yapılırken Network Level7 seviyesindeki değer kontrol edilir. Eğer Network Level4 seviyesindeki değer kontrol edilmek isteniyorsa buna uygun bir yazılım veya donanım kullanılmalıdır.

**Politika ayarlarını içeren görsele aşağıda yer verilmiştir:**

![Yasaklı IP Listesi Politikası](https://docs.apinizer.com/apinizer-dokumantasyonu/files/13272196/120522029/1/1737968075199/image-2025-1-27_11-54-35.png)

Politika konfigürasyonu için kullanılan alanlar aşağıdaki tabloda görülmektedir.

| Alan | Açıklama |
| --- | --- |
| İsim *(Name)* | Politikanın kullanımını ve yönetimini kolaylaştırmak için isim yazılabilir. Politikaların yönetimi ve seçimi sırasında bu isme ihtiyacınız olacaktır. |
| Açıklama *(Description)* | Politikanın kullanımını ve yönetimini kolaylaştırmak için açıklama yazılabilir. |
| İstemci IP'sini Kullan *(Use Client IP)* | Bu seçenek seçilirse İstemci IP'si kullanılır. |
| İstemci IP için Değişken *(Variable for Client IP)* | İstemci IP'sini Kullan aktif değilse bu alan gelir. İstemci IP'si için değişken seçilebilir. |
| Yasaklı IP Listesi *(Blocked IP List)* | IP aralığı belirtmek için ('*' ve '-') karakterleri kullanılabilir. Örneğin:<br/>- '10.3.10.*' değeri '10.3.10.0'ile '10.3.10.255' arasındaki IP'leri belirtir.<br/>- '10.3.10.4-18' değeri '10.3.10.4' ile '10.3.10.18'arasındaki IP'leri ifade eder (4 ve 18 dahil).<br/><br/>IP ekledikten sonra "enter" tuşuna basılmalıdır. |
| IP Grupları *(IP Groups)* | Politikaya bir ya da daha çok IP Grubu eklenebilir. IP Grubundaki bütün IP'ler politikaya eklenmiş olur. (IP Grupları hakkında daha fazla bilgi için [IP Grupları (IP Groups)](/tr/gelistirici/politikalar/ip-groups) sayfasını ziyaret edebilirsiniz.) |
| IP Geolocation *(IP Geolocation)* | Belirli ülkelerden veya şehirlerden gelen isteklerin kısıtlanmasını sağlar. |

**Koşullar (Conditions)** ve **Hata Mesajı Özelleştirme (Error Message Customization)** panellerinin detayı için [Politikalar (Policies)](/tr/gelistirici/politika-yonetimi) sayfasını ziyaret edebilirsiniz.
