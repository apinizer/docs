# WS-Security Username Token

WS-Security Username Token politikası, Web Servis mesajlarında kullanıcı adı ve şifre doğrulaması için kullanılır.

Bu politika, SOAP mesajlarında WS-Security standardına uygun olarak UsernameToken elementini doğrular ve kullanıcı kimlik bilgilerini kontrol eder.

Detaylı bilgi için eski dokümantasyon sayfasına bakınız: [WS-Security Username Token](https://docs.apinizer.com/display/APD/WS-Security+Username-2)
