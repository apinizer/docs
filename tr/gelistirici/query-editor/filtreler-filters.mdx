---
title: 'Filtreler (Filters)'
description: 'Filtre tanımları, filtre oluşturma, zaman aralığı, API Proxy filtreleme ve sonuç tipi filtreleme'
---

Filtre tanımı, sorgu çalıştırıldığında loglar ile eşleşmesi gereken zaman aralığı, belirli API Proxy'lerde arama yapma, isteğin sonuç tipine göre kısıtlama (başarılı, başarısız, bloklanmış) vs. gibi kriterler içerir.

Genel olarak sorgular ile birlikte kullanılır.

## Filtre Oluşturma

**Filtre ayarlarını içeren görsele aşağıda yer verilmiştir:**

![Filtre Ayarları](../../../images/filtre-ayarlari.png)

Filtre konfigürasyonu için kullanılan alanlar aşağıdaki tabloda görülmektedir.

| Alan | Açıklama |
|------|----------|
| **Ad** *(Name)* | Oluşturulan filtre için filtre adı bilgisidir. |
| **Açıklama** *(Description)* | Oluşturulan filtre ile ilgili yönetimi kolaylaştırmak için açıklama yazılabilir. |
| **Proxyler** *(Proxies)* | Filtrenin uygulanacağı Proxyler seçilir. Herhangi bir Proxy seçilmemiş ise filtre tüm Proxylere uygulanır. |
| **Tarih Aralığı** *(Date Range)* | Filtrenin çalışacağı tarih aralığı seçilir. |
| **İşlem Sonuç Tipi** *(Result Type)* | Log kayıtlarını filtrelemek için istek işleminin sonuç tip bilgileri seçilebilir. |
| **Sıralama** *(Order List)* | Sorgu sonucunun sıralanması için log kaydından yer alan alanlar üzerinden sıralama koşulları eklenebilir. |
