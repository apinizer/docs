## Filtre Operatörleri

### Karşılaştırma Operatörleri

* **Eşittir (=)**: Değer eşitliği kontrolü
* **Eşit Değildir (!=)**: Değer eşitsizliği kontrolü
* **Büyüktür (>)**: Büyüktür kontrolü
* **Küçüktür (<)**: Küçüktür kontrolü
* **Büyük Eşittir (>=)**: Büyük eşittir kontrolü
* **Küçük Eşittir (<=)**: Küçük eşittir kontrolü

### Metin Operatörleri

* **İçerir (LIKE)**: Metin içerme kontrolü
* **Başlar (STARTS WITH)**: Metin başlangıç kontrolü
* **Biter (ENDS WITH)**: Metin bitiş kontrolü
* **Regex**: Regex pattern eşleştirme

### Mantıksal Operatörler

* **VE (AND)**: Tüm koşulların sağlanması
* **VEYA (OR)**: Koşullardan birinin sağlanması
* **DEĞİL (NOT)**: Koşulun tersi

## Filtre Oluşturma

### Basit Filtre

Tek bir koşul ile filtre:

```
status = 'active'
```

### Karmaşık Filtre

Birden fazla koşul ile filtre:

```
status = 'active' AND created_at > '2024-01-01'
```
