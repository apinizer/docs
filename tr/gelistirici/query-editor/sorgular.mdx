## Sorgu Oluşturma

### SELECT Sorguları

Veri çekme sorguları:

```sql
SELECT * FROM users WHERE status = 'active';
```

### INSERT Sorguları

Veri ekleme sorguları:

```sql
INSERT INTO users (name, email) VALUES ('John Doe', 'john@example.com');
```

### UPDATE Sorguları

Veri güncelleme sorguları:

```sql
UPDATE users SET status = 'inactive' WHERE id = 1;
```

### DELETE Sorguları

Veri silme sorguları:

```sql
DELETE FROM users WHERE id = 1;
```

## Sorgu Test Etme

### Sorgu Çalıştırma

1. Sorguyu yazın
2. **Çalıştır** butonuna tıklayın
3. Sonuçları görüntüleyin

### Sonuç Görüntüleme

* **Tablo Görünümü**: Sonuçları tablo formatında görüntüleme
* **JSON Görünümü**: Sonuçları JSON formatında görüntüleme
* **Export**: Sonuçları CSV, JSON veya Excel formatında export etme
