Query Editor, Apinizer platformunda veritabanı sorguları ve filtreler oluşturmak, test etmek ve yönetmek için kullanılan bir araçtır. Bu sayfa, Query Editor'ın genel özelliklerini ve kullanım alanlarını açıklar.

## Query Editor Nedir?

Query Editor, veritabanı sorguları oluşturmanızı, düzenlemenizi ve test etmenizi sağlayan bir araçtır. Kod yazmadan, görsel bir arayüzle sorgular oluşturabilirsiniz.

## Kullanım Alanları

Query Editor şu durumlarda kullanılır:

* **Veritabanı Sorguları**: Veritabanından veri çekme sorguları
* **Filtreleme**: Veri filtreleme işlemleri
* **Test ve Doğrulama**: Sorguları test etme ve doğrulama
* **Veri Analizi**: Veri analizi için sorgular

## Temel Arayüz

Query Editor arayüzü şu bölümlerden oluşur:

### Sorgu Oluşturma Alanı

* **Sorgu Editörü**: Sorgu yazma ve düzenleme alanı
* **Syntax Highlighting**: SQL syntax vurgulama
* **Auto-complete**: Otomatik tamamlama desteği

### Filtre Paneli

* **Filtre Ekleme**: Filtre ekleme ve yönetme
* **Filtre Kombinasyonları**: Birden fazla filtre kombinasyonu
* **Filtre Operatörleri**: Eşitlik, büyüktür, küçüktür gibi operatörler

### Sonuç Görüntüleme

* **Tablo Görünümü**: Sonuçları tablo formatında görüntüleme
* **JSON Görünümü**: Sonuçları JSON formatında görüntüleme
* **Export**: Sonuçları export etme

## Query Editor Özellikleri

### Desteklenen Veritabanları

* **MySQL**: MySQL veritabanları
* **PostgreSQL**: PostgreSQL veritabanları
* **Oracle**: Oracle veritabanları
* **MSSQL**: Microsoft SQL Server
* **DB2**: IBM DB2

### Sorgu Türleri

* **SELECT**: Veri çekme sorguları
* **INSERT**: Veri ekleme sorguları
* **UPDATE**: Veri güncelleme sorguları
* **DELETE**: Veri silme sorguları
* **Stored Procedures**: Stored procedure çağrıları
