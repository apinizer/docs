## Filtreleme ve İşlemler

| Alan | Açıklama |
|------|----------|
| **Başlat** *(Start)* | Uygulama loglarının API Proxy bazında kaydının tutulmasını sağlar |
| **Ortam Filtresi** *(Environment)* | Uygulama loglarının belirtilen ortam için filtrelenmesini sağlar |
| **Zaman Filtresi** *(From, To)* | Uygulama loglarının belirtilen tarih aralığı için filtrelenmesini sağlar |
| **Log Seviyesi** *(Search by Log Levels)* | Belirtilen seviyedeki logların filtrelenmesini sağlar |
| **Correlation ID'ye göre ara** *(Search in Correlation ID)* | API Proxy'ye bir istek geldiği anda, isteğin takip edilebilmesi için Apinizer tarafından bu isteğe bir korelasyon ID'si tanımlanır. API Proxy Trafik loglarından bulunacak olan korelasyon ID ile uygulama loglarında detaylı arama yapılarak sadece bir isteğin uygulama loglarına erişilebilir |
| **Detaya göre ara** *(Search in Detail)* | Uygulama loglarının metin alanlarında arama yapılmasını sağlar |
| **Stack Trace'e göre ara** *(Search in Stack Trace)* | Exception oluşmuş ise exception metni üzerinde arama yapılmasını sağlar |

## İndirme ve Silme

    - **Arama Sonuçları**: Filtrelenen tüm logları indirir
  </Card>
  <Card title="Sil" icon="trash">
    Bu API Proxy'e ait uygulama log kayıtlarını siler
