## Özellikler

<CardGroup cols={2}>
  <Card title="Performans Özeti" icon="gauge-high">
    API Proxy'nin genel performans metriklerini görüntüler
  </Card>
  <Card title="Uç Nokta Kullanımı" icon="chart-pie">
    Hangi uç noktaların (endpoint) ne sıklıkla kullanıldığını gösterir
  </Card>
  <Card title="İstemci Aktivitesi" icon="users">
    İstemcilerin API kullanım istatistiklerini sunar
  </Card>
  <Card title="Otomatik Yenileme" icon="arrows-rotate">
    **Yenileme Zamanını Seçininiz** seçeneği ile verileri belirli aralıklarla otomatik güncelleyebilir
  </Card>
</CardGroup>

<Tip>
Yenileme aralığı seçildiğinde veriler, sayfa değiştirilene kadar otomatik olarak yenilenir.
</Tip>
