## XSLT Nedir?

XSLT (eXtensible Stylesheet Language Transformations), XML belgelerini başka XML belgelerine veya diğer formatlara (HTML, text, vb.) dönüştürmek için kullanılan bir dildir. XSLT stil sayfaları, kaynak XML yapısını hedef yapıya dönüştürmek için kullanılan kuralları tanımlar.

### XSLT Temel Kavramları

* **Template**: Dönüşüm kurallarını tanımlayan şablonlar
* **XPath**: XML içinde veri seçmek için kullanılan ifadeler
* **Namespace**: XML namespace'lerini yönetme
* **Output**: Dönüşüm sonucunun formatı

## XML Transformer Kullanımı

### Arayüz Bileşenleri

XML Transformer aracı üç ana bölümden oluşur:

1. **Input XML**: Dönüştürülecek kaynak XML verisi
2. **XSLT Stylesheet**: Dönüşüm kurallarını içeren XSLT stil sayfası
3. **Output XML**: Dönüşüm sonucu

### Adım 1: Input XML Girişi

Input XML bölümüne dönüştürülecek kaynak XML verisini girin:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<books>
  <book id="1">
    <title>XML Guide</title>
    <author>John Doe</author>
    <price>29.99</price>
  </book>
  <book id="2">
    <title>XSLT Tutorial</title>
    <author>Jane Smith</author>
    <price>39.99</price>
  </book>
</books>
```

#### Input Seçenekleri

* **Manuel Giriş**: XML'i doğrudan metin alanına yazın
* **Dosya Yükleme**: XML dosyasını yükleyin
* **Örnek Yükleme**: Hazır örneklerden birini seçin
* **Format**: XML formatını otomatik olarak düzenleme

### Adım 2: XSLT Stylesheet Yazma

XSLT Stylesheet bölümüne dönüşüm kurallarını yazın:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<xsl:stylesheet version="1.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform">
  <xsl:output method="xml" indent="yes"/>
  
  <xsl:template match="/">
    <catalog>
      <xsl:for-each select="books/book">
        <item>
          <id><xsl:value-of select="@id"/></id>
          <name><xsl:value-of select="title"/></name>
          <writer><xsl:value-of select="author"/></writer>
          <cost><xsl:value-of select="price"/></cost>
        </item>
      </xsl:for-each>
    </catalog>
  </xsl:template>
</xsl:stylesheet>
```

#### XSLT Stylesheet Özellikleri

* **Syntax Highlighting**: XSLT için syntax highlighting desteği
* **Auto-completion**: Otomatik tamamlama önerileri
* **Error Detection**: Syntax hatalarını anında tespit etme
* **Namespace Support**: Namespace desteği

### Adım 3: Dönüşüm Sonucunu Görüntüleme

Output XML bölümünde dönüşüm sonucu görüntülenir:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<catalog>
  <item>
    <id>1</id>
    <name>XML Guide</name>
    <writer>John Doe</writer>
    <cost>29.99</cost>
  </item>
  <item>
    <id>2</id>
    <name>XSLT Tutorial</name>
    <writer>Jane Smith</writer>
    <cost>39.99</cost>
  </item>
</catalog>
```

## Namespace Handling

XML Transformer aracı, XML namespace'lerini yönetmek için gelişmiş özellikler sunar:

### Namespace Tanımlama

Input XML'de namespace tanımlayabilirsiniz:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<books xmlns="http://example.com/books"
       xmlns:author="http://example.com/author">
  <book id="1">
    <title>XML Guide</title>
    <author:name>John Doe</author:name>
    <price>29.99</price>
  </book>
</books>
```

### XSLT'de Namespace Kullanımı

XSLT stil sayfasında namespace'leri kullanabilirsiniz:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<xsl:stylesheet version="1.0" 
                xmlns:xsl="http://www.w3.org/1999/XSL/Transform"
                xmlns:book="http://example.com/books"
                xmlns:author="http://example.com/author">
  
  <xsl:template match="book:books">
    <catalog>
      <xsl:for-each select="book:book">
        <item>
          <name><xsl:value-of select="book:title"/></name>
          <writer><xsl:value-of select="author:name"/></writer>
        </item>
      </xsl:for-each>
    </catalog>
  </xsl:template>
</xsl:stylesheet>
```

### Namespace Özellikleri

* **Namespace Prefix Mapping**: Prefix'leri namespace'lere eşleme
* **Default Namespace**: Varsayılan namespace tanımlama
* **Namespace Preservation**: Namespace'leri koruma
* **Namespace Declaration**: Output'ta namespace bildirimleri

## XSLT Debugging

XML Transformer aracı, XSLT hatalarını ayıklamak için çeşitli özellikler sunar:

### Hata Gösterimi

XSLT hataları şu şekilde gösterilir:

* **Satır Numarası**: Hatanın bulunduğu satır
* **Hata Mesajı**: Açıklayıcı hata mesajı
* **Vurgulama**: Hatalı bölümün vurgulanması
* **Stack Trace**: Hata stack trace'i

### Debugging Özellikleri

* **Step-by-Step Execution**: Adım adım çalıştırma
* **Variable Inspection**: Değişken değerlerini inceleme
* **Template Matching**: Template eşleşmelerini görüntüleme
* **XPath Evaluation**: XPath ifadelerini değerlendirme

### Hata Türleri

* **Syntax Errors**: XSLT syntax hataları
* **Runtime Errors**: Çalışma zamanı hataları
* **XPath Errors**: XPath ifade hataları
* **Namespace Errors**: Namespace hataları

## Kullanım Senaryoları

### Senaryo 1: Basit XML Dönüşümü

**Input XML:**
```xml
<person>
  <firstName>John</firstName>
  <lastName>Doe</lastName>
</person>
```

**XSLT Stylesheet:**
```xml
<xsl:stylesheet version="1.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform">
  <xsl:template match="/">
    <user>
      <name>
        <first><xsl:value-of select="person/firstName"/></first>
        <last><xsl:value-of select="person/lastName"/></last>
      </name>
    </user>
  </xsl:template>
</xsl:stylesheet>
```

**Output XML:**
```xml
<user>
  <name>
    <first>John</first>
    <last>Doe</last>
  </name>
</user>
```

### Senaryo 2: Dizi İşleme

**Input XML:**
```xml
<products>
  <product id="1" name="Product A" price="100"/>
  <product id="2" name="Product B" price="200"/>
</products>
```

**XSLT Stylesheet:**
```xml
<xsl:stylesheet version="1.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform">
  <xsl:template match="/">
    <items>
      <xsl:for-each select="products/product">
        <item>
          <id><xsl:value-of select="@id"/></id>
          <title><xsl:value-of select="@name"/></title>
          <cost><xsl:value-of select="@price"/></cost>
        </item>
      </xsl:for-each>
    </items>
  </xsl:template>
</xsl:stylesheet>
```

**Output XML:**
```xml
<items>
  <item>
    <id>1</id>
    <title>Product A</title>
    <cost>100</cost>
  </item>
  <item>
    <id>2</id>
    <title>Product B</title>
    <cost>200</cost>
  </item>
</items>
```

### Senaryo 3: Koşullu Dönüşüm

**Input XML:**
```xml
<orders>
  <order status="pending" amount="100"/>
  <order status="completed" amount="200"/>
</orders>
```

**XSLT Stylesheet:**
```xml
<xsl:stylesheet version="1.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform">
  <xsl:template match="/">
    <orderList>
      <xsl:for-each select="orders/order">
        <xsl:if test="@status='completed'">
          <completedOrder>
            <amount><xsl:value-of select="@amount"/></amount>
          </completedOrder>
        </xsl:if>
      </xsl:for-each>
    </orderList>
  </xsl:template>
</xsl:stylesheet>
```

**Output XML:**
```xml
<orderList>
  <completedOrder>
    <amount>200</amount>
  </completedOrder>
</orderList>
```
