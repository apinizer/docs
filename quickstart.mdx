## İlk API Proxy'nizi Oluşturun

Aşağıdaki adımları takip ederek ilk API Proxy'nizi oluşturabilirsiniz:

<Steps>
  <Step title="Demo ortamına giriş yapın">
    [demo.apinizer.com](https://demo.apinizer.com/) adresine gidip giriş yapın. Demo ortamında hesap oluşturduğunuzda Apinizer sizin için varsayılan bir Proje oluşturur.
  </Step>

  <Step title="API Proxy oluşturma sayfasına gidin">
    Sol menüden **Geliştirme → API Proxy'ler** seçeneğine tıklayın ve sağ üstteki **+Create** butonuna tıklayın.
  </Step>

  <Step title="Swagger dosyasını yükleyin">
    Swagger 2.x için **URL Gir** seçeneğine tıklayın ve aşağıdaki URL'yi girin:
    
    ```
    https://petstore.swagger.io/v2/swagger.json
    ```
    
    **Parse** butonuna tıklayarak API tanımını yükleyin.
  </Step>

  <Step title="API Proxy'yi yapılandırın">
    **Address** kısmında `https://petstore.swagger.io/v2` seçeneğini işaretleyin ve **Relative Path** alanına `/petstoreProxy` yazın. Kaydet butonuna tıklayın.
  </Step>

  <Step title="API Proxy'yi deploy edin">
    **Develop** sekmesine gidin ve üstteki **Deploy** butonuna tıklayın. Açılan pencerede ortamı seçin ve deploy işlemini tamamlayın.
  </Step>

  <Step title="API Proxy'yi test edin">
    Sol menüden bir endpoint seçin ve **Test Endpoint** butonuna tıklayın. Parametreleri ayarlayıp **Send** butonuna basarak test edin.
  </Step>
</Steps>

**Tebrikler! İlk API Proxy'nizi başarıyla oluşturdunuz ve test ettiniz.**
