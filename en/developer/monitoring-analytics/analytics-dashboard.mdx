Analytics Dashboard is an analysis tool that allows you to visualize the performance and usage of your API Proxies. With this dashboard, you can track API traffic, error rates, response times, and other important metrics.

## Overview

Analytics Dashboard is a tool used to visualize performance metrics of your API Proxies. This dashboard is particularly used for:

* Monitoring API traffic
* Analyzing performance metrics
* Tracking error rates
* Viewing usage statistics

## Dashboard Widgets

The dashboard consists of various widgets. Each widget displays a specific metric.

### Widget Types

#### Line Chart

Line chart showing change over time:

* **Request Count**: Request count over time
* **Response Time**: Response time over time
* **Error Rate**: Error rate over time

#### Bar Chart

Bar chart comparing by categories:

* **Requests by Method**: Request count by method
* **Requests by Status**: Request count by status code
* **Top Endpoints**: Most used endpoints

#### Pie Chart

Pie chart showing distribution:

* **Status Code Distribution**: Status code distribution
* **Method Distribution**: Method distribution
* **Error Type Distribution**: Error type distribution

#### Metric Card

Card displaying a single metric:

* **Total Requests**: Total request count
* **Average Response Time**: Average response time
* **Error Rate**: Error rate
* **Success Rate**: Success rate

#### Table

Detailed data table:

* **Top Endpoints**: Most used endpoints
* **Error Details**: Error details
* **Client Details**: Client details

### Widget Configuration

To configure widgets:

1. Click on the widget
2. Click the **Configure** button
3. Select metric type
4. Set time range
5. Apply filters
6. Click the **Save** button

### Widget Editing

To edit the dashboard:

* **Add Widget**: Add new widget
* **Remove Widget**: Remove widget
* **Resize Widget**: Resize widget
* **Move Widget**: Move widget
* **Configure Widget**: Configure widget

## Metric Types

Various metric types can be displayed on the dashboard.

### Traffic Metrics

#### Request Count

Total request count:

* **Total Requests**: Total request count
* **Requests per Second**: Requests per second
* **Requests per Minute**: Requests per minute
* **Requests per Hour**: Requests per hour

#### Request Volume

Request volume:

* **Request Size**: Request size
* **Response Size**: Response size
* **Total Data Transfer**: Total data transfer

### Performance Metrics

#### Response Time

Response time metrics:

* **Average Response Time**: Average response time
* **Median Response Time**: Median response time
* **P95 Response Time**: 95th percentile response time
* **P99 Response Time**: 99th percentile response time
* **Min Response Time**: Minimum response time
* **Max Response Time**: Maximum response time

#### Backend Performance

Backend performance metrics:

* **Backend Response Time**: Backend response time
* **Backend Connection Time**: Backend connection time
* **Backend Error Rate**: Backend error rate

### Error Metrics

#### Error Rate

Error rate metrics:

* **Total Errors**: Total error count
* **Error Rate**: Error rate (%)
* **4xx Errors**: Client errors
* **5xx Errors**: Server errors

#### Error Details

Error details:

* **Error Types**: Error types
* **Error Messages**: Error messages
* **Error Endpoints**: Endpoints where errors occurred

### Usage Metrics

#### Endpoint Usage

Endpoint usage metrics:

* **Top Endpoints**: Most used endpoints
* **Endpoint Request Count**: Endpoint request count
* **Endpoint Success Rate**: Endpoint success rate

#### Client Usage

Client usage metrics:

* **Top Clients**: Clients sending most requests
* **Client Request Count**: Client request count
* **Client Error Rate**: Client error rate

## Time Range Selection

You can select different time ranges on the dashboard.

### Predefined Ranges

* **Last 15 Minutes**: Last 15 minutes
* **Last Hour**: Last 1 hour
* **Last 24 Hours**: Last 24 hours
* **Last 7 Days**: Last 7 days
* **Last 30 Days**: Last 30 days
* **Last 90 Days**: Last 90 days

### Custom Time Range

To select a custom time range:

1. Select **Custom Range** option
2. Select start date
3. Select end date
4. Click the **Apply** button

### Time Granularity

Time granularity options:

* **1 Minute**: 1 minute
* **5 Minutes**: 5 minutes
* **15 Minutes**: 15 minutes
* **1 Hour**: 1 hour
* **1 Day**: 1 day

### Time Comparison

Comparing two time ranges:

* **Comparison**: Comparison results

### Trend Analysis

Trend analysis:

* **Increasing**: Increasing trend
* **Decreasing**: Decreasing trend

* **Fluctuating**: Fluctuating trend
