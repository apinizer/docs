API Traffic Tab is a monitoring tool that allows you to monitor requests arriving at your API Proxy in real-time. With this tab, you can view live requests, inspect request/response details, and perform performance analysis.

## Overview

API Traffic Tab allows you to monitor requests arriving at your API Proxy in real-time. This tab is particularly used for:

* Real-time traffic monitoring
* Inspecting request/response details
* Performance analysis
* Error detection

## Using API Traffic Tab

### Accessing API Traffic Tab

To access API Traffic Tab:

1. Go to the API Proxy detail page
2. Click on the **API Traffic** tab
3. Traffic is displayed

### Real-time Monitoring

API Traffic Tab displays incoming requests in real-time.

#### Real-time Updates

Real-time updates:

* **Auto Refresh**: Auto refresh (default: 5 seconds)
* **Manual Refresh**: Manual refresh
* **Pause**: Pause monitoring

#### Traffic List

The traffic list contains the following information:

* **Timestamp**: Time of the request
* **Method**: HTTP method
* **Path**: Request path
* **Status**: Response status code
* **Response Time**: Response time
* **Size**: Response size
* **Client IP**: Client IP address

## Settings

API Traffic Tab settings control the traffic display behavior.

### Display Settings

Display settings:

* **Auto Refresh Interval**: Auto refresh interval (seconds)
* **Max Visible Requests**: Maximum number of visible requests
* **Column Visibility**: Column visibility
* **Row Height**: Row height

### Capture Settings

Capture settings:

* **Capture Request Body**: Capture request body
* **Capture Response Body**: Capture response body
* **Capture Headers**: Capture headers
* **Max Body Size**: Maximum body size

### Filter Settings

Filter settings:

* **Default Filters**: Default filters
* **Saved Filters**: Saved filters
* **Filter Presets**: Filter presets
