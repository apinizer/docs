---
title: "ACL Report"
description: "You can create and query access control list reports based on API Proxy or organization/credential. You can view which users can access which API Proxy or which organizations can access which API Proxies."
---

<Info>
In the ACL Report screen, queries can be run in two options: **API Proxy-based** or **Organization/Credential-based**. Queries can be customized with filter fields.
</Info>

<Tabs>
  <Tab title="API Proxy-Based Query">
    In the API Proxy-based query screen, which users have access permission to the selected API Proxy is displayed.

    <img
      src="/images/yonetici/acl1.png"
      alt="API Proxy-Based Query"
      width="1000"
      height="600"
    />
  </Tab>
  <Tab title="Organization/Credential-Based Query">
    In the Organization/Credential-based query screen, which API Proxies the selected organization has access permission to is displayed.

    <img
      src="/images/yonetici/acl2.png"
      alt="Organization/Credential-Based Query"
      width="1000"
      height="600"
    />
  </Tab>
</Tabs>

<Note>
  You can create more specific queries and customize reports by using filter fields.
</Note>
