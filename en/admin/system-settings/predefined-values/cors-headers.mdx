<Info>
Apinizer console defines some headers by default for ease of use:

- Accept
- Accept-Language
- Content-Language
- Content-Type
</Info>

## Creating a New CORS Header Value

Click the **+New** button from the **Management → System Settings → Predefined Values → CORS Headers** screen.

**An image containing CORS Header Value creation settings is shown below:**

  src="/images/admin/system-settings/on-tanimli-degerler/cors-baslik-degeri-olusturma.png"
  alt="Creating CORS Header Value"
  width="1000"

/>

The fields used for CORS Header Value creation configuration are shown in the table below.

| Field | Description |
| --- | --- |
| Value *(Value)* | Field where the CORS Header Value is entered. |
