## Filtering Options

The fields used for advanced filtering configuration are shown in the table below.

| Field | Description |
|------|-------------|
| **Environment** _(Environment)_ | Token Records containing the value selected from this filter field come. |
| **Project** _(Project)_ | Token Records containing the value(s) selected from this filter field come. |
| **Date Range** _(Date Range)_ | Filters Token Records according to the selected date range. |
| **Proxy** _(API Proxy)_ | Token Records containing the value selected from this filter field come. |
| **Username or Key** _(Username or Key)_ | Token Records containing the value entered in this filter field come. |
| **To Client Body** _(To Client Body)_ | Token Records containing the value entered in this filter field within the response body returned to the client come. |
| **Status Code Bigger Than** _(Status Code Bigger Than)_ | Token Records containing values greater than the value entered in this filter field come. |
| **Status Code Smaller Than** _(Status Code Smaller Than)_ | Token Records containing values smaller than the value entered in this filter field come. |
