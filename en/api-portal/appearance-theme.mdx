## Saving and Preview

### Saving Process

**How to Save:**

- Click the "Save" button in the top right corner
- All appearance settings are saved
- After successful save, changes are reflected on the portal instantly.

### Basic Fields

| Field | Description | Where It Appears on Portal |
|------|----------|-------------------------|
| Primary Logo | Main logo used in the portal top bar | navbar and index.html |
| Logo Right | Right logo used in the portal top bar | navbar |
| Favicon | Favicon appearing in the browser tab | index.html |
| Primary Color | Theme primary color (e.g., plan cards, buttons) | Entire portal |
| Menu Link Color | Color of menu links | navbar |
| Component Link Color | Color of links outside the menu | Entire portal |
| Text Font | Font used throughout the portal | Entire portal |
| Login Page Title | Login page title | Login page |
| Register Page Title | Register page title | Register page |
| Contact Us | Contact URL | Portal top bar |
| API Product Home Page Header | API Product homepage header | API Product list page |

## Frequently Encountered Issues

<Accordion title="Why Are Changes Not Visible on Portal?">
- Refresh the page (Ctrl+F5 or Cmd+Shift+R)
- Clear browser cache
- Make sure you clicked the Save button
- Try in a different browser
</Accordion>

<Accordion title="Why is Favicon not updating?">
- Completely clear browser cache
- Close and reopen the browser
- Check favicon format and size
</Accordion>

<Accordion title="Why is HTML Preview Not Working?">
- There may be a syntax error in the HTML code
- Check browser console errors (F12)
- Check for unclosed tags
</Accordion>

<Accordion title="Why are Colors Different Than Expected?">
- Check hex code format (#RRGGBB)
- Clear browser cache
- There may be CSS specificity issues
</Accordion>

<Accordion title="Why are Multilingual Contents Not Visible?">
- Make sure both languages are active in Portal Settings
- Make sure you entered content for each language
- Make sure the user selected the correct language
</Accordion>

<Accordion title="Where can I edit homepage, How It Works, and Footer contents?">
- Management of these contents is now done from the "Portal Content Management" section on the [API Portal Settings](/en/api-portal/api-portal-settings#portal-content-management) page
- The Appearance and Theme page only contains basic appearance settings such as visual identity, colors, fonts, and page titles
</Accordion>
