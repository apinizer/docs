## Form Fields

| Field | Description / Operation |
|------|------------------|
| **Name**  | Name of the support package type. Should be short and descriptive. |
| **Description**  | - Detailed description of the support package type<br/>- Explains what this type provides to users<br/><br/>Example: "**Developer Tools Support Package** provides basic tools and resources to entrepreneurs and developers." |

<Info>
Categories defined on the Support Package Type page (Developer Tools, Enterprise Integration, Startup, etc.) are presented as a selection list in the "Support Packages" section when creating a new support package. You can categorize your packages by creating multiple types according to your needs. You can review the [Support Packages](/en/api-portal/support-packages) document for the usage of the Support Packages page.
