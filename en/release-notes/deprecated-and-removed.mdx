## Version 2025.11.0

<span style={{ display: 'inline-block', backgroundColor: '#e5e7eb', color: '#4b5563', padding: '0.25rem 0.75rem', borderRadius: '9999px', fontSize: '0.875rem', fontWeight: '500' }}>REMOVED FEATURES</span>

**WARNING!** In Management APIs, the relativePath field has been replaced with the clientRoute field in Create/Update API Proxy APIs. Users utilizing proxy management APIs in CI/CD processes should pay attention to this field. [See](https://apinizer.github.io/management-api-docs/02-api-reference/04-api-proxies/crud/create-api-proxy-from-url/)

## Version 2025.04.0

<span style={{ display: 'inline-block', backgroundColor: '#e5e7eb', color: '#4b5563', padding: '0.25rem 0.75rem', borderRadius: '9999px', fontSize: '0.875rem', fontWeight: '500' }}>REMOVED FEATURES</span>

**WARNING!** The feature to update applications through the application has been removed.

**WARNING!** The integration with Opsgenie product has been removed due to its termination by Jira.

## Version 2025.01.4

<span style={{ display: 'inline-block', backgroundColor: '#e5e7eb', color: '#4b5563', padding: '0.25rem 0.75rem', borderRadius: '9999px', fontSize: '0.875rem', fontWeight: '500' }}>FEATURES TO BE REMOVED</span>

* The feature to update applications through the application will be removed in the next release version.
* Opsgenie integration will be removed in the next release version.
* The Client Banner feature will be removed from the Routing section in the next release version. This feature will now be managed directly through the policy.

## Version 2024.09.0

<span style={{ display: 'inline-block', backgroundColor: '#e5e7eb', color: '#4b5563', padding: '0.25rem 0.75rem', borderRadius: '9999px', fontSize: '0.875rem', fontWeight: '500' }}>FEATURES TO BE REMOVED</span>

**WARNING!** Due to the requirement for data owners to control the deletion of log records, the automatic deletion of log records with retention period in Monitor Screens, Application Logs, API Integration, and Reports pages has been removed. Purge scripts have been prepared for manual deletion of these records from the database. [See](/en/operations/overview#database-growth-management)

**IMPORTANT!** Snapshot policy screens used for automatic backup of Elasticsearch indices through Apinizer screens have been removed due to snapshot policy usage requirements. Necessary scripts have been prepared for manual execution of these operations. [See](/en/setup/elasticsearch/backup-policy)

**WARNING!** All features related to the old revision logic have been removed due to usage difficulties. Now, completely switched to the "Deployment History" application started in Version 2023.06.1. Revision parameters in Apinizer Management APIs have also been removed. [See](/en/develop/deployment-and-version-management)

## Version 2024.05.3

<span style={{ display: 'inline-block', backgroundColor: '#e5e7eb', color: '#4b5563', padding: '0.25rem 0.75rem', borderRadius: '9999px', fontSize: '0.875rem', fontWeight: '500' }}>REMOVED FEATURES</span>

**WARNING** Multiple API Proxy selection has been enabled in API Proxy Group. With this setting, the feature of applying policies in API Proxy Group to API Proxies has been removed as it existed before. [See](/en/develop/routing/http-routing)

From now on, policies in API Proxy Group will only be applied to requests coming to API Proxy Group, if a request comes directly to API Proxy and this API Proxy is included in API Proxy Group, policies in API Proxy Group will not be executed in this case.

<span style={{ display: 'inline-block', backgroundColor: '#e5e7eb', color: '#4b5563', padding: '0.25rem 0.75rem', borderRadius: '9999px', fontSize: '0.875rem', fontWeight: '500' }}>REMOVED FEATURES</span>

**WARNING!** Simple method has been removed from LDAP settings in general settings. [See](/en/admin/system-settings/ldap-login-settings)

## Version 2024.01.1

<span style={{ display: 'inline-block', backgroundColor: '#e5e7eb', color: '#4b5563', padding: '0.25rem 0.75rem', borderRadius: '9999px', fontSize: '0.875rem', fontWeight: '500' }}>REMOVED FEATURES</span>

**WARNING** The feature of subjecting the value found with variable to URL decode operation at the time it is found has been removed. If this need exists, URL decode should be done with business rule.

## Version 2023.03.1

<span style={{ display: 'inline-block', backgroundColor: '#e5e7eb', color: '#4b5563', padding: '0.25rem 0.75rem', borderRadius: '9999px', fontSize: '0.875rem', fontWeight: '500' }}>REMOVED FEATURES</span>

* The option to Load from Spec for creating API Proxy in Swagger and Open API types has been removed.
* JWT 3rd Party policy has been removed because JOSE Validation policy included JWT 3rd Party policy.
* Due to the arrangement made with APNZ-3472, the feature to configure log settings on API Proxy Group basis has been removed.
* Due to the arrangement made with APNZ-3472, the Project Log Settings screen has been removed.
* Due to the arrangement made with APNZ-3472, the feature to set the severity level of log settings on a general settings basis has been removed.
