---
title: "Application Logs Tab"
description: "You can view software log records that occur during the execution of the flow for messages coming to and returning from the API Proxy. You can filter by log level, correlation ID, detail and stack trace, download logs, and delete them."
---

In this tab, records of software logs that occur during the execution of the flow for messages coming to and returning from the relevant API Proxy are displayed.

<Info>
It is appropriate to use this page when a software error is suspected or when it is desired to investigate how the application behaves in the background.
</Info>

<img src="/images/develop/izleme-analiz/application-logs-tab.png" alt="Application Logs Tab" style={{maxWidth: '1000px', width: '100%', height: 'auto'}} />

The fields used for operations and filtering on Application Logs are shown below.

| Field | Description |
|------|-------------|
| **Start** *(Start)* | Enables keeping records of application logs on an API Proxy basis |
| **Environment Filter** *(Environment)* | Enables filtering application logs for the specified environment |
| **Time Filter** *(From, To)* | Enables filtering application logs for the specified date range |
| **Log Level** *(Search by Log Levels)* | Enables filtering logs at the level specified by "Log Level" in application logs |
| **Search in Correlation ID** *(Search in Correlation ID)* | When a request comes to the API Proxy, Apinizer assigns a correlation id to this request so that the request can be tracked. By performing a detailed search in application logs with the correlation id to be found from API Proxy Traffic logs, only the application logs of a single request can be accessed |
| **Search in Detail** *(Search in Detail)* | Enables searching in text fields of application logs |
| **Search in Stack Trace** *(Search in Stack Trace)* | Enables searching on exception text if an exception has occurred in application logs |
| **Download Button** *(Download)* | Enables downloading application logs as a single file.<br/>If "All" option is selected, all logs related to this API Proxy are downloaded.<br/>If "Search Results" option is selected, all filtered logs are downloaded |
| **Delete Button** *(Delete)* | Enables deletion of application log records belonging to this API Proxy |
