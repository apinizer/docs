---
title: "Converting pfx File to jks File"
description: "Converting PFX certificates to JKS format using Keystore Explorer"
---

When it is desired to open Apinizer Worker applications with HTTPS, it supports pfx and jks certificates. However, organizations may still want to upload pfx certificates in jks format.

This document explains how this conversion can be done with the help of the "Keystore Explorer" program to help with this situation.

## Converting Certificate Files to jks Format with Keystore Explorer (KSE)

### Step 1: Opening PFX File

The pfx file is opened with KSE.

Right-click on the file inside and select **export key pair**.

  <img
    src="/images/tutorials/nasil-yapilir-kilavuzlari/yonetici-kilavuzlari/pfx-jks-donusturme/01-pfx-open-kse.png"
    alt="Opening PFX File with KSE"
    width="600"
  />

### Step 2: Export in PKCS12 Format

The keypair is exported in **pkcs12** format with **.p12** extension.

  <img
    src="/images/tutorials/nasil-yapilir-kilavuzlari/yonetici-kilavuzlari/pfx-jks-donusturme/02-export-keypair.jpg"
    alt="Keypair Export"
    width="600"
  />

### Step 3: Creating New JKS File

**JKS** type is selected with the new file creation option.

  <img
    src="/images/tutorials/nasil-yapilir-kilavuzlari/yonetici-kilavuzlari/pfx-jks-donusturme/03-new-jks-file.png"
    alt="JKS File Creation"
    width="200"
  />

### Step 4: Importing Keypair

The .p12 file exported just now is imported into the created file with the import keypair option:

  <img
    src="/images/tutorials/nasil-yapilir-kilavuzlari/yonetici-kilavuzlari/pfx-jks-donusturme/04-import-keypair.jpg"
    alt="Keypair Import"
    width="600"
  />

  <img
    src="/images/tutorials/nasil-yapilir-kilavuzlari/yonetici-kilavuzlari/pfx-jks-donusturme/05-import-keypair-2.png"
    alt="Import Keypair Screen"
    width="200"
  />

  <img
    src="/images/tutorials/nasil-yapilir-kilavuzlari/yonetici-kilavuzlari/pfx-jks-donusturme/06-import-keypair-3.jpg"
    alt="Import Keypair Details"
    width="600"
    height="142"
  />

**Note:** An optional alias can be entered. The point to note is that if there is another certificate with the same alias, it **should not be overwritten**.

  <img
    src="/images/tutorials/nasil-yapilir-kilavuzlari/yonetici-kilavuzlari/pfx-jks-donusturme/07-alias-input.png"
    alt="Alias Input"
    width="200"
  />

### Step 5: Setting Password

A new password is given, the point to note is that the given password and jks password must be the same, if there is no password, jks should also have no password.

  <img
    src="/images/tutorials/nasil-yapilir-kilavuzlari/yonetici-kilavuzlari/pfx-jks-donusturme/08-password-input.png"
    alt="Password Setting"
    width="400"
  />

### Step 6: Certificate Chain Export

Then right-click on the imported keypair and select export certificate chain option:

  <img
    src="/images/tutorials/nasil-yapilir-kilavuzlari/yonetici-kilavuzlari/pfx-jks-donusturme/09-export-certificate-chain.jpg"
    alt="Certificate Chain Export"
    width="600"
  />

When exporting, the point to note is that **Entire Chain** and **X.509** options must be selected.

  <img
    src="/images/tutorials/nasil-yapilir-kilavuzlari/yonetici-kilavuzlari/pfx-jks-donusturme/10-export-options.jpg"
    alt="Export Options"
    width="400"
  />

### Step 7: Certificate Chain Import

Then this exported certificate is imported into jks. An optional alias can be entered. The point to note is that if there is a certificate with the same alias, it **should not be overwritten**.

  <img
    src="/images/tutorials/nasil-yapilir-kilavuzlari/yonetici-kilavuzlari/pfx-jks-donusturme/11-import-certificate.jpg"
    alt="Certificate Chain Import"
    width="600"
  />

### Step 8: Final Status Check

What should be in the final state is:

  <img
    src="/images/tutorials/nasil-yapilir-kilavuzlari/yonetici-kilavuzlari/pfx-jks-donusturme/12-final-state.png"
    alt="JKS File Final State"
    width="600"
  />

### Step 9: Saving File

The file is saved with the password given to the keypair and closed.

  <img
    src="/images/tutorials/nasil-yapilir-kilavuzlari/yonetici-kilavuzlari/pfx-jks-donusturme/13-save-file.jpg"
