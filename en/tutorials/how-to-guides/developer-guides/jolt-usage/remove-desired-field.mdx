---
title: Removing Desired Field
description: "You can remove specific fields within JSON data. This page includes an example of cleaning unnecessary fields using remove operation."
---

This example shows how to delete the key value named "columnNames" within JSON data.

### Input Value

```json
{
  "statusCode": 200,
  "statusDescription": "OK",
  "columnNames": [
    "Id",
    "UstBirimId",
    "Adi",
    "TeskilatId",
    "Yol",
    "SehirId",
    "BirimGrupId",
    "no"
  ],
  "numRows": 1201
}
```

### JOLT Specification

```json
[
  {
    "operation": "remove",
    "spec": {
      "columnNames": ""
    }
  }
]
```

### Output Value

```json
{
  "statusCode": 200,
  "statusDescription": "OK",
  "numRows": 1201
}
```
