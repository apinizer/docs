FTP/SFTP Configuration allows you to create connections to FTP and SFTP servers for file transfer operations.

## Overview

FTP/SFTP Connections provide connectivity to file servers for uploading and downloading files.

## Configuration

### FTP Configuration

**FTP Properties:**

* **Host**: FTP server hostname
* **Port**: FTP port (default: 21)
* **Username**: FTP username
* **Password**: FTP password
* **Protocol**: FTP or FTPS

**Example:**

```json
{
  "name": "FTP Production",
  "host": "ftp.example.com",
  "port": 21,
  "username": "ftp_user",
  "password": "secure_password",
  "protocol": "ftp"
}
```

### SFTP Configuration

**SFTP Properties:**

* **Host**: SFTP server hostname
* **Port**: SFTP port (default: 22)
* **Username**: SFTP username
* **Password**: SFTP password or SSH key
* **Authentication**: Password or SSH key

**Example:**

```json
{
  "name": "SFTP Production",
  "host": "sftp.example.com",
  "port": 22,
  "username": "sftp_user",
  "password": "secure_password",
  "protocol": "sftp"
}
```
