SNMP Connections allow you to configure connections to SNMP-enabled devices for monitoring and management.

## Overview

SNMP Connections provide connectivity to SNMP agents for sending traps and performing SNMP operations.

## Configuration

### SNMP Properties

**Connection Properties:**

* **Host**: SNMP agent hostname or IP
* **Port**: SNMP port (default: 162 for traps)
* **Community**: SNMP community string
* **Version**: SNMP version (v1, v2c, v3)

**Example:**

```json
{
  "name": "SNMP Production",
  "host": "device.example.com",
  "port": 162,
  "community": "public",
  "version": "v2c"
}
```
