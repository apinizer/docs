## Configuration

### Basic Configuration

**Connection Properties:**

* **Connection Name**: Unique name for the connection
* **Description**: Connection description
* **Database Type**: Database vendor (MySQL, PostgreSQL, Oracle, etc.)
* **Host**: Database server hostname or IP address
* **Port**: Database server port
* **Database Name**: Name of the database
* **Username**: Database username
* **Password**: Database password

### MySQL Configuration

**Example:**

```json
{
  "name": "MySQL Production",
  "type": "mysql",
  "host": "db.example.com",
  "port": 3306,
  "database": "production",
  "username": "app_user",
  "password": "secure_password"
}
```

### PostgreSQL Configuration

**Example:**

```json
{
  "name": "PostgreSQL Production",
  "type": "postgresql",
  "host": "db.example.com",
  "port": 5432,
  "database": "production",
  "username": "app_user",
  "password": "secure_password"
}
```

### Oracle Configuration

**Example:**

```json
{
  "name": "Oracle Production",
  "type": "oracle",
  "host": "db.example.com",
  "port": 1521,
  "database": "ORCL",
  "username": "app_user",
  "password": "secure_password"
}
```

## Connection Pooling

### Pool Configuration

Configure connection pool:

**Pool Settings:**

* **Min Pool Size**: Minimum number of connections (default: 5)
* **Max Pool Size**: Maximum number of connections (default: 20)
* **Connection Timeout**: Connection acquisition timeout (ms)
* **Idle Timeout**: Idle connection timeout (ms)
* **Validation Query**: Query to validate connections

**Example:**

```json
{
  "pool": {
    "minSize": 5,
    "maxSize": 20,
    "connectionTimeout": 30000,
    "idleTimeout": 600000,
    "validationQuery": "SELECT 1"
  }
}
```

## SSL/TLS Configuration

### Enable SSL

Configure SSL/TLS for secure connections:

**SSL Settings:**

* **Use SSL**: Enable SSL/TLS
* **SSL Mode**: SSL mode (require, verify-ca, verify-full)
* **Trust Store**: SSL trust store path
* **Key Store**: SSL key store path

## Connection Testing

### Test Connection

Test database connection:

1. Click **Test Connection** button
2. System attempts to connect
3. View test results
4. Fix any connection issues
