Syslog Connections allow you to configure connections to syslog servers for centralized logging.

## Overview

Syslog Connections provide connectivity to syslog servers following RFC 3164 or RFC 5424 standards.

## Configuration

### Syslog Properties

**Connection Properties:**

* **Host**: Syslog server hostname or IP
* **Port**: Syslog port (default: 514)
* **Protocol**: UDP or TCP
* **Format**: RFC 3164 or RFC 5424

**Example:**

```json
{
  "name": "Syslog Production",
  "host": "syslog.example.com",
  "port": 514,
  "protocol": "UDP",
  "format": "RFC5424"
}
```
