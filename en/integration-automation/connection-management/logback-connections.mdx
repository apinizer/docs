## Configuration

### Logback Properties

**Connection Properties:**

* **Appender**: Logback appender name
* **Logger**: Logger name
* **Level**: Default log level

**Example:**

```json
{
  "name": "Logback Production",
  "appender": "fileAppender",
  "logger": "apinizer",
  "level": "INFO"
}
```
