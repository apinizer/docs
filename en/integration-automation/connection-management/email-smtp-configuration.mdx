## Overview

SMTP Connections provide connectivity to SMTP servers for sending emails.

## Configuration

### SMTP Properties

**Connection Properties:**

* **SMTP Host**: SMTP server hostname
* **SMTP Port**: SMTP port (25, 465, 587)
* **Username**: SMTP username
* **Password**: SMTP password
* **Security**: SSL/TLS configuration

### Example Configuration

**Standard SMTP:**

```json
{
  "name": "SMTP Production",
  "host": "smtp.example.com",
  "port": 25,
  "username": "noreply@example.com",
  "password": "secure_password"
}
```

**SMTP with SSL:**

```json
{
  "name": "SMTP SSL",
  "host": "smtp.example.com",
  "port": 465,
  "username": "noreply@example.com",
  "password": "secure_password",
  "ssl": true
}
```

**SMTP with TLS:**

```json
{
  "name": "SMTP TLS",
  "host": "smtp.example.com",
  "port": 587,
  "username": "noreply@example.com",
  "password": "secure_password",
  "tls": true
}
```
