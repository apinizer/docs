## Overview

Linux Connections provide SSH connectivity to Linux systems for remote command execution.

## Configuration

### SSH Properties

**Connection Properties:**

* **Host**: Linux server hostname or IP
* **Port**: SSH port (default: 22)
* **Username**: SSH username
* **Authentication**: Password or SSH key
* **Working Directory**: Default working directory

**Example:**

```json
{
  "name": "Linux Production",
  "host": "server.example.com",
  "port": 22,
  "username": "app_user",
  "authentication": "key",
  "keyPath": "/path/to/ssh/key"
}
```
