## Configuration

### Logback Connection

Configure Logback connection:

**Connection Properties:**

* **Appender**: Logback appender name
* **Logger**: Logger name
* **Level**: Log level (DEBUG, INFO, WARN, ERROR)

### Send Log Message

Send log message:

**Configuration:**

```json
{
  "level": "INFO",
  "message": "${logMessage}",
  "logger": "${loggerName}",
  "properties": {
    "orderId": "${orderId}",
    "customerId": "${customerId}"
  }
}
```
