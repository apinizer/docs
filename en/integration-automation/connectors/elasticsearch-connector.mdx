## Overview

Elasticsearch Connector provides connectivity to Elasticsearch, allowing you to perform search queries, index documents, and manage Elasticsearch data.

## Configuration

### Connection Selection

Select or create an Elasticsearch connection:

**Connection Properties:**

* **Hosts**: Elasticsearch cluster hosts
* **Port**: Elasticsearch port (default: 9200)
* **Authentication**: Username/password or API key
* **SSL/TLS**: SSL/TLS configuration

## Operations

### Search

Execute search queries:

**Query Configuration:**

```json
{
  "operation": "search",
  "index": "orders",
  "query": {
    "match": {
      "status": "pending"
    }
  }
}
```

### Index Document

Index documents:

**Index Configuration:**

```json
{
  "operation": "index",
  "index": "orders",
  "id": "${orderId}",
  "document": {
    "orderId": "${orderId}",
    "customerId": "${customerId}",
    "total": "${orderTotal}",
    "status": "pending"
  }
}
```

### Update Document

Update documents:

**Update Configuration:**

```json
{
  "operation": "update",
  "index": "orders",
  "id": "${orderId}",
  "document": {
    "status": "completed"
  }
}
```
