Syslog Connector enables sending syslog messages to syslog servers for centralized logging and monitoring.

## Overview

Syslog Connector provides capabilities to send syslog messages following RFC 3164 or RFC 5424 standards.

## Configuration

### Syslog Connection

Configure syslog connection:

**Connection Properties:**

* **Host**: Syslog server hostname or IP
* **Port**: Syslog port (default: 514)
* **Protocol**: UDP or TCP
* **Format**: RFC 3164 or RFC 5424

### Send Syslog Message

Send syslog message:

**Configuration:**

```json
{
  "facility": "local0",
  "severity": "info",
  "message": "${logMessage}",
  "hostname": "${hostname}",
  "appName": "TaskFlow"
}
```
