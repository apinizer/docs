Script Connector allows you to execute custom JavaScript or Groovy scripts within Task Flows. It enables data transformations, custom business logic, and complex calculations.

## Overview

Script Connector provides a scripting environment for executing custom code, allowing you to extend Task Flow capabilities beyond standard connectors.

## Configuration

### Script Type

Select script language:

* **JavaScript**: JavaScript/ECMAScript
* **Groovy**: Groovy scripting language

### Script Content

Write or paste script code:

**JavaScript Example:**

```javascript
// Access input variables
const customerId = variables.customerId;
const orderTotal = variables.orderTotal;

// Perform calculations
const discount = orderTotal > 1000 ? orderTotal * 0.1 : 0;
const finalTotal = orderTotal - discount;

// Set output variables
variables.discountedTotal = finalTotal;
variables.discountAmount = discount;
variables.isPremium = orderTotal > 1000;
```

**Groovy Example:**

```groovy
// Access input variables
def customerId = variables.customerId
def orderTotal = variables.orderTotal

// Perform calculations
def discount = orderTotal > 1000 ? orderTotal * 0.1 : 0
def finalTotal = orderTotal - discount

// Set output variables
variables.discountedTotal = finalTotal
variables.discountAmount = discount
variables.isPremium = orderTotal > 1000
```

## Variable Access

### Reading Variables

Access Task Flow variables:

**JavaScript:**

```javascript
const value = variables.variableName;
const nested = variables.customer.address.city;
```

**Groovy:**

```groovy
def value = variables.variableName
def nested = variables.customer.address.city
```

### Writing Variables

Set Task Flow variables:

**JavaScript:**

```javascript
variables.newVariable = "value";
variables.customer.status = "active";
```

**Groovy:**

```groovy
variables.newVariable = "value"
variables.customer.status = "active"
```

## Data Transformation

### JSON Transformation

Transform JSON data:

**JavaScript:**

```javascript
// Transform object
const transformed = {
  id: customer.id,
  fullName: `${customer.firstName} ${customer.lastName}`,
  email: customer.email.toLowerCase()
};

variables.transformedCustomer = transformed;
```

### Array Operations

Process arrays:

**JavaScript:**

```javascript
// Map array
const emails = customers.map(c => c.email);

// Filter array
const activeCustomers = customers.filter(c => c.status === 'active');

// Reduce array
const total = orders.reduce((sum, order) => sum + order.total, 0);
```

### String Operations

Manipulate strings:

**JavaScript:**

```javascript
const fullName = `${firstName} ${lastName}`;
const upperCase = name.toUpperCase();
const trimmed = value.trim();
```

## Common Patterns

### Data Validation

Validate data:

**JavaScript:**

```javascript
function validateEmail(email) {
  return /^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email);
}

if (!validateEmail(variables.email)) {
  throw new Error("Invalid email address");
}
```

### Conditional Logic

Implement conditional logic:

**JavaScript:**

```javascript
if (variables.orderTotal > 1000) {
  variables.discount = variables.orderTotal * 0.1;
  variables.customerType = "premium";
} else {
  variables.discount = 0;
  variables.customerType = "standard";
}
```

### Loops

Iterate over collections:

**JavaScript:**

```javascript
const results = [];
for (const item of variables.items) {
  const processed = processItem(item);
  results.push(processed);
}
variables.processedItems = results;
```

## Error Handling

### Throwing Errors

Throw errors to stop execution:

**JavaScript:**

```javascript
if (!variables.requiredField) {
  throw new Error("Required field is missing");
}
```

### Try-Catch

Handle errors:

**JavaScript:**

```javascript
try {
  const result = riskyOperation();
  variables.result = result;
} catch (error) {
  variables.error = error.message;
  variables.result = null;
}
```
