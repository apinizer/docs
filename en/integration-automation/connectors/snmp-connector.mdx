## Overview

SNMP Connector provides capabilities to send SNMP traps and perform SNMP operations.

## Configuration

### SNMP Connection

Configure SNMP connection:

**Connection Properties:**

* **Host**: SNMP agent hostname or IP
* **Port**: SNMP port (default: 162 for traps)
* **Community**: SNMP community string
* **Version**: SNMP version (v1, v2c, v3)

### Send SNMP Trap

Send SNMP trap:

**Configuration:**

```json
{
  "operation": "trap",
  "host": "${snmpHost}",
  "oid": "1.3.6.1.4.1.12345.1.1",
  "value": "${trapValue}",
  "community": "${community}"
}
```
