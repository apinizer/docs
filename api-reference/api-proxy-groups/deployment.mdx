## Endpoints

- [Deploy API Proxy Group](/api-reference/api-proxy-groups/deployment/deploy-api-proxy-group) - Deploy an API proxy group to one or more environments
- [Undeploy API Proxy Group](/api-reference/api-proxy-groups/deployment/undeploy-api-proxy-group) - Undeploy an API proxy group from one or more environments
- [List Environments](/api-reference/api-proxy-groups/deployment/list-environments) - List environments for API proxy group deployment
- [List Proxy Group Deploy History](/api-reference/api-proxy-groups/deployment/list-deploy-history) - List deploy history records for a group
- [Get Deploy History Detail](/api-reference/api-proxy-groups/deployment/get-deploy-history-detail) - Get a single history record by revision
- [Delete Deploy History](/api-reference/api-proxy-groups/deployment/delete-deploy-history) - Delete a history record by revision
