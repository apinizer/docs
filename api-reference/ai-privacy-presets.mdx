## Endpoints

- [List AI Privacy Presets](/api-reference/ai-privacy-presets/list-ai-privacy-presets) - List built-in and custom privacy presets
- [Get AI Privacy Preset](/api-reference/ai-privacy-presets/get-ai-privacy-preset) - Get a single privacy preset by name
- [Create AI Privacy Preset](/api-reference/ai-privacy-presets/create-ai-privacy-preset) - Create or update a custom privacy preset (upsert)
- [Update AI Privacy Preset](/api-reference/ai-privacy-presets/update-ai-privacy-preset) - Update an existing custom privacy preset
- [Delete AI Privacy Preset](/api-reference/ai-privacy-presets/delete-ai-privacy-preset) - Delete a custom privacy preset

## Authentication

All endpoints require authentication using a Personal API Access Token.

## Permissions

- **Admin Only** - All endpoints require admin privileges (sysAdmin user or a user with the `ADMIN` role). Presets are a platform-level catalog; a project-scoped `AI_DEVELOPMENT` token is **not** sufficient.
