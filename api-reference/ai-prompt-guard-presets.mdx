## Endpoints

- [List AI Prompt-Guard Presets](/api-reference/ai-prompt-guard-presets/list-ai-prompt-guard-presets) - List built-in and custom prompt-guard presets
- [Get AI Prompt-Guard Preset](/api-reference/ai-prompt-guard-presets/get-ai-prompt-guard-preset) - Get a single prompt-guard preset by name
- [Create AI Prompt-Guard Preset](/api-reference/ai-prompt-guard-presets/create-ai-prompt-guard-preset) - Create or update a custom prompt-guard preset (upsert)
- [Update AI Prompt-Guard Preset](/api-reference/ai-prompt-guard-presets/update-ai-prompt-guard-preset) - Update an existing custom prompt-guard preset
- [Delete AI Prompt-Guard Preset](/api-reference/ai-prompt-guard-presets/delete-ai-prompt-guard-preset) - Delete a custom prompt-guard preset

## Authentication

All endpoints require authentication using a Personal API Access Token.

## Permissions

- **Admin Only** - All endpoints require admin privileges (sysAdmin user or a user with the `ADMIN` role). Presets are a platform-level catalog; a project-scoped `AI_DEVELOPMENT` token is **not** sufficient.
