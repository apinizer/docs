---
title: "List MCP Connections"
description: "Retrieves all AI Gateway MCP outbound connections defined in the specified project. Secret fields (bearerToken, clientSecret) are returned masked as boolean presence flags."
---

## Endpoint

```
GET /apiops/projects/{projectName}/mcp-connections/
```

## Authentication

Requires a Personal API Access Token.

### Header

```
Authorization: Bearer YOUR_TOKEN
```

## Request

### Headers

| Header | Value | Required |
|--------|-------|----------|
| Authorization | Bearer \{token\} | Yes |

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| projectName | string | Yes | Project name |

### Query Parameters

None

#### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| status | string | Response status: `SUCCESS` or `FAILURE` |
| resultList | array | List of MCP connection objects |
| resultCount | integer | Total number of MCP connections returned |

#### MCP Connection Object Fields

| Field | Type | Description |
|-------|------|-------------|
| id | string | Internal connection ID (name is the API reference) |
| projectId | string | ID of the owning project |
| projectName | string | Name of the owning project |
| name | string | Connection name (unique within the project) |
| description | string | Connection description |
| enabled | boolean | Whether the connection is enabled |
| endpoint | string | MCP server base URL (e.g. `https://mcp.example.com/mcp`) |
| transport | string | Transport type: `HTTP_SSE` or `STDIO` |
| mcpProtocolVersion | string | MCP protocol version (adapter lookup key), default `2024-11-05` |
| allowPrivateNetworkEgress | boolean | Whether cluster-internal / private-network (RFC1918) egress is permitted |
| authScheme | string | Auth scheme: `BEARER`, `OAUTH2_CLIENT_CREDENTIALS`, `MTLS` or `NONE` |
| bearerTokenSet | boolean | `true` if a Bearer token is configured (the token value itself is never returned — INV-06) |
| credentialRef | string | Credential reference for `OAUTH2_CLIENT_CREDENTIALS` scheme |
| tokenEndpoint | string | OAuth2 token endpoint URL (`OAUTH2_CLIENT_CREDENTIALS`) |
| clientId | string | OAuth2 client_id (`OAUTH2_CLIENT_CREDENTIALS`) |
| scope | string | Optional OAuth2 scope (space-delimited) |
| clientSecretSet | boolean | `true` if an OAuth2 client_secret is configured (the secret value itself is never returned — INV-06) |
| sslConfigRef | string | SSL config reference for `MTLS` scheme |
| capabilityCacheTtlSeconds | integer | Capability cache TTL in seconds (`0` = no cache), default `300` |
| capabilityCacheEnabled | boolean | Whether the capability cache is enabled, default `true` |
| failoverList | array[string] | Ordered list of MCP connection IDs used as a failover chain |
| credentialMappingRef | string | Credential mapping reference (reserved for future use) |
| discoveredToolCount | integer | Number of statically discovered tools for this connection |
| toolsParsedAt | integer | Epoch millis of the last successful tools parse (`null` = never parsed) |
| healthStatus | string | Last health probe result: `HEALTHY`, `DOWN`, `UNKNOWN` or `null` (no probe yet) |
| lastHealthCheckAt | string | Timestamp of the last completed health probe |
| healthMessage | string | Short error description when status is `DOWN`; otherwise `null` |

### EnumStatus

- `SUCCESS` - Operation successful
- `FAILURE` - Operation failed

### Error Response (400 Bad Request)

```json
{
  "status": "FAILURE",
  "resultMessage": "Project with name (MyProject) is not found!"
}
```

### Error Response (401 Unauthorized)

```json
{
  "status": "FAILURE",
  "resultMessage": "Token is not valid!"
}
```


## Notes and Warnings

- **Secret Masking (INV-06)**: `bearerToken` and `clientSecret` are never returned; only the `bearerTokenSet` / `clientSecretSet` presence flags are exposed.
- **Empty List**: If the project has no MCP connections, an empty `resultList` is returned with `resultCount` = `0`.
- **`usedByRefs`**: The list of API proxies using a connection is populated only by [Get MCP Connection](/api-reference/mcp-connections/get-mcp-connection), not by this list endpoint.
- **Case Sensitivity**: Project and connection names are case-sensitive in path references.
- **Permission**: Requires `AI_DEVELOPMENT` + `VIEW` in the project.
