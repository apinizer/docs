---
title: "Update MCP Connection"
description: "Updates an existing AI Gateway MCP outbound connection in the specified project. A blank bearerToken or clientSecret preserves the stored secret. On save, @SecretData fields are encrypted and the connection is redeployed to the workers."
---

## Endpoint

```
PUT /apiops/projects/{projectName}/mcp-connections/{connectionName}/
```

## Authentication

Requires a Personal API Access Token.

### Header

```
Authorization: Bearer YOUR_TOKEN
```

## Request

### Headers

| Header | Value | Required |
|--------|-------|----------|
| Authorization | Bearer \{token\} | Yes |
| Content-Type | application/json | Yes |

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| projectName | string | Yes | Project name |
| connectionName | string | Yes | MCP outbound connection name (the connection must already exist) |

### Query Parameters

None

### Request Body

The body is a polymorphic `ConnectionConfig` document. Because MCP is a connection-config subtype,
the discriminator `"_class": "mcp"` is **required**. If `name` is omitted from the body it defaults
to the `connectionName` path variable.

#### Full JSON Body Example

```json
{
  "_class": "mcp",
  "name": "github-mcp",
  "description": "GitHub MCP server (updated)",
  "enabled": true,
  "endpoint": "https://mcp.example.com/mcp",
  "transport": "HTTP_SSE",
  "mcpProtocolVersion": "2024-11-05",
  "allowPrivateNetworkEgress": false,
  "authScheme": "BEARER",
  "bearerToken": "",
  "capabilityCacheTtlSeconds": 600,
  "capabilityCacheEnabled": true
}
```

> In the example above `bearerToken` is sent blank (`""`), so the previously stored, encrypted
> bearer token is preserved unchanged.

#### Request Body Fields

| Field | Type | Required | Default | Description |
|-------|------|----------|---------|-------------|
| _class | string | Yes | - | Connection-config discriminator — must be `"mcp"` |
| name | string | No | path variable | Connection name — if present must equal the `connectionName` path variable; if blank it falls back to the path variable |
| description | string | No | - | Connection description |
| enabled | boolean | No | `true` | Whether the connection is enabled |
| endpoint | string | Yes | - | MCP server base URL (e.g. `https://mcp.example.com/mcp`) |
| transport | string | No | `HTTP_SSE` | Transport type: `HTTP_SSE` or `STDIO` (STDIO is not supported yet) |
| mcpProtocolVersion | string | No | `2024-11-05` | MCP protocol version (adapter lookup key) |
| allowPrivateNetworkEgress | boolean | No | `false` | Permit cluster-internal / private-network (RFC1918) egress. Metadata, loopback, link-local, multicast and CGNAT targets remain blocked regardless |
| authScheme | string | No | `NONE` | Auth scheme: `BEARER`, `OAUTH2_CLIENT_CREDENTIALS`, `MTLS`, or `NONE` |
| bearerToken | string | No | - | Static bearer token for the `BEARER` scheme (`@SecretData`). **Blank preserves the stored secret** |
| credentialRef | string | No | - | Credential reference for the `OAUTH2_CLIENT_CREDENTIALS` scheme |
| tokenEndpoint | string | No | - | OAuth2 token endpoint URL (`OAUTH2_CLIENT_CREDENTIALS`) |
| clientId | string | No | - | OAuth2 client ID (`OAUTH2_CLIENT_CREDENTIALS`) |
| clientSecret | string | No | - | OAuth2 client secret (`@SecretData`). **Blank preserves the stored secret** |
| scope | string | No | - | OAuth2 scope, space-delimited (`OAUTH2_CLIENT_CREDENTIALS`) |
| sslConfigRef | string | No | - | SSL config reference for the `MTLS` scheme (placeholder) |
| capabilityCacheTtlSeconds | integer | No | `300` | Capability cache TTL in seconds (`0` = no cache) |
| capabilityCacheEnabled | boolean | No | `true` | Whether the capability cache is enabled |
| failoverList | array[string] | No | `[]` | Ordered list of MCP connection IDs used as a failover chain |
| discoveredTools | array | No | `[]` | Statically discovered tools. Normally populated by the "Discover / Reparse" flow, not set manually |

### Notes

- The connection identified by `connectionName` **must already exist**, otherwise a `400` is returned.
- If `name` is present in the body it must match the `connectionName` path variable (case-insensitive).
- `_class` must be `"mcp"`; without it the body cannot be deserialized as an MCP connection.

## Response

### Success Response (200 OK)

```json
{
  "status": "SUCCESS",
  "deploymentResult": {
    "success": true
  }
}
```

#### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| status | string | Response status: `SUCCESS` or `FAILURE` |
| deploymentResult | object | Deployment outcome |
| deploymentResult.success | boolean | `true` when the connection was updated and redeployed |

### Error Response (400 Bad Request)

```json
{
  "status": "FAILURE",
  "resultMessage": "MCP connection (name: github-mcp) was not found!"
}
```

or

```json
{
  "status": "FAILURE",
  "resultMessage": "MCP connection name in path (github-mcp) does not match name in body (github-mcpp)!"
}
```

### Error Response (401 Unauthorized)

```json
{
  "status": "FAILURE",
  "resultMessage": "Token is not valid!"
}
```

```bash
curl -X PUT \
  "https://demo.apinizer.com/apiops/projects/MyProject/mcp-connections/github-mcp/" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "_class": "mcp",

    "transport": "HTTP_SSE",
    "mcpProtocolVersion": "2024-11-05",
    "allowPrivateNetworkEgress": false,
    "authScheme": "BEARER",
    "bearerToken": "",
    "capabilityCacheTtlSeconds": 600,
    "capabilityCacheEnabled": true
  }'
```

## Notes and Warnings

- **Update-Only**: Unlike [Create MCP Connection](/api-reference/mcp-connections/create-mcp-connection)
  (which upserts), `PUT` fails with a `400` when the named connection does not exist.
- **Secret Preservation (INV-06)**: Sending a blank `bearerToken`/`clientSecret` keeps the stored,
  encrypted value unchanged. Send a non-blank value to rotate the secret.
- **Secrets Never Returned**: The response does not echo the object; read it back with
  [Get MCP Connection](/api-reference/mcp-connections/get-mcp-connection), where secrets are masked.
- **Polymorphic Body**: The `"_class": "mcp"` discriminator is mandatory.
- **Save = Encrypt + Deploy**: On success the connection is redeployed to the gateway workers.
