---
title: "Create A2A Connection"
description: "Creates an AI Gateway A2A outbound connection in the specified project. This endpoint is an upsert — if a connection with the same name already exists it is updated in place. On save, @SecretData fields are encrypted and the connection is deployed to the workers."
---

## Endpoint

```
POST /apiops/projects/{projectName}/a2a-connections/{connectionName}/
```

## Authentication

Requires a Personal API Access Token.

### Header

```
Authorization: Bearer YOUR_TOKEN
```

## Request

### Headers

| Header | Value | Required |
|--------|-------|----------|
| Authorization | Bearer \{token\} | Yes |
| Content-Type | application/json | Yes |

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| projectName | string | Yes | Project name |
| connectionName | string | Yes | A2A outbound connection name (must match `name` in the body) |

### Query Parameters

None

### Request Body

The body is a polymorphic `ConnectionConfig` document. Because A2A is a connection-config subtype,
the discriminator `"_class": "a2a"` is **required** so the payload is deserialized as an A2A
connection.

#### Full JSON Body Example - Bearer Token Auth

```json
{
  "_class": "a2a",
  "name": "weather-agent",
  "description": "External weather A2A agent",
  "enabled": true,
  "endpoint": "https://agent.example.com/a2a",
  "a2aProtocolVersion": "1.0",
  "allowPrivateNetworkEgress": false,
  "authScheme": "BEARER",
  "bearerToken": "eyJhbGciOi...secret...",
  "agentCardCacheTtlSeconds": 300,
  "agentCardCacheEnabled": true
}
```

#### Full JSON Body Example - OAuth2 Client Credentials Auth

```json
{
  "_class": "a2a",
  "name": "billing-agent",
  "description": "Partner billing A2A agent",
  "enabled": true,
  "endpoint": "https://partner.example.com/a2a",
  "a2aProtocolVersion": "1.0",
  "allowPrivateNetworkEgress": false,
  "authScheme": "OAUTH2_CLIENT_CREDENTIALS",
  "tokenEndpoint": "https://partner.example.com/oauth/token",
  "clientId": "apinizer-client",
  "clientSecret": "s3cr3t-value",
  "scope": "a2a.invoke a2a.read",
  "credentialRef": "partner-oauth-credential",
  "agentCardCacheTtlSeconds": 300,
  "agentCardCacheEnabled": true
}
```

#### Request Body Fields

| Field | Type | Required | Default | Description |
|-------|------|----------|---------|-------------|
| _class | string | Yes | - | Connection-config discriminator — must be `"a2a"` |
| name | string | Yes | - | Connection name — must equal the `connectionName` path variable |
| description | string | No | - | Connection description |
| enabled | boolean | No | `true` | Whether the connection is enabled |
| endpoint | string | Yes | - | A2A agent endpoint (AgentCard well-known URL / message endpoint) |
| a2aProtocolVersion | string | No | `1.0` | A2A protocol/adapter version |
| allowPrivateNetworkEgress | boolean | No | `false` | Permit cluster-internal / private-network egress. **System-admin only** — a non-admin caller setting this to `true` is rejected |
| authScheme | string | No | `NONE` | Auth scheme: `BEARER`, `OAUTH2_CLIENT_CREDENTIALS`, `MTLS`, or `NONE` |
| bearerToken | string | Conditional | - | Static bearer token for the `BEARER` scheme (`@SecretData`, encrypted on save). A blank value on update preserves the stored secret |
| credentialRef | string | No | - | Credential reference for the `OAUTH2_CLIENT_CREDENTIALS` scheme |
| tokenEndpoint | string | Conditional | - | OAuth2 token endpoint URL (`OAUTH2_CLIENT_CREDENTIALS`) |
| clientId | string | Conditional | - | OAuth2 client ID (`OAUTH2_CLIENT_CREDENTIALS`) |
| clientSecret | string | Conditional | - | OAuth2 client secret (`@SecretData`, encrypted on save). A blank value on update preserves the stored secret |
| scope | string | No | - | OAuth2 scope, space-delimited (`OAUTH2_CLIENT_CREDENTIALS`) |
| sslConfigRef | string | No | - | SSL config reference for the `MTLS` scheme (placeholder — handshake not yet wired) |
| agentCardCacheTtlSeconds | integer | No | `300` | AgentCard cache TTL in seconds (`0` = no cache) |
| agentCardCacheEnabled | boolean | No | `true` | Whether AgentCard caching is enabled |
| discoveredSkills | array | No | `[]` | Discovered skills. Normally populated by [Parse A2A Skills](/api-reference/a2a-connections/parse-a2a-skills), not set manually |

### Notes

- `name` in the body must match the `connectionName` path variable (case-insensitive), otherwise a `400` is returned.
- `_class` must be `"a2a"`; without it the body cannot be deserialized as an A2A connection.
- The response does not echo the saved object — use [Get A2A Connection](/api-reference/a2a-connections/get-a2a-connection) to read it back (with secrets masked).

## Response

### Success Response (200 OK)

```json
{
  "status": "SUCCESS",
  "deploymentResult": {
    "success": true
  }
}
```

#### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| status | string | Response status: `SUCCESS` or `FAILURE` |
| deploymentResult | object | Deployment outcome |
| deploymentResult.success | boolean | `true` when the connection was saved and deployed |

### Error Response (400 Bad Request)

```json
{
  "status": "FAILURE",
  "resultMessage": "A2A outbound connection name in path (weather-agent) does not match name in body (weather-agnt)!"
}
```

or

```json
{
  "status": "FAILURE",
  "resultMessage": "Private-network egress (allowPrivateNetworkEgress) can only be enabled by a system administrator!"
}
```

### Error Response (401 Unauthorized)

```json
{
  "status": "FAILURE",
  "resultMessage": "Token is not valid!"
}
```

## Notes and Warnings

- **Upsert Semantics**: `POST /{connectionName}/` creates the connection, or updates it in place
  if the name already exists in the project. To force an update-only operation use
  [Update A2A Connection](/api-reference/a2a-connections/update-a2a-connection).
- **Secret Handling**: `bearerToken` and `clientSecret` are `@SecretData` — encrypted on save and
  never returned on read. On an upsert that hits an existing connection, sending a **blank**
  `bearerToken`/`clientSecret` preserves the stored secret (INV-06).
- **Private-Network Egress**: `allowPrivateNetworkEgress = true` is rejected for non-system-admin
  callers (SSRF hardening). Loopback, link-local, cloud metadata (169.254.169.254), multicast and
  CGNAT targets remain blocked regardless of this flag.
- **Save = Encrypt + Deploy**: On success the connection is deployed to the workers (deferred,
  after commit).
