## Endpoints

- [Get AI Gateway Settings](/api-reference/ai-gateway-settings/get-ai-gateway-settings) - Read the global AI Gateway settings (singleton)
- [Update AI Gateway Settings](/api-reference/ai-gateway-settings/update-ai-gateway-settings) - Update the global AI Gateway settings (partial update)

## Authentication

All endpoints require authentication using a Personal API Access Token.

## Permissions

- **Admin Only** - All endpoints require admin privileges (sysAdmin user or a user with the `ADMIN` role). A project-scoped `AI_DEVELOPMENT` token is **not** sufficient.
