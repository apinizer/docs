## Settings Endpoints

### General Settings

- [Metadata](/api-reference/api-proxies/settings/update-metadata) - Update API proxy metadata (name, description, category, sharing)
- [Client Route](/api-reference/api-proxies/settings/update-client-route) - Configure client route metadata

### Security Settings

- [CORS Settings](/api-reference/api-proxies/settings/update-cors-settings) - Configure Cross-Origin Resource Sharing (CORS)
- [mTLS Settings](/api-reference/api-proxies/settings/update-mtls-settings) - Configure mutual TLS
- [NTLM Settings](/api-reference/api-proxies/settings/update-ntlm-settings) - Configure NTLM authentication
- [API Keys](/api-reference/api-proxies/settings/update-api-keys) - Configure API keys

### Routing Settings

- [Routing Addresses](/api-reference/api-proxies/settings/update-routing-addresses) - Configure backend routing addresses
- [Routing Status](/api-reference/api-proxies/settings/update-routing-status) - Enable or disable routing
- [Connection Settings](/api-reference/api-proxies/settings/update-connection-settings) - Configure connection settings
- [Proxy Server Settings](/api-reference/api-proxies/settings/update-proxy-server-settings) - Configure proxy server settings

### Performance Settings

- [Cache Settings](/api-reference/api-proxies/settings/update-cache-settings) - Configure caching behavior
- [Circuit Breaker Settings](/api-reference/api-proxies/settings/update-circuit-breaker-settings) - Configure circuit breaker

### Logging & Monitoring

- [Traffic Log Settings](/api-reference/api-proxies/settings/update-traffic-log-settings) - Configure traffic logging

### Error Handling

- [JSON Error Template](/api-reference/api-proxies/settings/update-json-error-template) - Configure JSON error response template
- [XML Error Template](/api-reference/api-proxies/settings/update-xml-error-template) - Configure XML error response template

### Other Settings

- [Forwarded IP Header](/api-reference/api-proxies/settings/update-forwarded-ip-header) - Configure forwarded IP header
- [Hide API Definition](/api-reference/api-proxies/settings/update-hide-spec-file) - Hide API specification file from public access
