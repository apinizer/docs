## Endpoints

- [List API Proxies](/api-reference/api-proxies/crud/list-api-proxies) - Get all API proxies for a project
- [Get API Proxy](/api-reference/api-proxies/crud/get-api-proxy) - Get detailed information about a specific API proxy
- [Create API Proxy from URL](/api-reference/api-proxies/crud/create-api-proxy-from-url) - Create an API proxy from an OpenAPI/Swagger/WSDL URL
- [Create API Proxy from File](/api-reference/api-proxies/crud/create-api-proxy-from-file) - Create an API proxy from an uploaded specification file
- [Update API Proxy](/api-reference/api-proxies/crud/update-api-proxy) - Update an existing API proxy
- [Update API Proxy from URL](/api-reference/api-proxies/crud/update-api-proxy-from-url) - Update API proxy specification from URL
- [Update API Proxy from File](/api-reference/api-proxies/crud/update-api-proxy-from-file) - Update API proxy specification from file
- [Import API Proxy](/api-reference/api-proxies/crud/import-api-proxy) - Import an API proxy
- [Import API Proxy and Override](/api-reference/api-proxies/crud/import-api-proxy-and-override) - Import API proxy with override options
- [Export API Proxy](/api-reference/api-proxies/crud/export-api-proxy) - Export an API proxy
- [Delete API Proxy](/api-reference/api-proxies/crud/delete-api-proxy) - Delete an API proxy
