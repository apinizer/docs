## Endpoints

- [List Endpoints](/api-reference/api-proxies/endpoints/list-endpoints) - Get all endpoints for an API proxy
- [Get Endpoint](/api-reference/api-proxies/endpoints/get-endpoint) - Get detailed information about a specific endpoint
- [Create Endpoint](/api-reference/api-proxies/endpoints/create-endpoint) - Create a new endpoint
- [Update Endpoint](/api-reference/api-proxies/endpoints/update-endpoint) - Update an existing endpoint
- [Delete Endpoint](/api-reference/api-proxies/endpoints/delete-endpoint) - Delete an endpoint
- [Update Endpoint Status](/api-reference/api-proxies/endpoints/update-endpoint-status) - Enable or disable an endpoint
- [Update Endpoint WSA](/api-reference/api-proxies/endpoints/update-endpoint-wsa) - Update SOAP WS-Addressing settings for an endpoint
- [Update Endpoint Cache](/api-reference/api-proxies/endpoints/update-endpoint-cache) - Update cache settings for an endpoint
- [List Endpoint Policies](/api-reference/api-proxies/endpoints/list-endpoint-policies) - List policies applied to an endpoint

## Methods

- [List Methods](/api-reference/api-proxies/endpoints/list-methods) - List all methods with optional policy details
- [Get Method Policies](/api-reference/api-proxies/endpoints/get-method-policies) - Get policies for a specific method by name and HTTP method
