---
title: "Get Available Connectors"
description: "Returns the list of connectors available for traffic log settings in the specified environment. Use this endpoint to discover which connectors can be configured for log delivery."
---

## Endpoint

```
GET /apiops/projects/{projectName}/api-proxies/{apiProxyName}/traffic-log-settings/connectors/{environmentName}/
```

## Authentication

Requires a Personal API Access Token.

### Header

```
Authorization: Bearer YOUR_TOKEN
```

## Request

### Headers

| Header | Value | Required |
|--------|-------|----------|
| Authorization | Bearer {token} | Yes |

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| projectName | string | Yes | Project name |
| apiProxyName | string | Yes | API Proxy name |
| environmentName | string | Yes | Environment name |

## Response

### Success Response (200 OK)

```json
{
  "status": "SUCCESS",
  "resultList": [
    {
      "connectionName": "Production Elasticsearch",
      "connectorType": "ELASTICSEARCH_INDEX",
      "displayLabel": "Elasticsearch (Production Elasticsearch)"
    },
    {
      "connectionName": "Backup Elasticsearch",
      "connectorType": "ELASTICSEARCH_INDEX",
      "displayLabel": "Elasticsearch (Backup Elasticsearch)"
    },
    {
      "connectionName": "Default Kafka",
      "connectorType": "KAFKA_SEND",
      "displayLabel": "Kafka (Default Kafka)"
    }
  ],
  "resultCount": 3
}
```

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| connectionName | string | Name of the connection configuration. Use this value as the key in `connectorSettingsMap` when updating traffic log settings. |
| connectorType | string | Connector type (e.g., ELASTICSEARCH_INDEX, KAFKA_SEND, DB_INSERT_UPDATE_DELETE, SYSLOG_SEND) |
| displayLabel | string | Human-readable display label combining the connector type with the connection name |

### Error Response (400 Bad Request)

```json
{
  "error": "bad_request",
  "error_description": "Environment (staging) is not found!"
}
```

## cURL Example

```bash
curl -X GET \
  "https://demo.apinizer.com/apiops/projects/MyProject/api-proxies/MyAPI/traffic-log-settings/connectors/production/" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## Permissions

User must have **`API_MANAGEMENT` + `VIEW`** permission in the project.

## Notes

- The connector list is dynamically generated based on what is configured in the specified environment.
- If the environment has no connectors, an empty list is returned.
- Use the `connectionName` from this response as the key in `connectorSettingsMap` when updating traffic log settings.
- For backward compatibility, connector type names (e.g., `ELASTICSEARCH_INDEX`) can also be used as keys in `connectorSettingsMap`.
