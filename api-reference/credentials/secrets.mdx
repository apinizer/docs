## Endpoints

### Secret Key Management

- [Set Secret Key](/api-reference/credentials/secrets/set-secret-key) - Assign a secret key to a credential
- [Remove Secret Key](/api-reference/credentials/secrets/remove-secret-key) - Remove secret key from a credential

### Certificate Management

- [Set Certificate](/api-reference/credentials/secrets/set-certificate) - Assign a certificate to a credential
- [Remove Certificate](/api-reference/credentials/secrets/remove-certificate) - Remove certificate from a credential

### Key Management

- [Set Public Key](/api-reference/credentials/secrets/set-public-key) - Assign a public key to a credential
- [Remove Public Key](/api-reference/credentials/secrets/remove-public-key) - Remove public key from a credential
- [Set Private Key](/api-reference/credentials/secrets/set-private-key) - Assign a private key to a credential
- [Remove Private Key](/api-reference/credentials/secrets/remove-private-key) - Remove private key from a credential

### Keystore Management

- [Set Keystore](/api-reference/credentials/secrets/set-keystore) - Assign a keystore to a credential
- [Remove Keystore](/api-reference/credentials/secrets/remove-keystore) - Remove keystore from a credential
- [Set Truststore](/api-reference/credentials/secrets/set-truststore) - Assign a truststore to a credential
- [Remove Truststore](/api-reference/credentials/secrets/remove-truststore) - Remove truststore from a credential
